# 法人設立・法人成りガイド

## 1. 株式会社設立の手順

### 1-1. 設立までの全体フロー
```
Step 1: 基本事項の決定
  ↓
Step 2: 定款の作成
  ↓
Step 3: 公証人による定款の認証
  ↓
Step 4: 資本金の払込み
  ↓
Step 5: 設立登記申請
  ↓
Step 6: 各種届出書の提出
```

### 1-2. Step 1: 基本事項の決定
| 決定事項 | 内容 |
|---------|------|
| 商号（社名） | 同一住所に同一商号がないことを確認（法務局で類似商号調査） |
| 事業目的 | 将来行う可能性のある事業も含めて記載 |
| 本店所在地 | 登記上の住所（自宅でも可） |
| 資本金額 | 1円以上（実務上は100万円〜1,000万円が多い） |
| 発起人 | 1人以上（出資者） |
| 取締役 | 1人以上（取締役会設置会社は3人以上+監査役1人以上） |
| 事業年度 | 任意に設定可能 |

### 1-3. Step 2: 定款の作成
**定款の絶対的記載事項**（記載がないと定款自体が無効）:
1. 目的
2. 商号
3. 本店の所在地
4. 設立に際して出資される財産の価額又はその最低額
5. 発起人の氏名又は名称及び住所

**定款の相対的記載事項**（記載しないとその事項の効力が生じない）:
- 株式の譲渡制限
- 取締役・監査役の任期の伸長（最長10年）
- 株主総会の開催地
- 現物出資に関する事項

**定款の任意的記載事項**:
- 事業年度
- 取締役の員数
- 公告方法（官報、電子公告、日刊紙）

### 1-4. Step 3: 公証人による定款の認証
- **認証場所**: 本店所在地を管轄する公証役場
- **費用**:
  - 認証手数料: 資本金100万円未満30,000円、100万円以上300万円未満40,000円、その他50,000円
  - 謄本交付手数料: 250円×ページ数
  - 電子定款の場合: 収入印紙40,000円が不要
- **持参物**: 定款3通（公証人保管用・会社保管用・法務局提出用）、発起人全員の印鑑証明書、実印
- **電子定款**: 電子署名を付して電子的に認証（収入印紙代40,000円が不要、合同会社は認証自体不要）

### 1-5. Step 4: 資本金の払込み
- 定款認証後、発起人代表の個人口座に各発起人が出資金を振り込む
- **払込証明書**の作成（通帳のコピーを添付）
- 法人口座の開設は設立後（設立登記完了後に法人口座を開設し、個人口座から法人口座に資金を移転）

### 1-6. Step 5: 設立登記申請
- **申請先**: 本店所在地を管轄する法務局
- **登記申請日 = 会社の設立日**
- **費用**: 登録免許税 = 資本金 × 0.7%（最低15万円）
- **必要書類**:
  - 設立登記申請書
  - 定款（認証済み）
  - 発起人の決定書（本店所在地・設立時取締役等を決定）
  - 設立時取締役の就任承諾書
  - 払込証明書
  - 印鑑届書（法人の実印登録）
  - 設立時取締役の印鑑証明書
  - 資本金の額の計上に関する証明書
- **処理期間**: 申請後約1週間で登記完了

### 1-7. 設立にかかる費用の総額

| 費目 | 株式会社 |
|------|---------|
| 定款認証手数料 | 30,000〜50,000円 |
| 定款の収入印紙（紙定款の場合） | 40,000円（電子定款なら不要） |
| 定款謄本交付手数料 | 約2,000円 |
| 登録免許税 | 150,000円（最低額） |
| 印鑑作成費 | 5,000〜30,000円 |
| **合計（電子定款の場合）** | **約190,000〜230,000円** |
| **合計（紙定款の場合）** | **約230,000〜270,000円** |

## 2. 合同会社設立の手順

### 2-1. 株式会社との違い
| 項目 | 株式会社 | 合同会社 |
|------|---------|---------|
| 公証人認証 | **必要** | **不要** |
| 登録免許税 | 150,000円（最低） | **60,000円**（最低） |
| 定款の印紙税 | 40,000円（紙の場合） | 40,000円（紙の場合） |
| 設立費用の合計 | 約20〜27万円 | **約6〜10万円** |
| 役員の名称 | 取締役・代表取締役 | 社員・業務執行社員・代表社員 |
| 決算公告 | 必要 | 不要 |
| 利益配分 | 出資比率に応じて | 定款で自由に設定可能 |
| 社会的信用 | 高い | やや低い |

### 2-2. 合同会社の設立フロー
```
Step 1: 基本事項の決定
  ↓
Step 2: 定款の作成（公証人認証不要）
  ↓
Step 3: 資本金の払込み
  ↓
Step 4: 設立登記申請
  ↓
Step 5: 各種届出書の提出
```

### 2-3. 合同会社設立にかかる費用

| 費目 | 合同会社 |
|------|---------|
| 定款認証手数料 | 不要 |
| 定款の収入印紙（紙定款の場合） | 40,000円（電子定款なら不要） |
| 登録免許税 | 60,000円（最低額） |
| 印鑑作成費 | 5,000〜30,000円 |
| **合計（電子定款の場合）** | **約65,000〜90,000円** |
| **合計（紙定款の場合）** | **約105,000〜130,000円** |

## 3. 設立後の届出書一覧

### 3-1. 税務署への届出

| 届出書 | 提出期限 | 添付書類 | 備考 |
|--------|---------|---------|------|
| **法人設立届出書** | 設立後2ヶ月以内 | 定款の写し、登記事項証明書、株主名簿の写し、設立時貸借対照表 | 必須 |
| **青色申告の承認申請書** | 設立後3ヶ月 or 第1期末のいずれか早い日の前日まで | なし | 強く推奨（必須レベル） |
| **給与支払事務所等の開設届出書** | 設立後1ヶ月以内 | なし | 役員報酬・給与を支払う場合 |
| **源泉所得税の納期の特例の承認に関する申請書** | 随時 | なし | 従業員10人未満の場合に推奨 |
| 棚卸資産の評価方法の届出書 | 第1期の確定申告期限まで | なし | 最終仕入原価法以外を選択する場合 |
| 減価償却資産の償却方法の届出書 | 第1期の確定申告期限まで | なし | 法定償却方法以外を選択する場合 |
| 消費税の新設法人に該当する旨の届出書 | 速やかに | なし | 資本金1,000万円以上の場合 |
| 適格請求書発行事業者の登録申請書 | 必要に応じて | なし | インボイス発行が必要な場合 |
| 事前確定届出給与に関する届出書 | 設立後2ヶ月以内等 | なし | 役員賞与を損金算入する場合 |

### 3-2. 都道府県税事務所への届出

| 届出書 | 提出期限 | 添付書類 |
|--------|---------|---------|
| **法人設立届出書（事業開始等届出書）** | 各都道府県の条例による（概ね設立後15日〜2ヶ月以内） | 定款の写し、登記事項証明書 |

**主な都道府県の提出期限**:
- 東京都: 事業開始の日から15日以内
- 神奈川県: 設立後2ヶ月以内
- 大阪府: 設立後2ヶ月以内
- 埼玉県: 設立後15日以内

### 3-3. 市区町村への届出

| 届出書 | 提出期限 | 添付書類 |
|--------|---------|---------|
| **法人設立届出書** | 各市区町村の条例による | 定款の写し、登記事項証明書 |

**注意**: 東京都23区内は都税事務所のみ（市区町村への届出不要）

### 3-4. 年金事務所への届出

| 届出書 | 提出期限 | 添付書類 |
|--------|---------|---------|
| **健康保険・厚生年金保険 新規適用届** | 事実発生から5日以内 | 登記事項証明書 |
| **健康保険・厚生年金保険 被保険者資格取得届** | 事実発生から5日以内 | なし |
| 健康保険 被扶養者（異動）届 | 事実発生から5日以内 | 被扶養者の収入証明等 |

**注意**: 法人は社長1人でも社会保険（健康保険・厚生年金）の加入義務がある

### 3-5. ハローワーク（公共職業安定所）への届出

| 届出書 | 提出期限 | 添付書類 |
|--------|---------|---------|
| **雇用保険 適用事業所設置届** | 設置の日の翌日から10日以内 | 登記事項証明書、労働保険関係成立届の控え、労働者名簿、賃金台帳 |
| **雇用保険 被保険者資格取得届** | 被保険者となった日の属する月の翌月10日まで | なし |

**注意**: 従業員（パート・アルバイト含む）を雇用する場合に必要。役員のみの場合は不要

### 3-6. 労働基準監督署への届出

| 届出書 | 提出期限 | 添付書類 |
|--------|---------|---------|
| **労働保険 保険関係成立届** | 保険関係成立の日の翌日から10日以内 | 登記事項証明書 |
| **労働保険 概算保険料申告書** | 保険関係成立の日の翌日から50日以内 | なし |

**注意**: 従業員を1人でも雇用したら加入義務あり

## 4. 資本金の決め方

### 4-1. 消費税との関係
- **資本金1,000万円未満**: 設立後2事業年度は消費税の免税事業者となれる（原則）
- **資本金1,000万円以上**: 設立初年度から消費税の課税事業者
- **特定期間の判定**: 設立1期目の上半期の課税売上高と給与支払額がともに1,000万円超の場合、2期目から課税事業者

### 4-2. 許認可との関係
| 業種 | 必要な資本金等 |
|------|-------------|
| 一般建設業 | 500万円以上の自己資本 |
| 特定建設業 | 2,000万円以上の自己資本 |
| 一般労働者派遣事業 | 2,000万円以上の資産要件 |
| 有料職業紹介事業 | 500万円以上の資産要件 |
| 古物商 | 資本金要件なし |

### 4-3. 融資との関係
- 日本政策金融公庫の創業融資: 自己資金要件（創業資金の1/10以上が目安）
- 信用保証協会の保証: 資本金の額が中小企業の定義に該当すること
- 金融機関の審査: 資本金の充実度は信用力の指標

### 4-4. 実務上のポイント
- **推奨**: 資本金は**100万円〜999万円**（消費税免税のメリットを享受）
- 資本金が少なすぎると信用力に影響
- 資本金は設立後に増資可能（登録免許税: 増加資本金 × 0.7%、最低3万円）

## 5. 決算月の選び方

### 5-1. 考慮すべきポイント

| ポイント | 詳細 |
|---------|------|
| **繁忙期を避ける** | 繁忙期に決算作業を行うのは負担。売上が安定する時期を決算月にする |
| **資金繰り** | 法人税等の納付は決算日から2ヶ月以内。資金に余裕がある時期にする |
| **消費税の免税期間の最大化** | 設立日から最も遠い月を決算月にすると、免税期間が最長（最大約2年間） |
| **税理士の繁忙期を避ける** | 2月〜3月（確定申告シーズン）、12月（年末調整）は税理士も繁忙期。3月決算（5月申告）は集中しがち |
| **業界の慣行** | 上場企業の取引先の場合、3月決算に合わせることも |

### 5-2. 消費税免税期間の最大化例
- 設立日: 2024年4月1日
- 決算月を**3月**にした場合: 第1期 2024/4/1〜2025/3/31（12ヶ月）→ 免税期間最大
- 決算月を**4月**にした場合: 第1期 2024/4/1〜2024/4/30（1ヶ月）→ 免税期間が短くなる

### 5-3. 決算月の変更
- 定款の変更（株主総会の特別決議 ※普通決議で足りる場合もある）
- 税務署・都道府県税事務所・市区町村に異動届出書を提出
- 変更した事業年度が1年を超えることはできない

## 6. 法人成りのメリット

### 6-1. 節税メリット
| メリット | 詳細 |
|---------|------|
| **法人税率の低さ** | 所得800万円以下: 15%（中小法人）、800万円超: 23.2%。個人の所得税は最高45%（+住民税10%） |
| **役員報酬による所得分散** | 法人の利益を役員報酬として支給し、給与所得控除を活用 |
| **退職金の活用** | 法人は退職金を損金算入可能。個人事業では事業主自身の退職金は経費にできない |
| **生命保険の活用** | 法人名義の生命保険料を損金算入可能（一定の要件あり） |
| **経費の幅の拡大** | 社宅、出張旅費日当、慶弔見舞金等を法人の経費にできる |
| **欠損金の繰越控除** | 法人: 10年間（個人: 3年間） |
| **消費税の免税期間** | 資本金1,000万円未満で設立すれば最大2年間免税 |

### 6-2. 社会的信用
- 法人の方が取引先からの信用度が高い
- 法人でないと取引しない企業もある（大手企業との取引、官公庁の入札等）
- 金融機関からの融資が受けやすい

### 6-3. その他のメリット
- 有限責任（出資額を限度とする責任）
- 事業承継がしやすい
- 社員の採用がしやすい（社会保険完備）

## 7. 法人成りのデメリット

### 7-1. コスト増
| デメリット | 詳細 |
|-----------|------|
| **社会保険料の負担** | 法人は健康保険・厚生年金の加入義務あり。事業主負担は給与の約15% |
| **法人住民税の均等割** | 赤字でも最低年間約7万円（都道府県2万円+市区町村5万円） |
| **設立費用** | 株式会社: 約20〜27万円、合同会社: 約6〜10万円 |
| **税理士報酬** | 法人は帳簿・申告が複雑。税理士報酬は個人より高い（月額2〜5万円+決算料10〜30万円が目安） |
| **決算公告の費用** | 株式会社は官報公告義務あり（約6万円/回） |

### 7-2. 事務負担増
- 複式簿記による帳簿作成が必須
- 社会保険・労働保険の手続き
- 法人税・消費税・地方税の申告（個人の確定申告より複雑）
- 役員変更等の登記手続き

### 7-3. その他のデメリット
- 利益を自由に引き出せない（役員報酬は定期同額、配当は二重課税）
- 解散・清算の手続きが面倒
- 社会保険の加入義務（個人事業では常時5人未満は任意）

## 8. 法人成りの最適タイミング

### 8-1. 所得基準（目安）
- **事業所得が概ね900万円超**: 法人化のメリットが出始める
- 所得税+住民税の実効税率が法人税の実効税率を上回る水準

### 8-2. 税率比較
| 所得金額 | 個人（所得税+住民税） | 法人（法人税+地方税の実効税率） |
|---------|---------------------|-------------------------------|
| 330万円超〜695万円以下 | 約30%（所得税20%+住民税10%） | 約22%（中小法人の800万円以下） |
| 695万円超〜900万円以下 | 約33%（所得税23%+住民税10%） | 約22% |
| 900万円超〜1,800万円以下 | 約43%（所得税33%+住民税10%） | 約22%〜34% |
| 1,800万円超〜4,000万円以下 | 約50%（所得税40%+住民税10%） | 約34% |
| 4,000万円超 | 約55%（所得税45%+住民税10%） | 約34% |

**注意**: 上記は単純比較であり、実際には社会保険料の負担、役員報酬の設定、給与所得控除等を総合的に考慮する必要がある

### 8-3. その他のタイミング指標
- 消費税の課税事業者になるタイミング（売上1,000万円超）
- 従業員を雇用するタイミング
- 取引先から法人化を求められたタイミング
- 事業拡大・融資を受けるタイミング

### 8-4. 法人成りの際の注意点
- 個人事業の資産・負債を法人に移転する必要がある（時価での売却 or 現物出資）
- 個人事業の廃業届・青色申告の取りやめ届出書等の提出
- 個人事業の最後の確定申告（廃業年分）
- 退職所得控除の計算上、個人事業の期間は勤続年数に含められない
- 不動産を法人に移転する場合、不動産取得税・登録免許税が発生
