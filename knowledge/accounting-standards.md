# 会計基準・帳簿作成ガイド

## 1. 企業会計原則の7つの一般原則

### 1-1. 真実性の原則
企業会計は、企業の財政状態及び経営成績に関して、真実な報告を提供するものでなければならない。
- 最上位の原則（他の6つの原則の上位概念）
- 粉飾決算・逆粉飾は真実性の原則に反する

### 1-2. 正規の簿記の原則
企業会計は、すべての取引につき、正規の簿記の原則に従って、正確な会計帳簿を作成しなければならない。
- **正規の簿記の3要件**: 網羅性（全ての取引を記録）、検証可能性（証拠書類に基づく）、秩序性（体系的に記録）
- 複式簿記の採用が原則

### 1-3. 資本取引・損益取引区分の原則
資本取引と損益取引とを明瞭に区別し、特に資本剰余金と利益剰余金とを混同してはならない。
- 資本取引: 株主との取引（増資、減資、配当等）
- 損益取引: 営業活動による取引（売上、仕入、経費等）

### 1-4. 明瞭性の原則
企業会計は、財務諸表によって、利害関係者に対し必要な会計事実を明瞭に表示し、企業の状況に関する判断を誤らせないようにしなければならない。
- 財務諸表の表示方法・注記事項に関する原則
- 重要な会計方針の開示

### 1-5. 継続性の原則
企業会計は、その処理の原則及び手続を毎期継続して適用し、みだりにこれを変更してはならない。
- 正当な理由がある場合のみ変更可能
- 変更した場合はその旨・理由・影響額を注記

### 1-6. 保守主義の原則
企業の財政に不利な影響を及ぼす可能性がある場合には、これに備えて適当に健全な会計処理をしなければならない。
- 収益は確実なものだけを計上、費用・損失は予想されるものも計上
- 過度の保守主義は真実性の原則に反する

### 1-7. 単一性の原則
株主総会提出のため、信用目的のため、租税目的のため等種々の目的のために異なる形式の財務諸表を作成する必要がある場合、それらの内容は、信頼しうる会計記録に基づいて作成されたものであって、政策の考慮のために事実の真実な表示をゆがめてはならない。
- 二重帳簿の禁止（実質一元、形式多元）

## 2. 中小企業の会計に関する指針の要点

### 2-1. 位置づけ
- 日本税理士会連合会・日本公認会計士協会・日本商工会議所・企業会計基準委員会の4団体が共同で作成
- 中小企業（会計監査人を設置しない会社）が準拠すべき会計指針
- 「中小企業の会計に関する基本要領」（中小会計要領）はさらに簡便な基準

### 2-2. 主要な要点
| 項目 | 内容 |
|------|------|
| 収益認識 | 実現主義（出荷基準、引渡基準、検収基準等） |
| 棚卸資産 | 最終仕入原価法による原価法が一般的 |
| 固定資産 | 税法の耐用年数・償却方法に準拠可能 |
| 引当金 | 税法で認められる範囲で計上可能（退職給付引当金は簡便法可） |
| リース | 所有権移転外ファイナンス・リースは賃貸借処理も可能 |
| 退職給付 | 簡便法（期末自己都合要支給額を計上）が認められる |
| 税効果会計 | 適用は任意 |
| 金融商品 | 有価証券の評価は取得原価も可能（上場株式は時価評価が望ましい） |

## 3. 帳簿の種類

### 3-1. 主要簿

#### 仕訳帳
- 全ての取引を発生順に記録する帳簿
- 日付・勘定科目・金額・摘要を記録
- 会計ソフトでは入力時に自動作成

#### 総勘定元帳
- 勘定科目ごとに全ての取引を記録する帳簿
- 仕訳帳の転記先
- 各勘定科目の残高を把握できる

### 3-2. 補助簿

| 補助簿 | 記録内容 | 用途 |
|--------|---------|------|
| **現金出納帳** | 現金の入出金を日付順に記録 | 現金残高の管理 |
| **預金出納帳** | 預金口座ごとの入出金を記録 | 預金残高の管理（口座別に作成） |
| **売掛金元帳（得意先元帳）** | 得意先ごとの売掛金の発生・回収を記録 | 売掛金残高の管理 |
| **買掛金元帳（仕入先元帳）** | 仕入先ごとの買掛金の発生・支払を記録 | 買掛金残高の管理 |
| **固定資産台帳** | 固定資産ごとの取得・減価償却・除却を記録 | 固定資産の管理・減価償却計算 |
| **手形記入帳** | 受取手形・支払手形の明細を記録 | 手形の期日管理 |
| **売上帳** | 売上取引の明細を記録 | 売上の分析 |
| **仕入帳** | 仕入取引の明細を記録 | 仕入の分析 |
| **経費帳** | 経費の支払明細を記録 | 経費の管理 |
| **賃金台帳** | 従業員ごとの給与明細を記録 | 給与計算・源泉徴収 |

## 4. 複式簿記の基本

### 4-1. 借方（左側）と貸方（右側）の概念

| 勘定科目の種類 | 増加時 | 減少時 |
|--------------|--------|--------|
| **資産**（現金、売掛金、建物等） | 借方 | 貸方 |
| **負債**（買掛金、借入金等） | 貸方 | 借方 |
| **純資産**（資本金、利益剰余金等） | 貸方 | 借方 |
| **収益**（売上、受取利息等） | 貸方 | 借方 |
| **費用**（仕入、給料、家賃等） | 借方 | 貸方 |

### 4-2. 仕訳の基本ルール
- 全ての取引は「借方」と「貸方」に同額を記録する（貸借平均の原則）
- 借方合計 = 貸方合計（必ず一致）

## 6. 決算整理仕訳の全パターン

### 6-1. 前払費用の計上
翌期以降に帰属する費用を当期の費用から控除し、資産に振り替える。

**例**: 3月決算法人が1月に1年分の保険料120,000円を支払い
- 当期分: 120,000円 × 3/12 = 30,000円
- 前払分: 120,000円 × 9/12 = 90,000円
```
借方: 前払費用 90,000  貸方: 保険料 90,000
```

### 6-2. 未払費用の計上
当期に帰属する費用で、まだ支払っていないものを計上する。

**例**: 3月末時点の未払給与（3月21日〜31日分）50,000円
```
借方: 給料手当 50,000  貸方: 未払費用 50,000
```

### 6-3. 前受収益の計上
翌期以降に帰属する収益を当期の収益から控除する。

**例**: 3月決算法人が2月に6ヶ月分の家賃600,000円を受領
- 当期分: 600,000円 × 2/6 = 200,000円
- 前受分: 600,000円 × 4/6 = 400,000円
```
借方: 受取家賃 400,000  貸方: 前受収益 400,000
```

### 6-4. 未収収益の計上
当期に帰属する収益で、まだ受け取っていないものを計上する。

**例**: 貸付金に対する利息（当期帰属分）10,000円が未収
```
借方: 未収収益 10,000  貸方: 受取利息 10,000
```

### 6-5. 減価償却費の計上
```
借方: 減価償却費 XXX  貸方: 減価償却累計額 XXX（間接法）
借方: 減価償却費 XXX  貸方: 建物 XXX（直接法）
```

### 6-6. 貸倒引当金の計上

#### 法定繰入率（中小法人の場合）
| 業種 | 法定繰入率 |
|------|-----------|
| 卸売業・小売業（飲食店業含む） | **10/1000**（1.0%） |
| 製造業 | **8/1000**（0.8%） |
| 金融・保険業 | **3/1000**（0.3%） |
| 割賦販売小売業等 | **13/1000**（1.3%） |
| その他 | **6/1000**（0.6%） |

#### 仕訳
```
借方: 貸倒引当金繰入 XXX  貸方: 貸倒引当金 XXX
```

#### 個別評価と一括評価
- **個別評価**: 回収不能の恐れがある債権を個別に評価（破産更生債権等）
- **一括評価**: 上記以外の売掛金等の債権に法定繰入率を適用

### 6-7. 棚卸資産の評価

#### 評価方法
| 方法 | 内容 |
|------|------|
| **最終仕入原価法** | 期末に最も近い仕入価格で評価（法定の方法） |
| **先入先出法** | 先に仕入れたものから先に売れたと仮定 |
| **移動平均法** | 仕入のたびに平均単価を計算 |
| **総平均法** | 期中の仕入総額÷期中の仕入総数量で平均単価を計算 |
| **売価還元法** | 売価から原価率を算定して評価 |
| **個別法** | 個々の資産ごとに取得原価で評価 |

#### 仕訳（売上原価の算定）
```
期首棚卸: 借方: 仕入 XXX / 貸方: 商品 XXX（繰越商品）
期末棚卸: 借方: 商品 XXX（繰越商品） / 貸方: 仕入 XXX
```

#### 棚卸減耗損・商品評価損
- **棚卸減耗損**: 帳簿棚卸数量 > 実地棚卸数量 の差額
- **商品評価損**: 正味売却価額が取得原価を下回る場合の評価損

### 6-8. 有価証券の評価

| 区分 | 評価方法 | 評価差額の処理 |
|------|---------|-------------|
| 売買目的有価証券 | **時価** | 損益計算書（有価証券評価損益） |
| 満期保有目的の債券 | **償却原価法** | 損益計算書 |
| 子会社・関連会社株式 | **取得原価** | 時価が著しく下落した場合は減損 |
| その他有価証券 | **時価**（中小企業は取得原価も可） | 純資産の部（その他有価証券評価差額金） |

## 7. 税効果会計の基礎

### 7-1. 繰延税金資産
- 税務上の損金算入が将来に認められるもの（一時差異）に対して計上
- **例**: 貸倒引当金の損金算入限度超過額、減価償却の償却超過額、未払事業税等
- 計算: 将来減算一時差異 × 法定実効税率

### 7-2. 繰延税金負債
- 税務上の益金算入が将来に認められるもの（一時差異）に対して計上
- **例**: その他有価証券の評価益、圧縮積立金等
- 計算: 将来加算一時差異 × 法定実効税率

### 7-3. 法定実効税率（中小法人の目安）
- 所得400万円以下: 約22%
- 所得400万円超800万円以下: 約24%
- 所得800万円超: 約34%
（法人税15%/23.2% + 住民税 + 事業税 + 地方法人税を加味した実効税率）

### 7-4. 中小企業での適用
- **中小企業の会計に関する指針**: 税効果会計の適用は「望ましい」（任意）
- **中小会計要領**: 税効果会計の適用は求めない
- 実務上、中小企業では税効果会計を適用しないケースが多い

## 8. 収益認識基準の概要

### 8-1. 新収益認識基準（企業会計基準第29号）
- 2021年4月1日以後開始する事業年度から適用（上場企業等）
- **5ステップモデル**:
  1. 契約の識別
  2. 履行義務の識別
  3. 取引価格の算定
  4. 取引価格の履行義務への配分
  5. 履行義務の充足による収益の認識

### 8-2. 中小企業の取扱い
- **中小企業の会計に関する指針**: 従来の実現主義に基づく収益認識も認められる
- **中小会計要領**: 新収益認識基準の適用は求めない
- 実務上、多くの中小企業は従来の基準（出荷基準、引渡基準等）を継続適用

### 8-3. 従来の収益認識基準
| 基準 | 内容 | 適用例 |
|------|------|--------|
| **出荷基準** | 商品を出荷した時点で売上計上 | 製造業、卸売業 |
| **引渡基準** | 商品を引き渡した時点で売上計上 | 小売業 |
| **検収基準** | 得意先が検収した時点で売上計上 | 建設業、IT業 |
| **役務完了基準** | サービスが完了した時点で売上計上 | サービス業 |
| **工事進行基準** | 工事の進捗度に応じて売上計上 | 建設業（大規模工事） |
| **割賦基準** | 代金の回収期日ごとに売上計上 | 割賦販売 |
