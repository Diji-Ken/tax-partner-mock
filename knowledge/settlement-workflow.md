# 決算・申告の標準フロー

> 税理士・会計事務所向けCRMナレッジ
> 最終更新: 2026-03-23

---

## 概要

法人の決算月から申告書提出・納税までの標準的な手順をフロー化。
CRMのタスク自動生成やチェックリスト機能の基礎データとして使用する。

### 全体タイムライン（3月決算法人の例）

```
3月末         : 決算日
4月上旬〜中旬  : ステップ1 - 決算整理仕訳
4月中旬〜下旬  : ステップ2 - 決算書作成
4月下旬〜5月中旬: ステップ3 - 法人税申告書作成
5月上旬〜中旬  : ステップ4 - 消費税申告書作成
5月上旬〜中旬  : ステップ5 - 地方税申告書作成
5月中旬〜下旬  : ステップ6 - 電子申告
5月末         : ステップ7 - 納税（申告期限）
```

## ステップ1: 決算整理仕訳

### やること

| 作業 | 内容 | 注意点 |
|------|------|--------|
| 減価償却費の計上 | 有形固定資産・無形固定資産・一括償却資産の減価償却費を計算・計上 | 耐用年数・償却方法（定額法/定率法）の確認。中小企業の少額減価償却資産の特例（30万円未満、年300万円限度）の適用判断 |
| 貸倒引当金の計上 | 売掛金・受取手形等の債権に対する貸倒引当金を設定 | 一般債権（法定繰入率）と個別債権（個別評価）の区分。中小法人の法定繰入率確認 |
| 棚卸資産の計上 | 期末商品棚卸高を確定し、棚卸資産を計上 | 実地棚卸の実施確認。評価方法（最終仕入原価法等）の確認。棚卸減耗損・商品評価損の計上 |
| 前払費用の計上 | 翌期以降に帰属する費用を前払費用として振替 | 保険料、地代家賃、リース料等の期間按分。短期前払費用の特例（1年以内、継続適用）の適用判断 |
| 未払費用の計上 | 当期に帰属するが未払いの費用を計上 | 給与・賞与の未払計上、社会保険料の未払計上、利息の未払計上 |
| 未収収益の計上 | 当期に帰属するが未収の収益を計上 | 受取利息の未収計上等 |
| 前受収益の計上 | 翌期以降に帰属する収益を前受収益として振替 | 前受家賃等 |
| 賞与引当金の計上 | 翌期支給予定の賞与のうち当期負担分を引当計上 | 支給対象期間の確認、社会保険料の見積額も考慮 |
| 退職給付引当金の計上 | 当期の退職給付費用を計上 | 中小企業退職金共済（中退共）加入の場合は掛金が費用。自社積立の場合は引当金計上 |
| 法人税等の計上 | 当期の法人税・住民税・事業税の見積額を計上 | 税引前当期純利益の確定後に計算。法人税申告書作成と並行して進める場合もある |

### 必要書類

- 固定資産台帳
- 減価償却計算書
- 棚卸表（実地棚卸の結果）
- 保険証券（保険料の期間按分用）
- 賃貸借契約書（家賃等の期間按分用）
- 借入金返済予定表（利息の未払計上用）
- 前期の決算書（引当金の前期計上額の確認）

### 注意点

| 項目 | 内容 |
|------|------|
| 税務上の限度額 | 減価償却費・貸倒引当金等は税務上の限度額を超えると損金不算入（別表調整が必要） |
| 交際費の損金不算入 | 中小法人: 年800万円まで損金算入可（又は接待飲食費の50%）。大法人: 接待飲食費の50%のみ |
| 役員報酬の注意 | 定期同額給与、事前確定届出給与、業績連動給与のいずれかに該当しない場合は損金不算入 |
| 寄附金の損金限度額 | 一般寄附金: （資本金等の額×0.25%＋所得金額×2.5%）×1/4 |

## ステップ2: 決算書作成

### やること

| 作業 | 内容 | 注意点 |
|------|------|--------|
| 貸借対照表（BS）の作成 | 資産・負債・純資産の期末残高を記載 | 流動/固定の区分、表示科目の適正性 |
| 損益計算書（PL）の作成 | 収益・費用の当期発生額を記載 | 売上原価の計算、営業外/特別の区分 |
| 株主資本等変動計算書の作成 | 純資産の変動内容を記載 | 資本金、利益剰余金、自己株式の変動 |
| 個別注記表の作成 | 重要な会計方針、後発事象等を記載 | 会社法上の開示要件を確認 |
| 勘定科目内訳明細書の作成 | 主要科目の内訳を記載（16種類） | 法人税申告書の添付書類として必要 |
| 事業概況説明書の作成 | 事業内容、主要な取引先、従業員数等を記載 | 法人税申告書の添付書類として必要 |

### 勘定科目内訳明細書（16種類）

| No. | 内訳明細書 |
|-----|-----------|
| 1 | 預貯金等の内訳書 |
| 2 | 受取手形の内訳書 |
| 3 | 売掛金（未収入金）の内訳書 |
| 4 | 仮払金（前渡金）の内訳書、貸付金及び受取利息の内訳書 |
| 5 | 棚卸資産（商品又は製品、半製品、仕掛品、原材料、貯蔵品）の内訳書 |
| 6 | 有価証券の内訳書 |
| 7 | 固定資産（土地、土地の上に存する権利及び建物に限る。）の内訳書 |
| 8 | 支払手形の内訳書 |
| 9 | 買掛金（未払金・未払費用）の内訳書 |
| 10 | 仮受金（前受金・預り金）の内訳書、源泉所得税預り金の内訳 |
| 11 | 借入金及び支払利子の内訳書 |
| 12 | 土地の売上高等の内訳書 |
| 13 | 売上高等の事業所別の内訳書 |
| 14 | 役員報酬手当等及び人件費の内訳書 |
| 15 | 地代家賃等の内訳書、工業所有権等の使用料の内訳書 |
| 16 | 雑益、雑損失等の内訳書 |

### 必要書類

- 月次試算表（12ヶ月分）
- 固定資産台帳
- 有価証券台帳
- 借入金一覧
- 役員報酬の決議書
- 株主総会議事録（前期分の利益処分案承認）

## ステップ3: 法人税申告書作成

### やること

| 作業 | 内容 | 注意点 |
|------|------|--------|
| 別表一（一）の作成 | 法人税額の計算（所得金額×税率） | 中小法人の軽減税率（年800万円以下: 15%、超: 23.2%）の適用確認 |
| 別表二の作成 | 同族会社等の判定 | 株主の持株割合の確認 |
| 別表四の作成 | 所得の金額の計算（税務調整） | 加算項目・減算項目の洗い出し |
| 別表五（一）の作成 | 利益積立金額の計算 | 前期からの繰越額の確認 |
| 別表五（二）の作成 | 租税公課の納付状況 | 各税目の中間納付額・確定額の確認 |
| 別表六の作成 | 所得税額の控除 | 受取利息・配当に係る源泉所得税の控除 |
| 別表七の作成 | 欠損金の繰越控除 | 繰越欠損金の残額確認（中小法人は全額控除可） |
| 別表八の作成 | 受取配当等の益金不算入 | 持株割合に応じた益金不算入額の計算 |
| 別表十一の作成 | 貸倒引当金の損金算入限度額 | 一括評価・個別評価の区分 |
| 別表十四の作成 | 寄附金の損金不算入額 | 限度額の計算 |
| 別表十五の作成 | 交際費等の損金不算入額 | 中小法人の800万円定額控除 or 接待飲食費50%の選択 |
| 別表十六の作成 | 減価償却資産の償却限度額 | 償却方法・耐用年数の確認 |
| 適用額明細書の作成 | 租税特別措置法の適用条文の明細 | 各種税額控除・特例の適用がある場合 |

### 主要な別表と内容

| 別表 | 名称 | 概要 |
|------|------|------|
| 別表一（一） | 各事業年度の所得に係る申告書 | 最終的な法人税額の計算。確定申告書の本体 |
| 別表二 | 同族会社等の判定に関する明細書 | 同族会社に該当するかの判定 |
| 別表四 | 所得の金額の計算に関する明細書 | 会計上の利益から税務上の所得への調整 |
| 別表五（一） | 利益積立金額及び資本金等の額の計算に関する明細書 | BS項目の税務上の簿価管理 |
| 別表五（二） | 租税公課の納付状況等に関する明細書 | 各税目の納付状況一覧 |
| 別表十五 | 交際費等の損金不算入額に関する明細書 | 交際費の損金算入限度額計算 |
| 別表十六 | 減価償却資産の償却額の計算に関する明細書 | 減価償却の税務計算 |

### 主要な税率（2026年3月現在）

| 区分 | 税率 |
|------|------|
| 法人税（普通法人） | 23.2% |
| 法人税（中小法人・年800万円以下） | 15%（租税特別措置法による軽減税率） |
| 地方法人税 | 法人税額 × 10.3% |
| 法人事業税（所得割）標準税率（年400万円以下） | 3.5% |
| 法人事業税（所得割）標準税率（年400万円超800万円以下） | 5.3% |
| 法人事業税（所得割）標準税率（年800万円超） | 7.0% |
| 特別法人事業税 | 法人事業税額 × 37% |
| 法人住民税（都道府県民税）法人税割 | 法人税額 × 1.0% |
| 法人住民税（市区町村民税）法人税割 | 法人税額 × 6.0%（標準税率） |
| 法人住民税 均等割 | 資本金等の額・従業員数により段階的（最低7万円/年） |

※税率は自治体により異なる場合がある（制限税率の範囲内で条例により超過課税あり）。

## ステップ4: 消費税申告書作成

### やること

| 作業 | 内容 | 注意点 |
|------|------|--------|
| 課税方式の確認 | 本則課税 or 簡易課税 or 2割特例 | 基準期間の課税売上高、届出書の提出状況を確認 |
| 課税売上高の集計 | 10%対象・8%軽減対象を区分して集計 | 非課税売上・免税売上・不課税売上を除外 |
| 課税仕入高の集計 | 税率区分別に集計（本則課税の場合） | インボイスの保存状況確認。経過措置（80%/50%）の適用判定 |
| 課税売上割合の計算 | 課税売上高 ÷（課税売上高 + 非課税売上高） | 95%以上なら全額控除可（課税売上高5億円以下の場合） |
| 仕入控除税額の計算 | 全額控除 or 個別対応方式 or 一括比例配分方式 | 課税売上割合が95%未満、又は課税売上高5億円超の場合に方式選択 |
| 申告書の作成 | 消費税及び地方消費税の確定申告書 | 付表1〜付表2の作成 |

### 簡易課税のみなし仕入率

| 事業区分 | 該当事業 | みなし仕入率 |
|---------|---------|-------------|
| 第一種事業 | 卸売業 | 90% |
| 第二種事業 | 小売業、農林水産業（飲食料品の譲渡） | 80% |
| 第三種事業 | 製造業、建設業、農林水産業（上記以外）等 | 70% |
| 第四種事業 | 飲食店業等、第一種〜第三種・第五種・第六種以外 | 60% |
| 第五種事業 | サービス業（飲食店業を除く）、運輸通信業、金融保険業 | 50% |
| 第六種事業 | 不動産業 | 40% |

### 必要書類

- 課税売上高の集計表（税率別）
- 課税仕入高の集計表（税率別）
- 適格請求書（インボイス）の保存確認
- 消費税の届出書（課税事業者選択届、簡易課税制度選択届等）の控え
- 前期の消費税確定申告書（中間申告額の確認）

## ステップ5: 地方税申告書作成

### やること

| 作業 | 内容 | 注意点 |
|------|------|--------|
| 法人事業税の申告書作成 | 所得割・付加価値割・資本割の計算 | 資本金1億円超は外形標準課税の対象 |
| 特別法人事業税の計算 | 法人事業税額 × 37% | 事業税と合わせて申告 |
| 都道府県民税の申告書作成 | 法人税割（法人税額×税率）＋ 均等割 | 税率は都道府県により異なる |
| 市区町村民税の申告書作成 | 法人税割（法人税額×税率）＋ 均等割 | 税率は市区町村により異なる。事業所が複数自治体にある場合は分割申告 |
| 均等割の計算 | 資本金等の額と従業員数で判定 | 赤字でも均等割は発生 |

### 均等割額（都道府県民税 + 市区町村民税の合計目安）

| 資本金等の額 | 従業員50人以下 | 従業員50人超 |
|-------------|---------------|-------------|
| 1,000万円以下 | 7万円 | 14万円 |
| 1,000万円超〜1億円以下 | 18万円 | 20万円 |
| 1億円超〜10億円以下 | 29万円 | 53万円 |

※金額は標準税率ベースの概算。自治体により異なる。

### 必要書類

- 法人税の確定申告書（別表一）
- 事業所一覧（複数事業所がある場合）
- 従業員数
- 前期の地方税確定申告書

## ステップ6: 電子申告（e-Tax / eLTAX）

### やること

| 作業 | 内容 | 注意点 |
|------|------|--------|
| e-Tax（国税）での申告 | 法人税・消費税・地方法人税の電子申告 | 電子証明書（マイナンバーカード or 法人の代表者の電子証明書 or 税理士の電子証明書）が必要 |
| eLTAX（地方税）での申告 | 法人事業税・法人住民税の電子申告 | 利用届出が必要。PCdeskまたは対応ソフトを使用 |
| 電子申告の送信 | 申告データの送信・受付確認 | 送信後に受信通知（受付番号・受付日時）を確認・保存 |
| 添付書類の送信 | 勘定科目内訳明細書、事業概況説明書等 | e-Taxでは一定の添付書類も電子送信可能 |

### 大法人の電子申告義務化

| 対象 | 内容 |
|------|------|
| 対象法人 | 事業年度開始時の資本金の額が1億円超の法人、相互会社、投資法人、特定目的会社 |
| 対象税目 | 法人税、消費税、法人住民税、法人事業税 |
| 義務化時期 | 2020年4月1日以後開始事業年度から |
| 罰則 | 電子申告によらない場合、申告書の提出がなかったものとみなされる（無申告扱い）可能性あり |

### 必要な環境

| 項目 | 内容 |
|------|------|
| 電子証明書 | マイナンバーカード（個人）、商業登記電子証明書（法人）、税理士用電子証明書 |
| ICカードリーダー | マイナンバーカードの場合に必要 |
| 申告ソフト | 市販の税務申告ソフト（達人、TKC等）またはe-Tax/eLTAXの公式ソフト |
| インターネット環境 | 安定した回線 |

## ステップ7: 納税

### やること

| 作業 | 内容 | 注意点 |
|------|------|--------|
| 納付税額の確定 | 法人税・消費税・地方税の確定納付額を算出 | 中間納付額がある場合は差引計算 |
| 納付方法の選択 | 下記の納付方法から選択 | 顧問先の希望・口座状況を確認 |
| 納付の実行 | 期限内に納付 | 期限は原則として申告期限と同日 |
| 納付書の控え保存 | 納付書の控え又は納付確認書の保存 | 電子納税の場合はメール通知等を保存 |

### 納付方法一覧

| 方法 | 概要 | メリット | 注意点 |
|------|------|---------|--------|
| ダイレクト納付 | e-Taxから指定口座で直接納付（事前届出要） | 金融機関に行かなくてよい。期日指定可能 | 事前に届出書の提出と利用登録が必要（届出から利用可能まで約1ヶ月） |
| インターネットバンキング（Pay-easy） | ネットバンキングで納付 | 金融機関に行かなくてよい | ネットバンキング契約が必要。納付情報の登録が必要 |
| クレジットカード納付 | 国税クレジットカードお支払サイトで納付 | 分割払い可能、ポイント付与（カードによる） | 決済手数料が発生（税額1万円ごとに83円程度）。納税額が1,000万円未満に限る |
| スマートフォンアプリ納付 | スマホの決済アプリ（PayPay等）で納付 | 手軽 | 納付額30万円以下に限る |
| コンビニ納付（QRコード） | QRコードを使ってコンビニで納付 | 金融機関の営業時間外でも納付可 | 納付額30万円以下に限る |
| 振替納税 | 指定口座から自動引落し | 手続き不要（申告のみ） | 法人は利用不可（個人のみ）。振替日は申告期限の約1ヶ月後 |
| 窓口納付 | 金融機関又は税務署の窓口で納付書により納付 | 確実 | 営業時間内に限る。納付書の準備が必要 |

### 還付の場合

| 項目 | 内容 |
|------|------|
| 還付の発生 | 中間納付額 > 確定税額 の場合、差額が還付される |
| 還付加算金 | 還付金には一定の利子相当額（還付加算金）が付される |
| 還付時期 | 申告から約1〜2ヶ月後（e-Tax申告の場合は早期還付の特例あり: 約3週間） |
| 還付先口座 | 申告書の「還付金の受取場所」欄に記載した口座に振り込まれる |
