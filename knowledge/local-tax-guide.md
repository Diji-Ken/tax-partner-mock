# 地方税 完全ガイド

> 最終更新: 2026年3月
> 根拠法令: 地方税法、各自治体条例
> 本ガイドは標準税率に基づく。自治体によって超過税率を適用する場合がある

---

## 1. 法人住民税

法人住民税は「法人税割」と「均等割」の2つで構成される。

### 1-1. 法人税割

法人税額を課税標準として課される。

**標準税率**:

| 課税主体 | 標準税率 | 制限税率 |
|---------|---------|---------|
| 都道府県民税 | **1.0%** | 2.0% |
| 市区町村民税 | **6.0%** | 8.4% |
| **合計** | **7.0%** | 10.4% |

**注意**:
- 東京都特別区は都民税として法人税割7.0%（都道府県分1.0% + 市町村分6.0%相当を一括）
- 超過税率を採用している自治体が多い（東京都特別区: 都民税の法人税割は超過税率を適用）
- 法人税額が基準（法人税額 × 税率）

### 1-2. 均等割

法人の資本金等の額と従業者数に基づいて課される定額の税金。赤字でも課税される。

#### 都道府県民税の均等割

| 資本金等の額 | 年額 |
|-------------|------|
| 1,000万円以下 | **20,000円** |
| 1,000万円超〜1億円以下 | **50,000円** |
| 1億円超〜10億円以下 | **130,000円** |
| 10億円超〜50億円以下 | **540,000円** |
| 50億円超 | **800,000円** |

#### 市区町村民税の均等割（9段階）

| 資本金等の額 | 従業者数50人以下 | 従業者数50人超 |
|-------------|----------------|--------------|
| 1,000万円以下 | **50,000円** | **120,000円** |
| 1,000万円超〜1億円以下 | **130,000円** | **150,000円** |
| 1億円超〜10億円以下 | **160,000円** | **400,000円** |
| 10億円超〜50億円以下 | **410,000円** | **1,750,000円** |
| 50億円超 | **410,000円** | **3,000,000円** |

**注意**:
- 均等割は事業年度の月数に応じて月割計算（1ヶ月未満の端数は1ヶ月に切り上げ）
- 均等割の月割計算: 年額 × 事業年度の月数 / 12
- 設立初年度や解散年度は月割りとなる
- 複数の都道府県・市区町村に事務所等がある場合は、それぞれに均等割が課される
- 東京都特別区の場合は都民税として均等割を納付

#### 均等割の合計額（都道府県 + 市区町村）の目安

| 資本金等の額 | 従業者数50人以下 | 従業者数50人超 |
|-------------|----------------|--------------|
| 1,000万円以下 | **70,000円** | **140,000円** |
| 1,000万円超〜1億円以下 | **180,000円** | **200,000円** |
| 1億円超〜10億円以下 | **290,000円** | **530,000円** |
| 10億円超〜50億円以下 | **950,000円** | **2,290,000円** |
| 50億円超 | **1,210,000円** | **3,800,000円** |

## 2. 法人事業税

### 2-1. 所得割（普通法人の場合）

**標準税率**（資本金1億円以下の普通法人）:

| 所得金額 | 税率 |
|---------|------|
| 400万円以下 | **3.5%** |
| 400万円超〜800万円以下 | **5.3%** |
| 800万円超 | **7.0%** |

**注意**:
- 3以上の都道府県に事務所等を設けている法人（資本金1,000万円以上）は軽減税率不適用
- 外形標準課税対象法人（資本金1億円超）は別途税率

**外形標準課税対象法人の所得割税率**:

| 所得金額 | 税率 |
|---------|------|
| 400万円以下 | **1.18%** |
| 400万円超〜800万円以下 | **1.78%** |
| 800万円超 | **2.35%** |

### 2-2. 外形標準課税（資本金1億円超の法人）

資本金1億円超の普通法人に対して、所得割に加えて付加価値割と資本割が課される。

| 税目 | 課税標準 | 標準税率 |
|------|---------|---------|
| **付加価値割** | 付加価値額（報酬給与額 + 純支払利子 + 純支払賃借料 + 単年度損益） | **1.2%** |
| **資本割** | 資本金等の額 | **0.5%** |
| **所得割** | 所得金額 | 上記の軽減税率 |

**付加価値額の計算**:
```
付加価値額 = 収益配分額 + 単年度損益
収益配分額 = 報酬給与額 + 純支払利子 + 純支払賃借料
```

- 報酬給与額: 役員報酬、給料、賃金、賞与、退職給与、その他これらに類するもの
- 純支払利子: 支払利子 - 受取利子（マイナスの場合は0）
- 純支払賃借料: 支払賃借料 - 受取賃借料（マイナスの場合は0）

**雇用安定控除**: 収益配分額のうち報酬給与額の占める割合が70%を超える場合、一定の控除あり

### 2-3. 特別法人事業税

法人事業税（所得割又は収入割）の額を課税標準として、国税として課される。都道府県が法人事業税と併せて徴収し、国に払い込む。

**税率**:

| 対象法人 | 税率 |
|---------|------|
| 外形標準課税対象法人 | **260%** |
| 外形標準課税対象外法人（所得割を申告する法人） | **37%** |
| 特別法人（収入割を申告する法人） | **30%** |

**計算例**（資本金1億円以下、所得1,000万円の場合）:
```
法人事業税（所得割）:
  400万円 × 3.5% = 140,000円
  400万円 × 5.3% = 212,000円
  200万円 × 7.0% = 140,000円
  合計: 492,000円

特別法人事業税:
  492,000円 × 37% = 182,040円

事業税合計: 492,000円 + 182,040円 = 674,040円
```

### 2-4. 法人事業税の申告・納付

- 確定申告: 事業年度終了の日の翌日から2ヶ月以内
- 中間申告: 事業年度開始の日以後6ヶ月を経過した日から2ヶ月以内
- 法人住民税と同時に申告（第6号様式等）

## 3. 個人住民税

### 3-1. 構成

個人住民税は「所得割」と「均等割」で構成される。

| 区分 | 都道府県民税 | 市区町村民税 | 合計 |
|------|------------|------------|------|
| **所得割** | 4% | 6% | **10%** |
| **均等割** | 1,500円 | 3,500円 | **5,000円** |

**注意**:
- 均等割5,000円の内訳: 2024年度から森林環境税1,000円が加わり、復興増税分（各500円）が終了。結果として合計5,000円は変わらない（都道府県1,500円 + 市区町村3,500円 = 5,000円）
- 2024年度以降: 森林環境税1,000円（国税）が均等割と併せて徴収される

### 3-2. 課税方式

前年の所得に対して翌年度に課税（前年所得課税）

**非課税の基準**（自治体により異なる。以下は一般的な基準）:
- 均等割・所得割ともに非課税: 生活保護受給者、障害者・未成年者・寡婦・ひとり親で前年の合計所得金額が135万円以下
- 均等割非課税: 前年の合計所得金額が一定額以下（扶養親族なしの場合、概ね45万円以下。自治体による）
- 所得割非課税: 前年の総所得金額等が一定額以下（扶養親族なしの場合、概ね45万円以下。自治体による）

### 3-3. 所得割の計算

```
所得割額 = （前年の総所得金額等 - 所得控除額）× 10% - 税額控除額
```

**所得控除**: 所得税の所得控除と概ね同じだが、控除額が異なるものがある

| 控除 | 所得税 | 住民税 |
|------|--------|--------|
| 基礎控除 | 48万円 | 43万円 |
| 配偶者控除（一般） | 38万円 | 33万円 |
| 扶養控除（一般） | 38万円 | 33万円 |
| 特定扶養控除 | 63万円 | 45万円 |
| 生命保険料控除上限 | 12万円 | 7万円 |
| 地震保険料控除上限 | 5万円 | 2.5万円 |

### 3-4. 徴収方法

| 方法 | 対象者 | 徴収時期 |
|------|--------|---------|
| **特別徴収**（給与天引き） | 給与所得者 | 6月〜翌年5月の12回に分割 |
| **普通徴収**（自分で納付） | 個人事業主等 | 6月・8月・10月・翌年1月の4回 |
| **年金からの特別徴収** | 65歳以上の年金受給者 | 年金支給時に天引き |

### 3-5. 調整控除

所得税と住民税の人的控除額の差に基づく控除（税源移譲に伴う負担増を調整）

合計所得金額200万円以下:
```
調整控除額 = 次のいずれか少ない金額 × 5%
(1) 人的控除額の差の合計額
(2) 合計課税所得金額
```

合計所得金額200万円超:
```
調整控除額 = {人的控除額の差の合計額 -（合計課税所得金額 - 200万円）} × 5%
（ただし、2,500円未満の場合は2,500円）
```

```
事業税額 = （事業所得 + 青色申告特別控除額 - 事業主控除290万円）× 税率
```

**注意点**:
- 事業所得から青色申告特別控除額は**加算**して計算する（事業税では青色申告特別控除を認めない）

- 損失の繰越控除あり（青色申告: 3年間）
- 事業税は翌年の所得税の必要経費に算入可能

- スポーツ選手
- その他法定業種に該当しない事業

### 4-5. 申告・納付

- 納付時期: 8月・11月の2回（都道府県税事務所から納税通知書が送付される）
- 第1期: 8月31日まで
- 第2期: 11月30日まで

## 5. 固定資産税（償却資産税）

### 5-1. 概要

事業用の有形減価償却資産（償却資産）に対して、市区町村が課する固定資産税。

### 5-2. 課税対象

| 課税対象 | 具体例 |
|---------|--------|
| 構築物 | 舗装路面、門、塀、広告塔、受変電設備等 |
| 機械及び装置 | 各種製造設備、建設機械、機械式駐車設備等 |
| 船舶 | ボート、遊覧船等 |
| 航空機 | 飛行機、ヘリコプター等 |
| 車両及び運搬具 | 大型特殊自動車、構内運搬車等（自動車税・軽自動車税の対象を除く） |
| 工具・器具及び備品 | パソコン、コピー機、机、椅子、エアコン、看板等 |

**課税対象外**:
- 土地・家屋（別途固定資産税が課税）
- 自動車税・軽自動車税の対象となる車両
- 無形減価償却資産（ソフトウェア、特許権等）
- 繰延資産
- 少額減価償却資産で一時に損金算入したもの（取得価額10万円未満）
- 一括償却資産として3年均等償却しているもの（取得価額20万円未満）

**注意**: 中小企業者の少額減価償却資産の特例（取得価額30万円未満）により即時償却したものは償却資産税の課税対象

### 5-3. 税率・免税点

| 項目 | 内容 |
|------|------|
| 税率 | **1.4%**（標準税率） |
| 免税点 | 課税標準額の合計が**150万円未満** |

### 5-4. 課税標準（評価額）

旧定率法（取得価額 × 残存率）により計算

```
初年度: 取得価額 × (1 - 減価率/2)  ※半年償却
2年目以降: 前年度評価額 × (1 - 減価率)
```

- 最低限度: 取得価額の5%（評価額が取得価額の5%を下回る場合は5%で据え置き）
- 耐用年数は法人税法の耐用年数に準拠

### 5-5. 申告・納付

| 項目 | 内容 |
|------|------|
| 申告期限 | **毎年1月31日**（1月1日現在の所有状況を申告） |
| 申告先 | 資産の所在する市区町村 |
| 納付時期 | 市区町村が決定（通常4月・7月・12月・翌2月の4回、又は一括） |
| 納税通知書 | 4〜6月頃に送付 |

**申告書の記載事項**:
- 資産の種類、名称、取得年月、取得価額、耐用年数
- 前年中に取得した資産、減少した資産を報告

### 5-6. 特例措置

- 中小企業等経営強化法に基づく先端設備等導入計画に係る償却資産: 課税標準を3年間1/2又はゼロに軽減（自治体の条例による）
- 再生可能エネルギー発電設備: 一定期間課税標準を軽減

## 6. 固定資産税・都市計画税（土地・家屋）

### 6-1. 固定資産税

| 項目 | 内容 |
|------|------|

| 課税標準 | 固定資産税評価額 |

|------|------|

| 課税標準 | 固定資産税評価額 |

**住宅用地の特例**:
- 小規模住宅用地: 課税標準 × 1/3
- 一般住宅用地: 課税標準 × 2/3

**住宅の軽減**: 新築住宅は課税標準から1,200万円控除（認定長期優良住宅は1,300万円）

**土地の軽減**: 住宅用土地は以下のいずれか多い方を税額から控除
- 45,000円
- 土地1平方メートルあたりの評価額 × 1/2 × 住宅の床面積の2倍（200平方メートル限度）× 3%

## 8. eLTAX（地方税ポータルシステム）

### 8-1. 概要

地方税の申告・申請・届出等をインターネットで行うことができるシステム。

### 8-2. 対応税目

- 法人住民税
- 法人事業税・特別法人事業税
- 個人住民税（特別徴収関連）
- 固定資産税（償却資産申告）
- 事業所税

### 8-3. 利用手順

1. **利用届出**: eLTAXのWebサイトから利用届出書を提出（電子証明書が必要）
2. **PCdesk（クライアント版又はWeb版）のインストール**: 申告データの作成・送信に使用
3. **電子証明書の取得**: 商業登記に基づく電子証明書、マイナンバーカード等
4. **申告データの作成**: PCdeskで申告書データを作成
5. **電子署名の付与**: 電子証明書で署名
6. **送信**: eLTAXに送信
7. **受付通知の確認**: 送信後に受付通知をメッセージボックスで確認

### 8-4. 電子納税

eLTAXを通じて以下の方法で電子納税が可能:
- ダイレクト納付（口座振替）
- インターネットバンキング（ペイジー）
- クレジットカード納付（一部自治体）

### 8-5. 特別徴収税額通知の電子化

個人住民税の特別徴収税額の決定通知書について、eLTAXでの電子受取が可能（順次拡大中）

## 9. 法人住民税・法人事業税の申告スケジュール

### 9-1. 確定申告

| 項目 | 期限 |
|------|------|
| 申告期限 | 事業年度終了の日の翌日から2ヶ月以内 |
| 申告の延長 | 法人税の申告期限延長の特例を受けている場合、1ヶ月の延長可（ただし見込納付が必要） |

### 9-2. 中間申告

| 項目 | 期限 |
|------|------|
| 申告期限 | 事業年度開始の日以後6ヶ月を経過した日から2ヶ月以内 |
| 予定申告 | 前期の税額の1/2 |
| 仮決算 | 6ヶ月間の仮決算に基づく申告も可 |

### 9-3. 修正申告・更正の請求

| 項目 | 内容 |
|------|------|
| 修正申告 | 申告額が過少であった場合、いつでも可能 |
| 更正の請求 | 申告額が過大であった場合、法定申告期限から5年以内 |
