# 固定資産税・償却資産税ガイド

## 1. 固定資産税の仕組み

### 1-1. 基本構造
```
固定資産税額 = 課税標準額 × 税率（標準税率: 1.4%）
```

- **課税主体**: 市区町村（東京23区は都）
- **課税対象**: 土地、家屋、償却資産
- **賦課期日**: 毎年**1月1日**（1月1日時点の所有者に課税）
- **納税義務者**: 1月1日時点の固定資産の所有者（登記簿上の所有者）
- **税率**: 標準税率**1.4%**（市区町村が条例で変更可能、超過税率の上限なし）
- **都市計画税**: 都市計画区域内の土地・家屋に別途課税（制限税率**0.3%**）

### 1-2. 納付時期
- 通常年4回の分割納付（4月・7月・12月・2月が一般的だが市区町村により異なる）
- 東京23区: 6月・9月・12月・2月
- 一括納付も可能（割引はない場合が多い）

### 1-3. 課税標準額の算定
- **土地**: 固定資産税評価額（地価公示価格の**70%**が目安）
- **家屋**: 固定資産税評価額（再建築価格 × 経年減点補正率）
- **償却資産**: 取得価額 × 残価率（定率法に類似した計算）

---

## 2. 評価替え

### 2-1. 評価替えの仕組み
- **3年ごと**に固定資産の評価額を見直す（評価替え）
- 直近の基準年度: **2024年度**（令和6年度）、次回: **2027年度**（令和9年度）
- 基準年度以外の年度は、原則として基準年度の価格を据え置き

### 2-2. 土地の評価
- 路線価方式（市街地）または標準地比準方式（農村部等）
- 固定資産税路線価は地価公示価格の約70%
- 地価の下落により前年度の評価額を下回る場合は、据置年度でも下落修正あり

### 2-3. 家屋の評価
- **再建築価格方式**: 同一の建物を新築した場合の建築費 × 経年減点補正率
- 経年減点補正率: 経過年数に応じて減少（最低20%まで）
- 増改築した場合はその部分の評価額が加算される

---

## 3. 免税点

### 3-1. 免税点の基準
同一の市区町村内で同一の者が所有する固定資産について、課税標準額の合計が以下の金額に満たない場合は固定資産税が課税されない。

| 資産の種類 | 免税点 |
|-----------|--------|
| **土地** | **30万円** |
| **家屋** | **20万円** |
| **償却資産** | **150万円** |

### 3-2. 注意点
- 複数の市区町村に資産がある場合は、市区町村ごとに判定
- 共有名義の場合は、共有者全員の持分を合算して判定
- 免税点は「課税標準額」で判定（評価額ではない）

---

## 4. 住宅用地の特例

### 4-1. 小規模住宅用地（200㎡以下の部分）
```
課税標準額 = 評価額 × 1/6
```

### 4-2. 一般住宅用地（200㎡超の部分）
```
課税標準額 = 評価額 × 1/3
```

### 4-3. 都市計画税の特例
- 小規模住宅用地: 評価額 × **1/3**
- 一般住宅用地: 評価額 × **2/3**

### 4-4. 計算例
**敷地面積300㎡、評価額3,000万円の住宅用地の場合**:

| 区分 | 面積 | 評価額按分 | 特例 | 課税標準額 |
|------|------|-----------|------|-----------|
| 小規模住宅用地 | 200㎡ | 3,000万円 × 200/300 = 2,000万円 | × 1/6 | 約333万円 |
| 一般住宅用地 | 100㎡ | 3,000万円 × 100/300 = 1,000万円 | × 1/3 | 約333万円 |
| **合計** | 300㎡ | - | - | **約666万円** |

- 固定資産税: 666万円 × 1.4% = **約93,000円**
- 特例なしの場合: 3,000万円 × 1.4% = **420,000円**
- **節税効果: 約327,000円**

### 4-5. 住宅用地の認定要件
- 住宅（居住用家屋）の敷地であること
- 住宅の床面積の10倍まで（それを超える部分は特例対象外）
- 空き家の場合: 住宅が存在する限り適用（ただし特定空家に指定されると適用除外の場合あり）

---

## 5. 新築住宅の軽減

### 5-1. 一般の新築住宅
| 項目 | 内容 |
|------|------|
| 軽減期間 | 新築後**3年間** |
| 軽減内容 | 固定資産税額の**1/2** |
| 対象面積 | 居住部分の床面積**120㎡**まで |
| 適用要件 | 居住部分の床面積が50㎡以上280㎡以下（貸家は40㎡以上） |

### 5-2. 中高層耐火建築物（マンション等、3階建以上）
| 項目 | 内容 |
|------|------|
| 軽減期間 | 新築後**5年間** |
| 軽減内容 | 固定資産税額の**1/2** |
| 対象面積 | 居住部分の床面積**120㎡**まで |
| 適用要件 | 同上 |

### 5-3. 認定長期優良住宅
| 項目 | 内容 |
|------|------|
| 軽減期間 | 一般: **5年間**、中高層: **7年間** |
| 軽減内容 | 固定資産税額の**1/2** |

### 5-4. 計算例
**新築一戸建て、固定資産税評価額1,500万円の場合**:
- 通常: 1,500万円 × 1.4% = 210,000円
- 軽減後: 210,000円 × 1/2 = **105,000円**（3年間）

## 6. 償却資産税

### 6-1. 償却資産税とは
- 事業用の機械装置、器具備品、構築物等の償却資産に対して課税される固定資産税の一種
- 課税主体は市区町村
- 税率: **1.4%**（固定資産税と同じ標準税率）

### 6-2. 申告対象資産

| 資産区分 | 具体例 |
|---------|--------|
| **構築物** | 舗装路面、門、塀、看板、外構、受変電設備、内装（賃借物件の内装造作）等 |
| **機械及び装置** | 各種製造設備、工作機械、印刷機械、建設機械等 |
| **船舶** | ボート、漁船等 |
| **航空機** | 飛行機、ヘリコプター等 |
| **車両及び運搬具** | 大型特殊自動車（ブルドーザー等）、構内運搬車、フォークリフト（小型特殊以外）等 |
| **工具、器具及び備品** | パソコン、コピー機、事務机、椅子、エアコン、冷蔵庫、陳列棚、看板、金型等 |

### 6-3. 申告対象外（非対象）の資産

| 非対象資産 | 理由 |
|-----------|------|
| **自動車税・軽自動車税の課税対象車両** | 別途自動車税等が課税されるため |
| **無形固定資産**（ソフトウェア、特許権等） | 償却資産税の対象外 |
| **繰延資産** | 償却資産に該当しない |
| **建物**（家屋として課税されるもの） | 固定資産税の家屋として課税 |
| **建物附属設備で家屋と一体のもの** | 家屋の評価に含まれる（電気・給排水・エレベーター等） |
| **リース資産**（所有権がリース会社にあるもの） | リース会社が申告義務者 |
| **取得価額10万円未満の資産（全額費用処理したもの）** | 少額資産として対象外 |
| **一括償却資産（3年均等償却を選択したもの）** | 申告対象外 |

**注意**: 少額減価償却資産の特例（30万円未満の即時償却）を適用した資産は**申告対象**

### 6-4. 評価額の計算方法

#### 取得初年度
```
評価額 = 取得価額 × (1 - 減価率/2)
```
- 半年償却（取得年度は半年分の減価）

#### 2年目以降
```
評価額 = 前年度評価額 × (1 - 減価率)
```

#### 減価率（主な耐用年数の減価率）
| 耐用年数 | 減価率 |
|---------|--------|
| 2年 | 0.684 |
| 3年 | 0.536 |
| 4年 | 0.438 |
| 5年 | 0.369 |
| 6年 | 0.319 |
| 7年 | 0.280 |
| 8年 | 0.250 |
| 10年 | 0.206 |
| 15年 | 0.142 |
| 20年 | 0.109 |

#### 最低評価額
- 評価額の下限: 取得価額の**5%**（これ以下にはならない）
- 5%に達した後も、資産が存在する限り申告・課税が続く

#### 計算例
**取得価額100万円、耐用年数5年（減価率0.369）のパソコン**:

| 年度 | 計算 | 評価額 |
|------|------|--------|
| 1年目 | 1,000,000 × (1 - 0.369/2) = 1,000,000 × 0.8155 | 815,500円 |
| 2年目 | 815,500 × (1 - 0.369) = 815,500 × 0.631 | 514,581円 |
| 3年目 | 514,581 × 0.631 | 324,701円 |
| 4年目 | 324,701 × 0.631 | 204,886円 |
| 5年目 | 204,886 × 0.631 | 129,283円 |
| 6年目 | 129,283 × 0.631 | 81,578円 |
| 7年目以降 | 最低評価額 | 50,000円（取得価額の5%） |

- 税額 = 評価額 × 1.4%
- 1年目: 815,500 × 1.4% = **11,417円**

### 6-5. 申告期限
- **毎年1月31日**まで（1月1日現在の償却資産を申告）
- 期限が土日祝の場合は翌営業日
- 新規取得・売却・移転等があった場合も含めて毎年申告が必要

### 6-6. 申告書の書き方

#### 申告書の種類
1. **償却資産申告書（償却資産課税台帳）**: 総括表
2. **種類別明細書（増加資産・全資産用）**: 新規取得資産の明細
3. **種類別明細書（減少資産用）**: 売却・廃棄資産の明細

#### 記載事項
- 資産の名称・数量
- 取得年月
- 取得価額
- 耐用年数
- 増加事由（新規取得、移転等）
- 減少事由（売却、廃棄、移転等）

#### 申告方法
- 書面申告（窓口提出・郵送）
- 電子申告（eLTAX）

#### 申告漏れの場合
- 過去5年分に遡って課税される
- 正当な理由なく申告しなかった場合は過料（10万円以下）の可能性

## 7. 固定資産税の縦覧制度と審査請求

### 7-1. 縦覧制度
| 項目 | 内容 |
|------|------|
| 目的 | 自己の土地・家屋の評価額が適正かどうか、他の土地・家屋の評価額と比較して確認する |
| 縦覧期間 | 毎年4月1日〜最初の納期限の日まで（市区町村により異なる） |
| 縦覧場所 | 市区町村の窓口 |
| 縦覧できる人 | 固定資産税の納税者（土地の納税者は土地の価格帳簿を、家屋の納税者は家屋の価格帳簿を縦覧可能） |
| 費用 | 無料 |

### 7-2. 固定資産課税台帳の閲覧
| 項目 | 内容 |
|------|------|
| 目的 | 自己の固定資産の評価額・課税標準額等を確認する |
| 閲覧期間 | 通年（開庁時間内） |
| 閲覧できる人 | 所有者本人、借地人・借家人（賃借物件に関する部分のみ）、代理人 |
| 費用 | 市区町村により異なる（無料〜数百円） |

### 7-3. 審査請求
| 項目 | 内容 |
|------|------|
| 請求先 | 固定資産評価審査委員会（各市区町村に設置） |
| 請求期限 | 納税通知書の交付を受けた日後3ヶ月以内（基準年度の場合） |
| 審査対象 | 固定資産の**評価額**に不服がある場合 |
| 注意 | 税率や課税標準の特例に関する不服は、市区町村長に対する審査請求 |
| 結果への不服 | 裁判所に取消訴訟を提起（裁決の翌日から6ヶ月以内） |

## 8. 固定資産税・償却資産税の実務チェックリスト

### 年次業務
- [ ] 1月1日時点の償却資産の棚卸し
- [ ] 償却資産申告書の作成・提出（1月31日まで）
- [ ] 固定資産税の納税通知書の確認（4〜5月頃届く）
- [ ] 縦覧期間内の評価額の確認（必要に応じて）
- [ ] 固定資産税の分割納付（各期限までに納付）
- [ ] 新規取得・売却資産の台帳への反映

### 確認ポイント
- [ ] 取得価額10万円未満の資産を誤って申告していないか
- [ ] 一括償却資産（3年均等償却）を誤って申告していないか
- [ ] 少額減価償却資産の特例適用資産を申告に含めているか
- [ ] リース資産で自社申告が必要なもの（所有権移転ファイナンスリース等）がないか
- [ ] 賃借物件の内装造作等を申告に含めているか
- [ ] 建物附属設備で家屋評価に含まれているものを二重申告していないか
- [ ] 除却・売却した資産を減少資産として申告しているか
- [ ] 他の市区町村に移転した資産がないか
