# 所得税法 完全ガイド

> 最終更新: 2026年3月
> 根拠法令: 所得税法、所得税法施行令、租税特別措置法
> 本ガイドは令和7年（2025年）分以降の申告に対応

---

## 1. 所得の10区分と計算方法

所得税法は所得を以下の10種類に区分し、それぞれ異なる計算方法を適用する。

### 1-1. 利子所得（所得税法第23条）

**定義**: 預貯金・公社債の利子、合同運用信託・公社債投資信託の収益分配金

**計算式**:
```
利子所得 = 収入金額（源泉徴収前の金額）
```
- 必要経費の控除なし
- 原則として**源泉分離課税**（所得税15% + 復興特別所得税0.315% + 住民税5% = 合計20.315%）
- 特定公社債等の利子は申告分離課税の選択可

### 1-2. 配当所得（所得税法第24条）

**定義**: 法人から受ける剰余金の配当、利益の配当、剰余金の分配、基金利息、投資信託（公社債投資信託を除く）の収益分配金

**計算式**:
```
配当所得 = 収入金額 - 株式等を取得するための負債の利子
```

**課税方式**（選択制）:

| 区分 | 課税方式 | 税率 |
|------|---------|------|
| 上場株式等の配当（大口株主除く） | 申告不要（源泉徴収で完結） | 20.315% |
| 上場株式等の配当（大口株主除く） | 総合課税（配当控除あり） | 累進税率 |
| 上場株式等の配当（大口株主除く） | 申告分離課税 | 20.315% |
| 非上場株式の配当 | 総合課税（配当控除あり） | 累進税率 |
| 非上場株式の少額配当（10万円×配当計算期間月数/12以下） | 申告不要（源泉徴収で完結） | 20.42% |

- 大口株主: 発行済株式総数の3%以上を保有する個人

### 1-3. 不動産所得（所得税法第26条）

**定義**: 不動産、不動産の上に存する権利、船舶又は航空機の貸付けによる所得（事業所得・譲渡所得に該当するものを除く）

**計算式**:
```
不動産所得 = 総収入金額 - 必要経費
```

**総収入金額の例**: 家賃、地代、権利金、更新料、敷金・保証金のうち返還不要部分、共益費

**必要経費の例**: 固定資産税、損害保険料、減価償却費、修繕費、借入金利子、管理費、管理会社への委託費

**注意**:
- 事業的規模（5棟10室基準）の場合: 青色申告特別控除65万円適用可、資産損失の全額経費算入、未回収家賃の貸倒損失計上可
- 事業的規模でない場合: 青色申告特別控除は10万円まで、資産損失は不動産所得の金額を限度

### 1-4. 事業所得（所得税法第27条）

**定義**: 農業、漁業、製造業、卸売業、小売業、サービス業その他の事業から生ずる所得

**計算式**:
```
事業所得 = 総収入金額 - 必要経費
```

**総収入金額**: 売上高、雑収入、事業に付随する収入

**必要経費**: 売上原価、給料賃金、地代家賃、減価償却費、旅費交通費、通信費、水道光熱費、消耗品費、接待交際費、広告宣伝費、専従者給与（青色申告の場合）等

### 1-5. 給与所得（所得税法第28条）

**定義**: 俸給、給料、賃金、歳費及び賞与並びにこれらの性質を有する給与

**計算式**:
```
給与所得 = 収入金額 - 給与所得控除額
```

**給与所得控除額の速算表**（令和2年分以降）:

| 給与等の収入金額 | 給与所得控除額 |
|----------------|--------------|
| 1,625,000円以下 | 550,000円 |
| 1,625,001円 〜 1,800,000円 | 収入金額 × 40% - 100,000円 |
| 1,800,001円 〜 3,600,000円 | 収入金額 × 30% + 80,000円 |
| 3,600,001円 〜 6,600,000円 | 収入金額 × 20% + 440,000円 |
| 6,600,001円 〜 8,500,000円 | 収入金額 × 10% + 1,100,000円 |
| 8,500,001円以上 | 1,950,000円（上限） |

- 最低保障額: 550,000円
- 給与所得控除後の金額がマイナスの場合は0円

**所得金額調整控除**（令和2年分以降）:
- 給与収入850万円超で、本人が特別障害者、23歳未満の扶養親族あり、又は特別障害者の同一生計配偶者・扶養親族がいる場合
- 控除額 = （収入金額（上限1,000万円）- 850万円）× 10%

**特定支出控除**: 給与所得控除額の1/2を超える特定支出がある場合、超過分を追加控除可能
- 特定支出: 通勤費、転居費、研修費、資格取得費、帰宅旅費、勤務必要経費（上限65万円）

### 1-6. 退職所得（所得税法第30条）

**定義**: 退職手当、一時恩給その他の退職により一時に受ける給与

**計算式**:
```
退職所得 = （収入金額 - 退職所得控除額）× 1/2
```

**退職所得控除額**:

| 勤続年数 | 退職所得控除額 |
|---------|--------------|
| 20年以下 | 40万円 × 勤続年数（最低80万円） |
| 20年超 | 800万円 + 70万円 ×（勤続年数 - 20年） |

**具体例**:
- 勤続10年: 40万円 × 10 = 400万円
- 勤続20年: 40万円 × 20 = 800万円
- 勤続25年: 800万円 + 70万円 × 5 = 1,150万円
- 勤続30年: 800万円 + 70万円 × 10 = 1,500万円
- 勤続35年: 800万円 + 70万円 × 15 = 1,850万円
- 勤続38年: 800万円 + 70万円 × 18 = 2,060万円

**注意事項**:
- 勤続年数の1年未満の端数は切り上げ（例: 20年3ヶ月 → 21年）
- 障害者になったことを直接の原因として退職した場合: 上記控除額 + 100万円
- 短期退職手当等（勤続5年以下の役員等以外）: 退職所得控除額を控除した残額のうち300万円超の部分は1/2課税なし
- 特定役員退職手当等（勤続5年以下の役員等）: 1/2課税なし（全額課税）
- 「退職所得の受給に関する申告書」未提出の場合: 収入金額 × 20.42% の源泉徴収（確定申告で精算）

### 1-7. 山林所得（所得税法第32条）

**定義**: 山林の伐採又は譲渡による所得（取得後5年以内の伐採・譲渡は事業所得又は雑所得）

**計算式**:
```
山林所得 = 総収入金額 - 必要経費 - 特別控除額（最高50万円）
```

- **5分5乗方式**で税額計算（山林所得 × 1/5 に対する税額を5倍）
- 青色申告特別控除（10万円のみ適用可）

### 1-8. 譲渡所得（所得税法第33条）

**定義**: 資産の譲渡による所得（棚卸資産、事業用固定資産、山林の譲渡を除く）

#### 総合課税の譲渡所得（ゴルフ会員権、書画骨董等）

```
譲渡所得 = 総収入金額 - （取得費 + 譲渡費用）- 特別控除額（最高50万円）
```
- 短期譲渡所得（取得後5年以内）: 全額を総合課税
- 長期譲渡所得（取得後5年超）: 1/2を総合課税

#### 分離課税の譲渡所得

**土地建物等**:
- 短期譲渡所得（取得後5年以内、1/1基準）: 所得税30% + 復興税0.63% + 住民税9% = 39.63%
- 長期譲渡所得（取得後5年超、1/1基準）: 所得税15% + 復興税0.315% + 住民税5% = 20.315%
- 10年超所有の居住用財産の軽減税率: 6,000万円以下の部分は所得税10% + 復興税0.21% + 住民税4% = 14.21%

**居住用財産の3,000万円特別控除**（租税特別措置法第35条）

**上場株式等**:
- 申告分離課税: 所得税15% + 復興税0.315% + 住民税5% = 20.315%
- 上場株式等の譲渡損失と配当所得の損益通算可
- 譲渡損失の3年間繰越控除可

### 1-9. 一時所得（所得税法第34条）

**定義**: 利子所得〜譲渡所得のいずれにも該当しない一時の所得で、営利目的の継続行為から生じたものでないもの

**例**: 懸賞当選金、満期保険金、競馬の払戻金（一時的なもの）、ふるさと納税の返礼品

**計算式**:
```
一時所得 = 総収入金額 - 収入を得るために支出した金額 - 特別控除額（最高50万円）
```
- 課税対象: 一時所得 × 1/2 を総合課税

### 1-10. 雑所得（所得税法第35条）

**定義**: 他の9種類の所得のいずれにも該当しない所得

**区分と計算**:

| 区分 | 計算式 |
|------|--------|
| 公的年金等 | 収入金額 - 公的年金等控除額 |
| 業務に係るもの（原稿料、講演料、副業収入等） | 総収入金額 - 必要経費 |
| その他（上記以外） | 総収入金額 - 必要経費 |

**公的年金等控除額**（65歳未満、公的年金等の収入金額に応じて）:

| 公的年金等の収入金額（65歳未満） | 控除額（他の所得1,000万円以下） |
|-------------------------------|------------------------------|
| 130万円以下 | 60万円 |
| 130万円超〜410万円以下 | 収入金額 × 25% + 27.5万円 |
| 410万円超〜770万円以下 | 収入金額 × 15% + 68.5万円 |
| 770万円超〜1,000万円以下 | 収入金額 × 5% + 145.5万円 |
| 1,000万円超 | 195.5万円 |

| 公的年金等の収入金額（65歳以上） | 控除額（他の所得1,000万円以下） |
|-------------------------------|------------------------------|
| 330万円以下 | 110万円 |
| 330万円超〜410万円以下 | 収入金額 × 25% + 27.5万円 |
| 410万円超〜770万円以下 | 収入金額 × 15% + 68.5万円 |
| 770万円超〜1,000万円以下 | 収入金額 × 5% + 145.5万円 |
| 1,000万円超 | 195.5万円 |

**注意**: 他の所得が1,000万円超2,000万円以下の場合は上記より10万円少ない控除額、2,000万円超の場合は20万円少ない控除額

## 2. 所得税の税率表（超過累進税率）

所得税法第89条に基づく税率（令和2年分以降）:

| 課税される所得金額 | 税率 | 控除額 |
|------------------|------|--------|
| 1,000円 〜 1,949,000円 | 5% | 0円 |
| 1,950,000円 〜 3,299,000円 | 10% | 97,500円 |
| 3,300,000円 〜 6,949,000円 | 20% | 427,500円 |
| 6,950,000円 〜 8,999,000円 | 23% | 636,000円 |
| 9,000,000円 〜 17,999,000円 | 33% | 1,536,000円 |
| 18,000,000円 〜 39,999,000円 | 40% | 2,796,000円 |
| 40,000,000円以上 | 45% | 4,796,000円 |

**速算式**:
```
所得税額 = 課税所得金額 × 税率 - 控除額
```

**計算例**:
- 課税所得500万円: 500万円 × 20% - 427,500円 = 572,500円
- 課税所得800万円: 800万円 × 23% - 636,000円 = 1,204,000円
- 課税所得1,200万円: 1,200万円 × 33% - 1,536,000円 = 2,424,000円

## 3. 復興特別所得税

**根拠法**: 東日本大震災からの復興のための施策を実施するために必要な財源の確保に関する特別措置法

```
復興特別所得税額 = 基準所得税額 × 2.1%
```

- **適用期間**: 2013年（平成25年）1月1日 〜 2037年（令和19年）12月31日（25年間）
- 源泉徴収税率にも上乗せ（例: 所得税率10% → 10.21%）
- 確定申告書に復興特別所得税額を記載

## 4. 所得控除（15種類）

所得控除は以下の15種類。課税所得金額の計算において総所得金額等から控除する。

### 4-1. 雑損控除（所得税法第72条）

災害、盗難、横領による資産の損失

**控除額**: 次のいずれか大きい金額
1. （損失額 - 保険金等による補填額）- 総所得金額等 × 10%
2. 災害関連支出の金額 - 5万円

- 控除しきれない金額は翌年以後3年間繰越可
- 詐欺・恐喝による損失は対象外

### 4-2. 医療費控除（所得税法第73条）

**通常の医療費控除**:
```
控除額 = 支払った医療費 - 保険金等で補填される金額 - 10万円（又は総所得金額等の5%のいずれか少ない方）
```
- 控除限度額: 200万円
- 対象: 本人及び生計を一にする配偶者・親族の医療費

**セルフメディケーション税制**（選択適用）:
```
控除額 = 特定一般用医薬品等購入費 - 保険金等で補填される金額 - 12,000円
```
- 控除限度額: 88,000円
- 要件: 健康診断、予防接種等の「一定の取組」を行っていること
- 適用期限: 2027年（令和9年）12月31日

### 4-3. 社会保険料控除（所得税法第74条）

本人及び生計を一にする配偶者・親族の社会保険料を支払った場合

**控除額**: 支払った社会保険料の全額（上限なし）

対象: 健康保険料、厚生年金保険料、国民健康保険料（税）、国民年金保険料、介護保険料、後期高齢者医療保険料、雇用保険料等

### 4-4. 小規模企業共済等掛金控除（所得税法第75条）

**控除額**: 支払った掛金の全額

| 対象 | 上限額 |
|------|--------|
| 小規模企業共済 | 月額70,000円（年間840,000円） |
| 個人型確定拠出年金（iDeCo） | 加入区分による（自営業者: 月68,000円、会社員: 月12,000〜23,000円等） |
| 心身障害者扶養共済 | 掛金全額 |

### 4-5. 生命保険料控除（所得税法第76条）

**新制度**（2012年1月1日以後の契約）:

| 区分 | 年間払込保険料 | 控除額 |
|------|-------------|--------|
| 一般生命保険料 | 20,000円以下 | 払込保険料の全額 |
| | 20,001円〜40,000円 | 払込保険料 × 1/2 + 10,000円 |
| | 40,001円〜80,000円 | 払込保険料 × 1/4 + 20,000円 |
| | 80,001円以上 | 40,000円 |
| 介護医療保険料 | 同上 | 同上 |
| 個人年金保険料 | 同上 | 同上 |

- 各区分の上限: 40,000円
- 3区分合計の上限: **120,000円**

**旧制度**（2011年12月31日以前の契約）:

| 区分 | 年間払込保険料 | 控除額 |
|------|-------------|--------|
| 一般生命保険料 | 25,000円以下 | 払込保険料の全額 |
| | 25,001円〜50,000円 | 払込保険料 × 1/2 + 12,500円 |
| | 50,001円〜100,000円 | 払込保険料 × 1/4 + 25,000円 |
| | 100,001円以上 | 50,000円 |
| 個人年金保険料 | 同上 | 同上 |

- 各区分の上限: 50,000円
- 新旧制度の組み合わせ: 各区分で新旧いずれか有利な方を選択可（ただし各区分の上限は40,000円、合計上限120,000円）

### 4-6. 地震保険料控除（所得税法第77条）

| 区分 | 控除額 |
|------|--------|
| 地震保険料 | 払込保険料の全額（上限50,000円） |
| 旧長期損害保険料 | 10,000円以下: 全額、10,001〜20,000円: 払込保険料 × 1/2 + 5,000円、20,001円以上: 15,000円 |
| 両方ある場合 | 合計（上限50,000円） |

### 4-7. 寄附金控除（所得税法第78条）

```
控除額 = 特定寄附金の額（又は総所得金額等の40%のいずれか少ない方）- 2,000円
```

対象: 国・地方公共団体への寄附（ふるさと納税含む）、指定寄附金、特定公益増進法人への寄附、認定NPO法人への寄附等

**ふるさと納税**: ワンストップ特例制度（5団体以内、確定申告不要の給与所得者等）

### 4-8. 障害者控除（所得税法第79条）

| 区分 | 控除額 |
|------|--------|
| 障害者 | 27万円 |
| 特別障害者 | 40万円 |
| 同居特別障害者 | 75万円 |

- 本人、同一生計配偶者、扶養親族が対象

### 4-9. 寡婦控除（所得税法第80条）

**控除額**: 27万円

**要件**（以下のいずれか）:
- 夫と離婚後婚姻しておらず、扶養親族を有し、合計所得金額500万円以下で、事実婚状態でない
- 夫と死別後婚姻しておらず（又は夫の生死不明）、合計所得金額500万円以下で、事実婚状態でない

### 4-10. ひとり親控除（所得税法第81条）

**控除額**: 35万円

**要件**（全て満たすこと）:
- 現に婚姻していない又は配偶者の生死不明
- 生計を一にする子（総所得金額等48万円以下）を有する
- 合計所得金額500万円以下
- 事実婚状態でない

### 4-11. 勤労学生控除（所得税法第82条）

**控除額**: 27万円

**要件**: 勤労による所得があり、合計所得金額75万円以下（うち勤労以外の所得10万円以下）の学生

### 4-12. 配偶者控除（所得税法第83条）

生計を一にする配偶者（合計所得金額48万円以下）がいる場合

| 納税者の合計所得金額 | 控除額（一般） | 控除額（老人: 70歳以上） |
|-------------------|-------------|---------------------|
| 900万円以下 | 38万円 | 48万円 |

## 5. 税額控除

### 5-1. 配当控除（所得税法第92条）

総合課税を選択した配当所得に対して適用

| 課税総所得金額等 | 配当控除率（一般） | 配当控除率（証券投資信託） |
|----------------|-----------------|----------------------|
| 1,000万円以下の部分 | 10% | 5% |
| 1,000万円超の部分 | 5% | 2.5% |

- 外貨建等証券投資信託: 上記の1/2
- 申告分離課税を選択した場合は適用なし

### 5-2. 住宅借入金等特別控除（住宅ローン控除）（租税特別措置法第41条）

**令和4年（2022年）〜令和7年（2025年）入居分**:

| 住宅の種類 | 借入限度額 | 控除率 | 控除期間 |
|-----------|----------|--------|---------|
| 認定住宅（長期優良・低炭素） | 4,500万円（R6以降: 子育て世帯等5,000万円） | 0.7% | 13年 |
| ZEH水準省エネ住宅 | 3,500万円（R6以降: 子育て世帯等4,500万円） | 0.7% | 13年 |
| 省エネ基準適合住宅 | 3,000万円（R6以降: 子育て世帯等4,000万円） | 0.7% | 13年 |
| その他の住宅 | 0円（R5入居まで: 2,000万円） | 0.7% | 10年（R5入居まで: 13年） |
| 既存住宅（認定等） | 3,000万円 | 0.7% | 10年 |
| 既存住宅（その他） | 2,000万円 | 0.7% | 10年 |

**主な要件**:
- 床面積50平方メートル以上（合計所得金額1,000万円以下の場合は40平方メートル以上）
- 合計所得金額2,000万円以下
- 引渡しから6ヶ月以内に居住開始
- 住宅ローンの返済期間10年以上
- 初年度は確定申告が必要（2年目以降は年末調整可）

### 5-3. 外国税額控除（所得税法第95条）

外国で課された所得税に相当する税額を日本の所得税額から控除

```
控除限度額 = その年の所得税額 × （その年の国外所得金額 / その年の所得総額）
```

- 控除しきれない金額は翌年以後3年間繰越可
- 地方税からも一定額を控除可（住民税の控除限度額）

## 6. 青色申告制度

### 6-1. 承認申請

**申請期限**:
- 新規開業の場合: 開業日から2ヶ月以内
- 既に事業を行っている場合: 適用を受けようとする年の3月15日まで
- 相続により事業を承継した場合: 相続開始を知った日の時期に応じた期限あり

**申請書**: 「所得税の青色申告承認申請書」を所轄税務署長に提出

### 6-2. 青色申告特別控除

| 控除額 | 要件 |
|--------|------|
| **65万円** | 正規の簿記の原則（複式簿記）による帳簿作成 + 貸借対照表・損益計算書を確定申告書に添付 + e-Tax又は電子帳簿保存 |
| **55万円** | 正規の簿記の原則（複式簿記）による帳簿作成 + 貸借対照表・損益計算書を確定申告書に添付（e-Tax・電子帳簿保存なし） |
| **10万円** | 上記以外（簡易帳簿でも可） |

- 適用対象: 事業所得、不動産所得（事業的規模）、山林所得
- 不動産所得が事業的規模でない場合: 10万円のみ
- 期限後申告の場合: 65万円・55万円の控除は適用不可（10万円は可）

### 6-3. 青色事業専従者給与（所得税法第57条）

**要件**:
- 「青色事業専従者給与に関する届出書」を所轄税務署に提出（適用を受ける年の3月15日まで。新たに事業開始・専従者がいることとなった日から2ヶ月以内）
- 青色申告者と生計を一にする配偶者その他の親族
- その年の12月31日現在で15歳以上
- その年を通じて6月を超える期間（又は事業従事可能期間の1/2超）事業に専ら従事
- 届出書に記載した金額の範囲内で、労務の対価として相当と認められる金額

**注意**: 専従者給与を支払った場合、その配偶者・親族は配偶者控除・扶養控除の対象外

### 6-4. 純損失の繰越控除・繰戻還付

**繰越控除**: 青色申告者に生じた純損失を翌年以後**3年間**にわたり繰り越して各年分の所得金額から控除可能

**繰戻還付**: 青色申告者が純損失を生じた場合、前年分（**1年間**）の所得税の還付を請求可能
```
還付税額 = 前年分の所得税額 × 当年の純損失の金額 / 前年分の課税所得金額
```

### 6-5. その他の青色申告の特典

- 貸倒引当金の設定（一括評価: 事業所得の金銭債権 × 5.5%等）
- 棚卸資産の低価法の選択
- 減価償却の特例（少額減価償却資産の即時償却: 取得価額30万円未満、年間合計300万円まで）
- 更正の理由附記

## 7. 損益通算（所得税法第69条）

### 損益通算の対象（損失が生じた場合に他の所得と通算可能）

**通算可能な所得**: 不動産所得、事業所得、山林所得、譲渡所得（「ふじさんじょう」と覚える）

### 損益通算の順序

1. **第1次通算**: 各所得内で通算
   - 経常所得グループ（利子・配当・不動産・事業・給与・雑）内で通算
   - 譲渡・一時所得グループ内で通算
2. **第2次通算**: グループ間で通算
   - 経常所得グループの損失 → 譲渡・一時所得グループから控除
   - 譲渡・一時所得グループの損失 → 経常所得グループから控除
3. **第3次通算**: 山林所得・退職所得との通算

### 損益通算の制限

- **不動産所得の損失のうち通算不可**: 土地取得に係る借入金利子に対応する部分
- **生活に通常必要でない資産**の譲渡損失: 通算不可（競走馬、別荘、1個30万円超の貴金属等）
- **株式等の譲渡損失**: 他の所得との通算不可（株式等の譲渡所得内で通算、上場株式等は配当所得と通算可）
- **土地建物等の譲渡損失**: 他の所得との通算不可（居住用財産の買換え等の特例を除く）
- **雑所得の損失**: 通算不可

## 8. 必要経費の判定基準と家事按分

### 必要経費の基本原則（所得税法第37条）

1. **総収入金額に対応する売上原価その他の費用**
2. **その年における販売費・一般管理費その他の業務上の費用**

### 必要経費の判定基準

- 業務の遂行上直接必要であること
- 収入を得るために必要な支出であること
- 金額が適正であること（社会通念上相当）
- 証拠書類（領収書、請求書等）で裏付けられること

### 家事按分（所得税法第45条、所得税法施行令第96条）

業務用と家事用の両方に関連する経費（家事関連費）は、業務に直接必要な部分を明確に区分できる場合に限り、その部分を必要経費に算入可能

**主な按分基準**:

| 経費 | 按分基準（例） |
|------|-------------|
| 家賃 | 事業用面積 / 総面積 |
| 水道光熱費 | 使用時間、使用量、事業用面積比 |
| 通信費（電話・インターネット） | 使用時間、通話記録 |
| 自動車関連費 | 走行距離、使用日数 |
| 減価償却費（自宅兼事務所） | 事業用面積 / 総面積 |

**按分割合の記録**: 按分の根拠を合理的に説明できるよう記録を保持すること

## 9. 事業主貸・事業主借

個人事業主の事業用資金と個人用資金の区分に使用する勘定科目

### 事業主貸（事業 → 個人）

事業の資金を個人的な用途に使用した場合

**例**:
- 生活費の引き出し
- 個人的な買い物の支払い
- 所得税・住民税の支払い（所得税は必要経費にならない）
- 国民健康保険料・国民年金保険料の支払い（事業の経費ではなく所得控除）
- 家事按分で個人使用部分とした金額

### 事業主借（個人 → 事業）

個人の資金を事業に充てた場合

**例**:
- 個人の預金から事業用口座への入金
- 個人のクレジットカードで事業経費を支払った場合
- 事業用口座で受け取った利子（事業所得ではなく利子所得）

### 期末処理

```
期末の元入金 = 期首の元入金 + 事業主借残高 - 事業主貸残高 + 青色申告特別控除前の所得金額
```

   - 給与収入が2,000万円を超える
   - 給与所得・退職所得以外の所得が20万円を超える

4. 土地建物や株式を譲渡した場合

## 11. 予定納税（所得税法第104条〜）

### 対象者

前年分の申告納税額（予定納税基準額）が**15万円以上**の場合

### 予定納税基準額

前年分の所得税額（源泉徴収税額を除く）から、前年分の譲渡所得、一時所得、雑所得、分離課税の所得等を除いて計算した金額

### 納付期限・納付額

| 期 | 納付期限 | 納付額 |
|----|---------|--------|
| 第1期 | 7月31日 | 予定納税基準額の1/3 |
| 第2期 | 11月30日 | 予定納税基準額の1/3 |

- 第3期（確定申告期限: 翌年3月15日）に残額を精算
- 納付期限が土日祝日の場合は翌営業日

### 予定納税額の減額申請

その年の所得が前年より減少する見込みの場合:
- 第1期分: 7月15日まで
- 第2期分: 11月15日まで
