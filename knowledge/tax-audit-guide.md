# 税務調査対応 完全ガイド

## 1. 税務調査の種類

### 1-1. 一般調査（任意調査）
- **概要**: 国税通則法第74条の2〜6に基づく質問検査権による調査
- **頻度**: 法人は概ね3〜5年に1回（規模・業種による）、個人は5〜10年に1回程度
- **調査期間**: 通常1〜3日間（大規模法人は1〜2週間）
- **調査対象年度**: 通常3年分（問題が見つかれば5年、悪質な場合は7年）
- **特徴**: 事前通知あり、調査日時の調整が可能

### 1-2. 特別調査（特官案件）
- **概要**: 大口・悪質な脱税が疑われるケースに対する重点調査
- **担当**: 特別国税調査官（特官）または資料調査課
- **調査期間**: 数週間〜数ヶ月に及ぶことがある
- **特徴**: 入念な事前準備の上で実施、無予告で行われることもある

### 1-3. 反面調査
- **概要**: 調査対象者の取引先に対して行う調査
- **目的**: 調査対象者の申告内容の裏付けを取る
- **対象**: 仕入先、得意先、金融機関等
- **注意点**: 正当な理由なく拒否はできない

### 1-4. 資料せん
- **概要**: 税務署が取引情報を収集するために送付する「一般収集資料」
- **提出義務**: 法的義務はないが、協力が求められる
- **収集内容**: 取引先への支払金額、取引内容等
- **利用目的**: 他の納税者の調査のための情報収集

### 1-5. 査察調査（マルサ）
- **概要**: 国税犯則取締法に基づく強制調査（現在は国税通則法第11章）
- **担当**: 国税局査察部（通称マルサ）
- **対象**: 悪質な脱税（1億円以上が目安）
- **特徴**: 裁判所の許可状に基づく強制捜索・差押え
- **結果**: 検察庁への告発→刑事裁判

## 2. 調査対象の選定基準

### 2-1. 調査されやすい会社の特徴
- **申告内容の異常値**: 売上の急激な増減、利益率の大幅な変動
- **同業他社との比較**: 業種平均と大きく乖離している項目がある
- **過去の調査で問題があった**: 修正申告の経験がある
- **無申告・期限後申告**: 申告義務があるのに申告していない
- **タレコミ（情報提供）**: 元従業員等からの内部告発
- **資料せんとの不一致**: 取引先の申告内容と矛盾がある
- **現金商売**: 飲食店、小売店、理美容業等
- **設立後初めての調査**: 設立3〜5年目の法人
- **代表者の個人資産と法人の申告の不整合**: 生活水準と所得の乖離
- **特定の業種**: 不動産業、建設業、IT関連、医療法人等

### 2-2. KSK（国税総合管理システム）による選定
- 全国の申告データを一元管理
- 過去の申告との比較分析
- 業種別の平均値との比較
- 異常値を自動検出してリストアップ

## 3. 事前通知の内容と対応方法

### 3-1. 事前通知の内容（国税通則法第74条の9）
税務署は、原則として調査の前に以下の事項を通知する:

1. **調査対象者の氏名・名称**
2. **調査の開始日時**
3. **調査の場所**
4. **調査の目的**（所得税、法人税、消費税等）
5. **調査の対象税目**
6. **調査の対象期間**
7. **調査対象の帳簿書類等**
8. **調査の相手方（担当者名）**
9. **調査担当者の所属・氏名**

### 3-2. 事前通知を受けた場合の対応
- **日程調整**: 合理的な理由があれば日程変更の申し出が可能
- **税理士への連絡**: 直ちに顧問税理士に連絡し、立会いを依頼
- **書類の準備**: 対象期間の帳簿・証憑書類を整理
- **従業員への周知**: 調査当日の対応について説明

### 3-3. 事前通知が行われない場合（無予告調査）
- 事前通知をすると証拠の隠滅・逃亡のおそれがある場合
- 現金商売で現金の確認が必要な場合
- 内観調査（飲食店等で客を装って実態を確認する調査）
- 無予告でも任意調査であれば拒否は可能（ただし正当な理由が必要）

## 4. 調査当日の流れ

### 4-1. 初日
| 時間帯 | 内容 |
|--------|------|
| 10:00頃 | 調査官到着、身分証明書の提示、名刺交換 |
| 10:00〜12:00 | **概況聴取**: 事業内容、組織体制、取引の流れ、経理処理の方法等のヒアリング |
| 12:00〜13:00 | 昼食休憩（調査官は別の場所で食事） |
| 13:00〜16:30 | **帳簿調査**: 総勘定元帳、仕訳帳、各種補助簿の確認。請求書・領収書等の証憑書類との突合 |
| 16:30〜17:00 | 初日のまとめ、翌日の準備事項の確認 |

### 4-2. 2日目以降
- 帳簿調査の継続
- 個別論点の深掘り（疑義のある取引の詳細確認）
- 反面調査の必要性の判断
- 銀行口座の確認（代表者個人口座を含む場合がある）

### 4-3. 最終日〜その後
- 調査結果の概要説明
- 指摘事項の提示
- 修正申告の勧奨（該当する場合）
- 更正の予知（修正申告に応じない場合は更正処分の可能性を示唆）

### 4-4. 調査対応の注意点
- **嘘をつかない**: 質問には正直に答える。虚偽の答弁は重加算税の対象となりうる
- **余計なことは言わない**: 聞かれたことにだけ答える。自発的に不利な情報を提供しない
- **書類は見せるが渡さない**: 原本は手元に置き、必要に応じてコピーを提供
- **税理士の立会い**: 必ず顧問税理士の立会いを求める（税理士法第30条の書面添付がある場合は意見聴取が先行）
- **プライベートの質問**: 代表者の個人的な資産・生活状況の質問にも一定の範囲で回答義務がある

## 5. 帳簿書類の保存期間

### 5-1. 法人の場合
| 書類の種類 | 保存期間 | 起算日 |
|-----------|---------|--------|
| 帳簿（総勘定元帳、仕訳帳等） | **7年** | 事業年度の確定申告書の提出期限の翌日 |
| 決算関係書類（貸借対照表、損益計算書等） | **7年** | 同上 |
| 取引関係書類（請求書、領収書、契約書等） | **7年** | 同上 |
| 欠損金が生じた事業年度の書類 | **10年** | 同上 |
| 電子取引データ | **7年**（欠損金がある場合10年） | 同上 |

### 5-2. 個人事業者の場合
| 書類の種類 | 保存期間 | 起算日 |
|-----------|---------|--------|
| 帳簿（仕訳帳、総勘定元帳等） | **7年** | 確定申告期限の翌日 |
| 決算関係書類（損益計算書、貸借対照表等） | **7年** | 同上 |
| 現金預金取引関係書類（領収書、預金通帳等） | **7年**（前々年分の所得300万円以下は5年） | 同上 |
| その他の書類（請求書、見積書、納品書等） | **5年** | 同上 |

### 5-3. 消費税関連
- 仕入税額控除の適用を受けるための帳簿・請求書等: **7年**
- インボイス（適格請求書）の保存: **7年**

## 6. 調査で指摘されやすいポイント

### 6-1. 売上関連
| 指摘事項 | 内容 |
|---------|------|
| **売上除外** | 現金売上の未計上、簿外口座への入金 |
| **期ずれ** | 売上の計上時期のずらし（期末の売上を翌期に繰り延べ） |
| **仮装売上** | 関連会社間での不自然な取引 |
| **売上値引き・返品の過大計上** | 架空の値引き・返品の計上 |

### 6-2. 経費関連
| 指摘事項 | 内容 |
|---------|------|
| **架空経費** | 実在しない取引先への支払い、水増し請求 |
| **私的経費の混入** | 代表者の個人的な飲食費・旅行代・車両費等 |
| **交際費の損金不算入** | 5,000円超の飲食費を会議費として処理 |
| **外注費と給与の区分** | 実質的に給与である外注費の否認 |
| **消耗品費の水増し** | 期末に大量購入し全額経費計上 |

### 6-3. 在庫関連
| 指摘事項 | 内容 |
|---------|------|
| **在庫操作** | 期末棚卸の過少計上（利益の圧縮） |
| **仕掛品の未計上** | 建設業・製造業での仕掛品の計上漏れ |
| **評価損の不適切な計上** | 要件を満たさない棚卸資産の評価損 |

### 6-4. 人件費関連
| 指摘事項 | 内容 |
|---------|------|
| **架空人件費** | 実在しない従業員への給与支払い |
| **役員報酬の損金不算入** | 定期同額でない役員報酬の変更、過大役員報酬 |
| **使用人兼務役員の判定** | 常勤役員を使用人兼務として賞与を損金算入 |

### 6-5. その他
| 指摘事項 | 内容 |
|---------|------|
| **貸倒損失の要件不備** | 法的整理等の事実がない貸倒処理 |
| **寄附金の損金不算入** | 限度額を超える寄附金 |
| **源泉徴収漏れ** | 報酬・料金等の源泉徴収漏れ |
| **印紙税の貼付漏れ** | 契約書・領収書への収入印紙の貼付漏れ |
| **消費税の仕入税額控除否認** | 帳簿・請求書等の保存要件不備 |

## 7. 修正申告と更正の違い・対応方法

### 7-1. 修正申告
| 項目 | 内容 |
|------|------|
| **定義** | 納税者が自らの意思で申告内容を修正すること |
| **メリット** | 不服申立て（再調査の請求・審査請求）ができないが、更正処分に比べて円満に解決しやすい |
| **デメリット** | 不服申立ての権利を放棄することになる |
| **期限** | 法定申告期限から5年以内（偽りその他不正の行為の場合は7年） |
| **加算税** | 過少申告加算税（自主的な修正の場合は不要、調査通知後は5%、更正予知後は10%/15%） |

### 7-2. 更正
| 項目 | 内容 |
|------|------|
| **定義** | 税務署長が職権で申告内容を修正する処分 |
| **納税者の対応** | 更正通知書が届く→不服があれば不服申立てが可能 |
| **期限** | 法定申告期限から5年以内（偽りその他不正の行為の場合は7年） |
| **増額更正** | 税額が増える方向の修正 |
| **減額更正** | 税額が減る方向の修正（嘆願書の提出により求めることも可能） |

### 7-3. 対応の判断基準
- **指摘が正当な場合**: 修正申告に応じる（早期解決が望ましい）
- **指摘に疑問がある場合**: 税理士と協議の上、反論・交渉する
- **指摘が不当な場合**: 修正申告に応じず、更正処分を待って不服申立てを行う

## 8. 加算税・延滞税の全種類と税率

### 8-1. 過少申告加算税
| 状況 | 税率 | 備考 |
|------|------|------|
| 自主的な修正申告 | **なし**（0%） | 調査の事前通知前に自主的に修正した場合 |
| 事前通知後〜更正予知前 | **5%**（50万円超の部分は10%） | 2017年1月1日以後適用 |
| 更正予知後（調査により更正を受けることを予知した後） | **10%**（期限内申告税額と50万円のいずれか多い金額を超える部分は15%） | |
| 正当な理由がある場合 | **なし** | 税法の解釈に関して見解の相違がある場合等 |

### 8-2. 無申告加算税
| 状況 | 税率 | 備考 |
|------|------|------|
| 自主的な期限後申告 | **5%** | 調査の事前通知前に自主的に申告した場合 |
| 事前通知後〜更正予知前 | **10%**（50万円超の部分は15%） | |
| 更正予知後 | **15%**（50万円超の部分は20%、300万円超の部分は30%） | 2024年1月1日以後に法定申告期限が到来するものから30%が適用 |
| 正当な理由がある場合 | **なし** | |
| 過去5年以内に無申告加算税等を課されたことがある場合 | 上記税率に**10%加算** | 繰り返し防止措置 |

### 8-3. 不納付加算税
| 状況 | 税率 | 備考 |
|------|------|------|
| 法定納期限後に自主的に納付 | **5%** | |
| 税務署の指摘後に納付 | **10%** | |
| 法定納期限から1ヶ月以内に自主的に納付し、過去1年間に期限後納付がない場合 | **なし** | 免除規定 |

### 8-4. 重加算税
| 状況 | 税率 | 備考 |
|------|------|------|
| 過少申告加算税に代えて（仮装・隠蔽があった場合） | **35%** | |
| 無申告加算税に代えて（仮装・隠蔽があった場合） | **40%** | |
| 過去5年以内に重加算税を課されたことがある場合 | 上記税率に**10%加算**（45%/50%） | 繰り返し防止措置 |

**仮装・隠蔽の具体例**:
- 帳簿の改ざん、二重帳簿の作成
- 架空の経費の計上
- 売上の故意の除外
- 取引先との通謀による仮装

### 8-5. 延滞税
| 期間 | 税率（2024年） | 算出方法 |
|------|--------------|---------|
| 納期限の翌日から2ヶ月以内 | **年2.4%** | 特例基準割合（延滞税特例基準割合）+ 1% |
| 納期限の翌日から2ヶ月超 | **年8.7%** | 特例基準割合（延滞税特例基準割合）+ 7.3% |

**延滞税の計算方法**:
- 延滞税 = 本税 × 延滞税率 × 延滞日数 / 365
- 計算の基礎となる本税: 1万円未満の端数は切り捨て
- 計算した延滞税: 100円未満の端数は切り捨て
- 本税が1万円未満の場合は延滞税は発生しない

## 9. 不服申立て

### 9-1. 制度の概要
税務署長の処分に不服がある場合、以下の3段階で争うことができる。

### 9-2. 再調査の請求（旧: 異議申立て）
| 項目 | 内容 |
|------|------|
| **請求先** | 処分を行った税務署長等 |
| **請求期限** | 処分の通知を受けた日の翌日から3ヶ月以内 |
| **審理期間** | 3ヶ月以内に決定 |
| **注意** | 再調査の請求を経ずに直接審査請求することも可能 |

### 9-3. 審査請求
| 項目 | 内容 |
|------|------|
| **請求先** | 国税不服審判所 |
| **請求期限** | 再調査の決定の通知を受けた日の翌日から1ヶ月以内。直接審査請求の場合は処分の通知を受けた日の翌日から3ヶ月以内 |
| **審理期間** | 1年以内に裁決（目標） |
| **特徴** | 第三者機関による審理。国税審判官が担当 |

### 9-4. 訴訟（取消訴訟）
| 項目 | 内容 |
|------|------|
| **提訴先** | 地方裁判所 |
| **提訴期限** | 裁決の通知を受けた日の翌日から6ヶ月以内 |
| **注意** | 審査請求の裁決を経た後でなければ提訴できない（不服申立前置主義）。ただし、審査請求後3ヶ月を経過しても裁決がない場合は提訴可能 |
| **審級** | 地方裁判所→高等裁判所→最高裁判所 |

### 9-5. 不服申立ての流れ

```
処分の通知
  ↓
再調査の請求（3ヶ月以内）→ 決定
  ↓                          ↓
  ↓（スキップ可能）          審査請求（1ヶ月以内）
  ↓                          ↓
審査請求（3ヶ月以内）    →  裁決
                              ↓
                          訴訟（6ヶ月以内）
                              ↓
                          地裁→高裁→最高裁
```

### 9-6. 実務上のポイント
- 不服申立ての件数のうち、納税者の主張が認められる割合は約10〜15%
- 訴訟まで進めると費用と時間がかかるため、修正申告での解決が一般的
- 税理士法第30条の書面添付がある場合は、更正前に意見聴取の機会がある（調査省略の可能性もある）
