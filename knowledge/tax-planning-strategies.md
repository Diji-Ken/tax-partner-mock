# 節税提案ネタ集

## 1. 役員報酬の最適化

### 1-1. 概要
法人の利益を役員報酬として支給することで、法人税の課税所得を減少させつつ、給与所得控除の恩恵を受ける。

### 1-2. 効果の計算例
**前提**: 法人の利益（役員報酬差引前）1,500万円、代表者1人

| パターン | 役員報酬 | 法人所得 | 法人税等（約34%） | 所得税+住民税（概算） | 社会保険料（会社+個人） | 合計負担額 |
|---------|---------|---------|-----------------|-------------------|---------------------|-----------|
| A: 報酬0円 | 0円 | 1,500万円 | 約510万円 | 0円 | 0円 | 約510万円 |
| B: 報酬600万円 | 600万円 | 900万円 | 約306万円 | 約77万円 | 約170万円 | 約553万円 |
| C: 報酬900万円 | 900万円 | 600万円 | 約204万円 | 約143万円 | 約250万円 | 約597万円 |
| D: 報酬1,200万円 | 1,200万円 | 300万円 | 約66万円 | 約233万円 | 約310万円 | 約609万円 |

※社会保険料は標準報酬月額の上限があるため、報酬が一定額を超えると保険料は頭打ちになる

### 1-3. 適用要件
- 定期同額給与であること（事業年度開始から3ヶ月以内に決定、期中の変更は原則不可）
- 不相当に高額でないこと（職務内容、法人の規模・業績、同業他社の水準等で判定）

### 1-4. 注意点
- 社会保険料の事業主負担を考慮する必要がある
- 個人の所得税率と法人税率のバランスを見て最適額を決定
- 将来の退職金の受取時の税負担も考慮
- 一度決めると期中変更は原則不可（業績悪化事由等を除く）

## 2. 小規模企業共済

### 2-1. 概要
個人事業主・中小企業の役員が退職後の生活資金として積み立てる制度。掛金の全額が所得控除。

### 2-2. 制度の詳細
| 項目 | 内容 |
|------|------|
| 掛金 | 月額**1,000円〜70,000円**（500円単位で設定可能） |
| 年間最大掛金 | **840,000円** |
| 所得控除 | 掛金の**全額が小規模企業共済等掛金控除** |
| 加入資格 | 個人事業主（共同経営者含む）、会社等の役員（従業員20人以下、商業・サービス業は5人以下） |
| 受取時の課税 | 一括: 退職所得、分割: 公的年金等の雑所得 |

### 2-3. 効果の計算例
**月額7万円（年額84万円）を掛けた場合の節税効果**:

| 課税所得 | 所得税率+住民税率 | 年間節税額 |
|---------|-----------------|-----------|
| 400万円 | 30%（所得税20%+住民税10%） | 84万円 × 30% = **252,000円** |
| 700万円 | 33%（所得税23%+住民税10%） | 84万円 × 33% = **277,200円** |
| 1,000万円 | 43%（所得税33%+住民税10%） | 84万円 × 43% = **361,200円** |

### 2-4. 注意点
- 掛金納付月数が12ヶ月未満の場合、掛け捨て（解約手当金なし）
- 20年未満の任意解約は元本割れの可能性
- 法人の経費にはならない（個人の所得控除）

## 3. 経営セーフティ共済（中小企業倒産防止共済）

### 3-1. 概要
取引先の倒産に備えて積み立てる共済制度。掛金は法人の損金（個人の必要経費）に算入可能。

### 3-2. 制度の詳細
| 項目 | 内容 |
|------|------|
| 掛金 | 月額**5,000円〜200,000円**（5,000円単位で設定可能） |
| 年間最大掛金 | **2,400,000円** |
| 掛金の積立限度額 | **800万円**（上限に達すると掛金の積立は停止） |
| 損金算入 | 掛金の**全額が損金**（個人は必要経費） |
| 解約手当金 | **40ヶ月以上で掛金の100%返戻** |
| 解約時の課税 | 法人: 解約手当金は益金算入。個人: 雑所得 |
| 前納制度 | 1年分まで前納可能（前納減額金あり） |

### 3-3. 効果の計算例
**月額20万円（年額240万円）を掛けた場合**:
- 法人税等の実効税率34%の場合: 240万円 × 34% = **816,000円の節税**
- 法人税等の実効税率22%の場合: 240万円 × 22% = **528,000円の節税**

### 3-4. 活用パターン
- **利益が出た年に掛金を増額**、利益が少ない年に解約して収入にする
- **決算月に1年分を前納**して損金を計上
- 退職金の原資として積み立てる

### 3-5. 注意点
- 解約手当金は益金算入されるため、解約時に課税される（課税の繰延べ）
- 解約時に退職金の支給等の損金と相殺する計画が重要
- 12ヶ月未満の解約は掛け捨て
- 40ヶ月未満の解約は元本割れ
- **2024年10月以後の解約**: 解約後2年間は損金算入不可（再加入制限）

## 4. 中退共（中小企業退職金共済）

### 4-1. 概要
中小企業の従業員の退職金制度。掛金は全額損金。

### 4-2. 制度の詳細
| 項目 | 内容 |
|------|------|
| 掛金 | 月額**5,000円〜30,000円**（16段階から選択） |
| 損金算入 | 掛金の**全額が損金** |
| 加入資格 | 中小企業の**従業員**（役員は原則加入不可） |
| 退職金の受取 | 従業員に直接支給される（法人を通さない） |
| 新規加入助成 | 掛金月額の1/2を加入後4ヶ月目から1年間助成（上限5,000円） |
| 掛金増額助成 | 増額分の1/3を1年間助成 |

### 4-3. 注意点
- 退職金は従業員に直接支給されるため、法人が管理できない
- 役員は原則加入不可（使用人兼務役員で従業員部分は可能な場合あり）
- 短期間で退職した場合は元本割れ（1年未満は支給なし、1〜2年は掛金を下回る）

## 5. 確定拠出年金（iDeCo・企業型DC）

### 5-1. iDeCo（個人型確定拠出年金）
| 項目 | 内容 |
|------|------|
| 掛金上限（自営業者） | 月額**68,000円**（年額816,000円） |
| 掛金上限（企業年金なしの会社員） | 月額**23,000円**（年額276,000円） |
| 掛金上限（企業型DCありの会社員） | 月額**20,000円** |
| 掛金上限（公務員） | 月額**12,000円** |
| 掛金上限（専業主婦等） | 月額**23,000円** |
| 所得控除 | 掛金の**全額が小規模企業共済等掛金控除** |
| 運用益 | **非課税** |
| 受取時 | 一時金: 退職所得、年金: 公的年金等の雑所得 |

### 5-2. 企業型DC
| 項目 | 内容 |
|------|------|
| 事業主掛金上限 | 月額**55,000円**（他の企業年金がない場合） |
| 事業主掛金の取扱い | **全額損金**（福利厚生費） |
| 従業員の掛金（マッチング拠出） | 事業主掛金以下かつ合計55,000円以内 |
| 従業員掛金の所得控除 | **全額が小規模企業共済等掛金控除** |

### 5-3. 注意点
- 原則60歳まで引き出し不可
- 運用リスクは加入者自身が負う
- 口座管理手数料がかかる（月額数百円程度）

## 6. ふるさと納税

### 6-1. 概要
自治体への寄附金のうち、自己負担2,000円を除く全額が所得税・住民税から控除される。

### 6-2. 控除の仕組み
```
控除額 = 寄附金額 - 2,000円（上限あり）
```
- 所得税: (寄附金額 - 2,000円) × 所得税率 × 1.021
- 住民税（基本分）: (寄附金額 - 2,000円) × 10%
- 住民税（特例分）: (寄附金額 - 2,000円) × (90% - 所得税率 × 1.021)

### 6-3. 控除上限額の目安
| 給与収入 | 独身/共働き | 夫婦+子1人 |
|---------|------------|-----------|
| 400万円 | 約42,000円 | 約33,000円 |
| 500万円 | 約61,000円 | 約49,000円 |
| 600万円 | 約77,000円 | 約69,000円 |
| 700万円 | 約108,000円 | 約86,000円 |
| 800万円 | 約129,000円 | 約120,000円 |
| 1,000万円 | 約176,000円 | 約166,000円 |
| 1,500万円 | 約389,000円 | 約377,000円 |
| 2,000万円 | 約564,000円 | 約552,000円 |

### 6-4. 注意点
- 法人からのふるさと納税は寄附金として損金算入（特定寄附金の限度額あり）
- 個人事業主は事業経費ではなく、個人の所得控除

## 7. 生命保険の活用

### 7-1. 概要
法人名義で生命保険に加入し、保険料の一部を損金に算入する。

### 7-2. 損金算入ルール（2019年7月8日以後の契約）

| 最高解約返戻率 | 資産計上期間 | 資産計上割合 | 損金算入割合 |
|--------------|------------|------------|------------|
| 50%以下 | なし（全期間損金） | 0% | **100%** |
| 50%超〜70%以下 | 保険期間の前半40% | 40% | **60%** |
| 70%超〜85%以下 | 保険期間の前半40% | 60% | **40%** |
| 85%超 | 保険期間開始から最高返戻率の期間まで | 最高返戻率×90%（10年超は70%） | 残り |

### 7-3. 活用パターン
- **退職金の原資確保**: 解約返戻金を退職金の支払いに充てる
- **事業保障**: 経営者の万一に備える
- **福利厚生**: 従業員の養老保険等

### 7-4. 注意点
- 解約返戻金を受け取ると益金算入される（課税の繰延べに過ぎない）
- 解約時に退職金等の損金と相殺する出口戦略が重要
- 保険料の資金負担が発生する（キャッシュフローへの影響）
- 2019年の改正で全損保険は大幅に制限された

## 8. 決算期変更による消費税免税期間の延長

### 8-1. 概要
設立初年度の事業年度を短くすることで、消費税の免税期間を実質的に延長する。

### 8-2. 効果の計算例
- 設立日: 4月1日、当初決算月: 3月
  - 第1期: 4/1〜3/31（12ヶ月）→ 免税
  - 第2期: 4/1〜3/31（12ヶ月）→ 免税（特定期間の判定次第）
  - 免税期間: 最大24ヶ月

### 8-3. 注意点
- 2023年10月以降のインボイス制度により、取引先がインボイスを求める場合は免税事業者のメリットが限定的
- 特定期間（前半6ヶ月）の課税売上高と給与支払額がともに1,000万円超の場合、2期目から課税

## 9. 設備投資の税額控除

### 9-1. 中小企業投資促進税制
| 項目 | 内容 |
|------|------|
| 対象資産 | 機械装置（160万円以上）、工具（120万円以上）、ソフトウェア（70万円以上）等 |
| 特別償却 | 取得価額の**30%** |
| 税額控除（資本金3,000万円以下） | 取得価額の**7%**（法人税額の20%が上限） |
| 適用期限 | 2027年3月31日まで |

### 9-2. 中小企業経営強化税制
| 項目 | 内容 |
|------|------|
| 対象資産 | A類型（生産性向上）、B類型（収益力強化）、C類型（デジタル化）、D類型（経営資源集約化） |
| 即時償却 | 取得価額の**100%** |
| 税額控除 | 取得価額の**7%**（資本金3,000万円以下は**10%**）、法人税額の20%が上限 |
| 適用期限 | 2027年3月31日まで |
| 手続き | 経営力向上計画の認定が必要 |

### 9-3. 効果の計算例
**機械装置1,000万円を購入した場合（資本金3,000万円以下の中小企業）**:
- 税額控除: 1,000万円 × 10% = **100万円**（法人税から直接控除）
- 即時償却の場合: 1,000万円 × 法人税率34% = **340万円の初年度節税効果**（ただし翌年以降の償却費がなくなる）

## 10. 賃上げ税制

### 10-1. 概要
前年度と比較して給与等の支給額を増加させた場合、増加額の一定割合を法人税額から控除する。

### 10-2. 中小企業向け（2024年4月1日以後開始事業年度）
| 給与増加率 | 税額控除率 | 上乗せ措置 |
|-----------|-----------|-----------|
| 1.5%以上 | **15%** | 教育訓練費増加10%以上: +10%、くるみん認定等: +5%。最大**45%** |
| 2.5%以上 | **30%** | 同上。最大**45%** |

### 10-3. 控除限度額
- 法人税額の**20%**が上限
- 控除しきれない金額は5年間の繰越控除が可能（2024年4月以後開始事業年度から）

### 10-4. 計算例
**前年の給与総額3,000万円 → 当年3,150万円（増加率5%）**
- 増加額: 150万円
- 税額控除: 150万円 × 30% = **45万円**

### 10-5. 注意点
- 給与の「増加」であるため、新規雇用による増加も対象
- 役員・役員の親族への給与は対象外
- 適用を受けるには確定申告書に明細書を添付

## 11. 交際費の5,000円基準

### 11-1. 概要
1人あたり5,000円以下の飲食費（社外の者との飲食）は、交際費から除外して全額損金算入可能。

### 11-2. 適用要件
- 飲食その他これに類する行為（飲食費）であること
- 社外の者（取引先等）との飲食であること（社内飲食は対象外）
- 1人あたりの金額が**5,000円以下**であること
- 以下の事項を記載した書類を保存すること:
  1. 飲食の年月日
  2. 参加した得意先等の氏名・名称と関係
  3. 参加した者の数
  4. 飲食の金額とその飲食店の名称・所在地
  5. その他参考事項

### 11-3. 中小法人の交際費の損金算入
| 区分 | 損金算入限度額 |
|------|-------------|
| 定額控除限度額 | **年800万円**まで全額損金 |
| 接待飲食費の50% | 接待飲食費の50%を損金算入（800万円超でも適用可能） |

※中小法人はいずれか有利な方を選択可能

## 12. 少額減価償却資産の即時償却

### 12-1. 概要
取得価額30万円未満の減価償却資産を、取得した事業年度に全額損金算入できる。

### 12-2. 制度の詳細
| 項目 | 内容 |
|------|------|
| 対象法人 | 青色申告の中小企業者等（資本金1億円以下、従業員500人以下） |
| 対象資産 | 取得価額**30万円未満**の減価償却資産 |
| 年間限度額 | **合計300万円**まで |
| 適用期限 | 2026年3月31日まで（延長の可能性あり） |
| 注意 | 償却資産税の申告対象（固定資産税がかかる） |

### 12-3. 効果の計算例
**パソコン25万円を5台購入した場合（合計125万円）**:
- 通常: 耐用年数4年で減価償却（初年度約31万円の損金）
- 特例適用: 125万円全額を初年度に損金算入
- 初年度の節税効果: (125万円 - 31万円) × 34% = **約32万円の追加節税**

### 12-4. 一括償却資産（20万円未満）との比較
| 項目 | 少額減価償却資産の特例 | 一括償却資産 |
|------|---------------------|------------|
| 金額要件 | 30万円未満 | 20万円未満 |
| 償却期間 | 即時（全額初年度） | 3年均等 |
| 年間限度額 | 300万円 | なし |
| 固定資産税（償却資産税） | **課税対象** | **非課税** |
| 適用対象 | 中小企業者等のみ | 全法人・個人 |

## 13. その他の節税施策

### 13-1. 社宅制度
- 法人が賃貸物件を借り上げ、役員・従業員に社宅として貸し出す
- 家賃の50〜80%を法人の経費にできる（役員の場合は一定の計算式あり）
- 役員の社宅家賃: 固定資産税の課税標準額に基づく計算式で算出した金額以上を徴収

### 13-2. 出張旅費日当
- 出張旅費規程に基づき、日当を支給
- 日当は法人の損金、受取側は非課税（所得税・住民税・社会保険料の対象外）
- 社会通念上相当な金額であること（国内日当: 2,000〜5,000円程度が目安）

### 13-3. 決算賞与
- 決算日までに支給額を各従業員に通知し、翌月末までに支給すれば当期の損金
- 要件: 決算日までに全従業員に書面で個別通知、翌月末までに全額支給、未払計上

### 13-4. 不良在庫の処分
- 陳腐化した在庫の評価損計上
- 廃棄処分（廃棄の事実を証明する書類を保存）

### 13-5. 不良債権の処理
- 法律上の貸倒（破産、民事再生等）: 全額損金算入
- 事実上の貸倒: 資産状況、支払能力等から全額回収不能と判断される場合
- 形式上の貸倒: 取引停止後1年以上経過、同一地域の売掛金合計が取立費用に満たない場合（備忘価額1円を残す）
