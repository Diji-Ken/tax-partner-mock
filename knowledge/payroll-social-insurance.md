> 最終更新: 2026年3月
> 根拠法令: 所得税法、健康保険法、厚生年金保険法、雇用保険法、労働者災害補償保険法、労働基準法

## 1. 給与計算フロー

### 1-1. 基本フロー

```
[1] 総支給額の計算
    ├─ 基本給
    ├─ 各種手当（役職手当、家族手当、住宅手当等）
    ├─ 通勤手当
    ├─ 時間外手当（残業手当、深夜手当、休日手当）
    └─ その他変動給

[2] 控除額の計算
    ├─ 健康保険料（介護保険料含む）
    ├─ 厚生年金保険料
    ├─ 雇用保険料
    ├─ 所得税（源泉徴収税額）
    ├─ 住民税（特別徴収）
    └─ その他控除（財形貯蓄、組合費、社宅使用料等）

[3] 差引支給額（手取り額）
    = 総支給額 - 控除額合計
```

### 1-2. 計算順序の注意

1. まず社会保険料（健康保険料、厚生年金保険料、雇用保険料）を計算
2. 課税対象額 = 総支給額 - 非課税通勤手当 - 社会保険料
3. 源泉所得税を税額表から算出
4. 住民税は市区町村からの通知額をそのまま控除

## 2. 通勤手当の非課税限度額

### 2-1. 電車・バス等の交通機関を利用する場合

**月額150,000円**が非課税限度額

- 通勤に通常必要と認められる運賃・時間・距離等の事情に照らし最も経済的かつ合理的な経路及び方法による通勤定期券等の金額
- 新幹線通勤の特急料金も合理的であれば非課税（グリーン車料金は原則課税）

### 2-2. マイカー・自転車等で通勤する場合（片道通勤距離別）

| 片道の通勤距離 | 1ヶ月あたりの非課税限度額 |
|-------------|----------------------|
| 2km未満 | 全額課税（非課税なし） |
| 2km以上〜10km未満 | **4,200円** |
| 10km以上〜15km未満 | **7,100円** |
| 15km以上〜25km未満 | **12,900円** |
| 25km以上〜35km未満 | **18,700円** |
| 35km以上〜45km未満 | **24,400円** |
| 45km以上〜55km未満 | **28,000円** |
| 55km以上 | **31,600円** |

### 2-3. 交通機関とマイカー等の併用

交通機関の通勤定期券代（月15万円限度）+ マイカー等の距離別非課税限度額の合計額（月15万円限度）

## 3. 時間外手当の計算

### 3-1. 割増賃金の種類と割増率

| 種類 | 割増率 | 適用場面 |
|------|--------|---------|
| 時間外労働（法定労働時間超） | **25%以上** | 1日8時間・週40時間を超える労働 |
| 時間外労働（月60時間超） | **50%以上** | 月60時間を超える時間外労働（中小企業も2023年4月から適用） |
| 深夜労働 | **25%以上** | 午後10時〜午前5時の労働 |
| 法定休日労働 | **35%以上** | 法定休日（週1日又は4週4日）の労働 |

### 3-2. 重複する場合の割増率

| 種類 | 割増率 |
|------|--------|
| 時間外労働 + 深夜労働 | **50%以上**（25% + 25%） |
| 月60時間超の時間外 + 深夜労働 | **75%以上**（50% + 25%） |
| 法定休日労働 + 深夜労働 | **60%以上**（35% + 25%） |

### 3-3. 1時間あたりの基礎賃金の計算

```
1時間あたりの賃金 = 月給 ÷ 月平均所定労働時間数
月平均所定労働時間数 = （365日 - 年間所定休日数）× 1日の所定労働時間 ÷ 12
```

**除外される手当**（割増賃金の基礎に算入しない手当）:
- 家族手当（扶養人数に応じて支給されるもの）
- 通勤手当
- 別居手当
- 子女教育手当
- 住宅手当（住宅費用に応じて支給されるもの）
- 臨時に支払われた賃金
- 1ヶ月を超える期間ごとに支払われる賃金

**注意**: 上記手当でも、一律定額で支給するものは除外されず基礎賃金に含める

### 3-4. 計算例

月給300,000円、月平均所定労働時間160時間の場合:
```
1時間あたりの賃金: 300,000円 ÷ 160時間 = 1,875円
時間外手当（25%増）: 1,875円 × 1.25 = 2,343.75円 → 2,344円（端数切り上げ）
深夜手当（25%増）: 1,875円 × 0.25 = 468.75円 → 469円（深夜の追加分）
法定休日手当（35%増）: 1,875円 × 1.35 = 2,531.25円 → 2,532円
```

## 4. 源泉徴収税額の計算

### 4-1. 月額給与の源泉徴収

**税額表の種類**:

| 欄 | 対象者 | 適用条件 |
|----|--------|---------|
| **甲欄** | 「給与所得者の扶養控除等（異動）申告書」を提出した者 | 主たる給与の支払者1ヶ所のみ |
| **乙欄** | 上記申告書を提出していない者 | 従たる給与（副業・掛け持ち等） |
| **丙欄** | 日額表の丙欄 | 日雇い・短期アルバイト等 |

**甲欄の使い方**:
1. 社会保険料等控除後の給与等の金額を求める
2. 扶養親族等の数を確認
3. 税額表から該当する行・列の税額を読み取る

**甲欄と乙欄の税額差**:
- 甲欄: 扶養親族等の数に応じた税額（88,000円未満で扶養親族0人なら税額0円）
- 乙欄: 扶養親族等の数に関係なく高い税率（最低3.063%）

**社会保険料等控除後の給与等の金額**:
```
= 総支給額 - 非課税通勤手当 - 社会保険料（健康保険料 + 厚生年金保険料 + 雇用保険料）
```

### 4-2. 甲欄の概要（扶養親族等0人の場合の税額例）

| 社会保険料等控除後の給与等の金額 | 税額（扶養0人） |
|-------------------------------|---------------|
| 88,000円未満 | 0円 |
| 88,000円〜89,000円未満 | 130円 |
| 150,000円〜153,000円未満 | 2,980円 |
| 200,000円〜203,000円未満 | 5,130円 |
| 250,000円〜253,000円未満 | 6,530円 |
| 300,000円〜303,000円未満 | 8,040円 |
| 400,000円〜403,000円未満 | 14,650円 |
| 500,000円〜503,000円未満 | 24,400円 |

- 扶養親族等が1人増えるごとに税額は減少
- 詳細は国税庁「給与所得の源泉徴収税額表（月額表）」を参照

### 4-3. 電算機計算の特例（所得税法施行令第309条）

給与計算ソフト等で源泉徴収税額を計算する場合の算式:

```
(1) 社会保険料等控除後の給与等の金額から給与所得控除を算出
(2) 所得控除額を控除
(3) 税額を計算（税率表適用）
(4) 年税額を月割りして税額を算出
```

実務上は給与計算ソフトが自動計算するが、手計算や検算の際は税額表を使用

## 5. 賞与の源泉徴収

### 5-1. 原則的な計算方法

```
Step 1: 前月の社会保険料等控除後の給与等の金額を確認
Step 2: 扶養親族等の数を確認
Step 3: 「賞与に対する源泉徴収税額の算出率の表」から税率を求める
Step 4: 源泉徴収税額 = （賞与の額 - 社会保険料等）× 税率
```

### 5-3. 特例（前月の給与の10倍を超える賞与等）

前月の社会保険料等控除後の給与等の金額がない場合や、賞与の額が前月の給与の10倍を超える場合は別途計算方法あり

```
(1) (賞与 - 社会保険料) ÷ 6
(2) (1) + 前月の社会保険料控除後の給与
(3) (2) の金額を月額表に当てはめて税額を算出
(4) (3) - 前月の給与に対する税額
(5) (4) × 6 = 賞与に対する源泉徴収税額
```

## 6. 社会保険料

### 6-1. 健康保険

#### 概要（協会けんぽの場合）

| 項目 | 内容 |
|------|------|
| 保険者 | 全国健康保険協会（協会けんぽ）又は健康保険組合 |
| 被保険者 | 適用事業所に使用される者（正社員、一定の要件を満たすパート等） |
| 保険料率 | 都道府県ごとに異なる（令和6年度全国平均: 約10.00%） |
| 負担割合 | **労使折半**（事業主50%、被保険者50%） |
| 標準報酬月額 | 1等級58,000円〜50等級1,390,000円（全50等級） |

#### 介護保険料

| 項目 | 内容 |
|------|------|
| 対象者 | 40歳以上65歳未満の被保険者（第2号被保険者） |
| 保険料率 | **1.60%**（令和6年度、協会けんぽ。全国一律） |
| 負担割合 | **労使折半** |

**健康保険料 + 介護保険料の計算例**（東京都、協会けんぽ、令和6年度）:
- 健康保険料率: 9.98%（東京都の場合）
- 介護保険料率: 1.60%（40歳以上65歳未満）
- 合計: 11.58%
- 被保険者負担: 5.79%
- 事業主負担: 5.79%

### 6-2. 厚生年金保険

| 項目 | 内容 |
|------|------|
| 保険料率 | **18.3%**（2017年9月以降固定） |
| 負担割合 | **労使折半**（各9.15%） |
| 標準報酬月額 | 1等級88,000円〜32等級650,000円（全32等級） |
| 適用対象 | 70歳未満の被保険者 |

### 6-3. 標準報酬月額の決定方法

| 決定方法 | 時期 | 適用期間 | 内容 |
|---------|------|---------|------|
| **資格取得時決定** | 入社時 | 入社月〜次の定時決定まで | 採用時の給与額から決定 |
| **定時決定** | 毎年7月 | 9月〜翌年8月 | 4月・5月・6月の報酬の平均額で決定（算定基礎届を提出） |
| **随時改定** | 随時 | 改定月から | 固定的賃金が変動し、3ヶ月間の報酬の平均が2等級以上変動した場合 |
| **育児休業等終了時改定** | 育休終了後 | 改定月から | 育児休業終了日の翌日が属する月以後3ヶ月間の報酬の平均で改定 |
| **産前産後休業終了時改定** | 産休終了後 | 改定月から | 産前産後休業終了日の翌日が属する月以後3ヶ月間の報酬の平均で改定 |

#### 定時決定（算定基礎届）の詳細

- 対象月: 4月・5月・6月に支払われた報酬
- 報酬支払基礎日数が17日未満の月は除外（短時間労働者は11日以上）
- 届出期限: 7月1日〜7月10日
- 適用: その年の9月〜翌年8月

#### 随時改定の要件（3つ全て満たすこと）

1. 固定的賃金（基本給、手当等）に変動があった
2. 変動月からの3ヶ月間に受けた報酬の平均月額の標準報酬月額と従前の標準報酬月額との間に**2等級以上の差**がある
3. 3ヶ月間とも報酬支払基礎日数が**17日以上**

### 6-4. 賞与の社会保険料

賞与に対しても健康保険料、厚生年金保険料が課される。

**標準賞与額**: 賞与の額の1,000円未満を切り捨て

| 保険種別 | 上限額 |
|---------|--------|
| 健康保険 | **年間573万円**（4月1日〜翌3月31日の累計） |
| 厚生年金保険 | **1ヶ月あたり150万円**（同月内に2回以上の賞与がある場合は合算） |

**保険料計算**:
```
健康保険料 = 標準賞与額 × 健康保険料率（介護保険料含む）
厚生年金保険料 = 標準賞与額 × 18.3%
```

## 7. 雇用保険料

### 7-1. 保険料率（令和6年度、一般の事業）

| 負担者 | 料率 |
|--------|------|
| 事業主負担 | **9.5/1000**（0.95%） |
| 被保険者（労働者）負担 | **6/1000**（0.6%） |
| **合計** | **15.5/1000**（1.55%） |

### 7-2. 事業の種類別保険料率

| 事業の種類 | 事業主負担 | 被保険者負担 | 合計 |
|-----------|----------|------------|------|
| 一般の事業 | 9.5/1000 | 6/1000 | 15.5/1000 |
| 農林水産・清酒製造の事業 | 10.5/1000 | 7/1000 | 17.5/1000 |
| 建設の事業 | 11.5/1000 | 7/1000 | 18.5/1000 |

### 7-3. 雇用保険料の計算

```
雇用保険料（被保険者負担分）= 賃金総額（毎月の総支給額）× 被保険者負担率
```

- 賃金総額: 基本給 + 各種手当 + 通勤手当 + 時間外手当 + 賞与等（税引前の総支給額）
- 標準報酬月額ではなく、実際の賃金総額に料率を乗じる
- 1円未満の端数: 50銭以下切り捨て、50銭超切り上げ（労使協定がある場合は別途）

### 7-4. 雇用保険の適用対象

以下の要件を**全て**満たす労働者:
1. 1週間の所定労働時間が**20時間以上**
2. **31日以上**の雇用見込みがあること

- 65歳以上も雇用保険の被保険者（マルチジョブホルダー制度あり）
- 学生は原則適用除外（昼間学生。ただし夜間・通信は適用）

## 8. 労災保険

### 8-1. 概要

| 項目 | 内容 |
|------|------|
| 保険料負担 | **全額事業主負担**（労働者の負担なし） |
| 適用対象 | 全ての労働者（パート・アルバイト含む） |
| 保険料計算 | 賃金総額 × 労災保険率 |

### 8-2. 業種別労災保険率（主なもの、令和6年度）

| 事業の種類 | 労災保険率 |
|-----------|----------|
| 金融業、保険業又は不動産業 | 2.5/1000 |
| その他の各種事業（一般事務等） | 3/1000 |
| 卸売業・小売業、飲食店又は宿泊業 | 3/1000 |
| 通信業、放送業、新聞業又は出版業 | 2.5/1000 |
| 清掃、火葬又はと畜の事業 | 13/1000 |
| 建築事業（既設建築物設備工事業） | 12/1000 |
| 機械器具製造業 | 5/1000 |
| 食料品製造業 | 6/1000 |
| 交通運輸事業 | 4/1000 |
| 林業 | 52/1000 |
| 鉱業（金属鉱業等） | 88/1000 |
| 電気、ガス、水道又は熱供給の事業 | 3/1000 |

- 全業種で最低2.5/1000〜最高88/1000
- メリット制: 一定規模以上の事業場は、過去の災害発生率に基づいて保険率が増減（±40%の範囲）

### 8-3. 労働保険の年度更新

- 保険年度: 4月1日〜翌3月31日
- 年度更新期間: 毎年**6月1日〜7月10日**
- 前年度の確定保険料と当年度の概算保険料を申告・納付
- 労災保険料と雇用保険料を一括して「労働保険料」として申告

### 8-4. 特別加入制度

労災保険の適用対象外の者が任意加入できる制度:
- 中小事業主等（一定規模以下の事業主及びその家族従事者）
- 一人親方等（建設業、運送業等）
- 特定作業従事者
- 海外派遣者
- フリーランス（一定の業務に従事する者、2024年11月から対象拡大）

## 9. 住民税の特別徴収

### 9-1. 概要

事業主（特別徴収義務者）が、従業員の個人住民税を毎月の給与から天引きして市区町村に納付する制度。

### 9-2. 特別徴収のスケジュール

| 時期 | 手続き |
|------|--------|
| 1月31日まで | 事業主が「給与支払報告書」を従業員の住所地の市区町村に提出 |
| 5月頃 | 市区町村から「特別徴収税額の決定通知書」が届く |
| **6月〜翌年5月** | 毎月の給与から税額を天引きして納付 |
| 翌月10日まで | 天引きした住民税を市区町村に納付 |

### 9-3. 年税額の12分割

```
6月分: 年税額 ÷ 12 の端数を含む金額（100円未満の端数は6月に加算）
7月〜翌5月分: 年税額 ÷ 12（均等額）
```

**例**: 年税額240,500円の場合
- 月額: 240,500円 ÷ 12 = 20,041.66... → 20,000円（7月〜翌5月の各月分）
- 6月分: 240,500円 - 20,000円 × 11 = 20,500円

### 9-4. 納期の特例

常時10人未満の従業員の事業所は、市区町村の承認を受けて年2回の納付が可能:
- 6月〜11月分: 12月10日まで
- 12月〜翌5月分: 翌年6月10日まで

### 9-5. 退職時の取扱い

| 退職時期 | 取扱い |
|---------|--------|
| **1月1日〜4月30日に退職** | **一括徴収が義務**（残りの住民税を最後の給与又は退職金から一括天引き） |
| **5月に退職** | 最後の1ヶ月分を徴収 |
| **6月1日〜12月31日に退職** | 本人の申出があれば一括徴収。申出がなければ普通徴収に切替え |

### 9-6. 転職時の取扱い

退職者が転職先で引き続き特別徴収を希望する場合、「特別徴収に係る給与所得者異動届出書」に転職先の情報を記載して市区町村に提出

## 10. 退職金の税務

### 10-1. 退職所得の計算

```
退職所得 = （退職金の額 - 退職所得控除額）× 1/2
```

### 10-2. 退職所得控除額

| 勤続年数 | 控除額 |
|---------|--------|
| 20年以下 | 40万円 × 勤続年数（最低80万円） |
| 20年超 | 800万円 + 70万円 ×（勤続年数 - 20年） |

- 勤続年数の1年未満の端数は切り上げ
- 障害退職の場合: 上記 + 100万円

### 10-3. 退職所得に対する税額計算

```
所得税額 = 退職所得 × 税率 - 控除額（所得税の速算表による）
復興特別所得税 = 所得税額 × 2.1%
住民税 = 退職所得 × 10%（都道府県民税4% + 市区町村民税6%）
```

### 10-4. 退職所得の受給に関する申告書

| 提出の有無 | 源泉徴収の方法 |
|-----------|-------------|
| **提出あり** | 退職所得控除を適用した正確な税額を源泉徴収。確定申告不要（原則） |
| **提出なし** | 退職金の額 × 20.42% を一律源泉徴収。確定申告で精算 |

### 10-5. 1/2課税の例外

| 区分 | 1/2課税 |
|------|---------|
| 一般の退職手当 | **適用あり** |
| 特定役員退職手当等（勤続5年以下の役員等） | **適用なし**（全額課税） |
| 短期退職手当等（勤続5年以下の役員以外）のうち300万円超の部分 | **適用なし** |
| 短期退職手当等のうち300万円以下の部分 | **適用あり** |

### 10-6. 計算例

**ケース**: 勤続25年、退職金2,000万円、退職所得の受給に関する申告書提出済み

```
退職所得控除額: 800万円 + 70万円 × 5 = 1,150万円
退職所得: (2,000万円 - 1,150万円) × 1/2 = 425万円

所得税: 425万円 × 20% - 427,500円 = 422,500円
復興特別所得税: 422,500円 × 2.1% = 8,872円
住民税: 425万円 × 10% = 425,000円

税額合計: 422,500円 + 8,872円 + 425,000円 = 856,372円
手取り額: 2,000万円 - 856,372円 = 19,143,628円
```

## 11. 社会保険の適用拡大

### 11-1. 短時間労働者への適用拡大

| 時期 | 対象企業 | 適用条件 |
|------|---------|---------|
| 2022年10月〜 | 従業員101人以上 | 下記4要件全て |
| 2024年10月〜 | 従業員51人以上 | 下記4要件全て |

**4要件**:
1. 週の所定労働時間が**20時間以上**
2. 月額賃金が**88,000円以上**（年収106万円相当）
3. 雇用期間が**2ヶ月超**見込み
4. **学生でない**こと

### 11-2. 130万円の壁（被扶養者の認定基準）

健康保険の被扶養者認定:
- 年間収入見込みが**130万円未満**（60歳以上又は障害者は180万円未満）
- かつ被保険者の収入の1/2未満

## 12. 年末調整

### 12-1. 概要

給与所得者の年間の所得税を精算する手続き。毎月の源泉徴収税額の合計と年税額の差額を調整する。

### 12-2. スケジュール

| 時期 | 手続き |
|------|--------|
| 10月〜11月 | 従業員から各種申告書・証明書を回収 |
| 12月（最後の給与支払時） | 年末調整の計算・過不足額の精算 |
| 翌年1月10日まで | 源泉所得税の納付（納期の特例: 1月20日まで） |
| 翌年1月31日まで | 法定調書の提出（源泉徴収票、支払調書等） |
| 翌年1月31日まで | 給与支払報告書の市区町村への提出 |

### 12-3. 必要書類

| 書類 | 内容 |
|------|------|
| 給与所得者の扶養控除等（異動）申告書 | 扶養親族等の情報 |
| 給与所得者の基礎控除申告書兼配偶者控除等申告書兼所得金額調整控除申告書 | 基礎控除、配偶者控除等、所得金額調整控除 |
| 給与所得者の保険料控除申告書 | 生命保険料、地震保険料、社会保険料、小規模企業共済等掛金 |
| 住宅借入金等特別控除申告書 | 住宅ローン控除（2年目以降） |

### 12-4. 年末調整の対象外（確定申告が必要）

- 給与収入が2,000万円超
- 災害減免法の適用を受けている
- 年の途中で非居住者になった
- 2ヶ所以上から給与を受け、主たる給与以外の給与が20万円超

## 13. 法定調書の提出

### 13-1. 主な法定調書

| 法定調書 | 提出期限 | 提出先 |
|---------|---------|--------|
| 給与所得の源泉徴収票 | 翌年1月31日 | 税務署 |
| 退職所得の源泉徴収票 | 退職後1ヶ月以内 | 税務署 |
| 報酬、料金、契約金及び賞金の支払調書 | 翌年1月31日 | 税務署 |
| 不動産の使用料等の支払調書 | 翌年1月31日 | 税務署 |
| 不動産等の譲受けの対価の支払調書 | 翌年1月31日 | 税務署 |
| 給与支払報告書 | 翌年1月31日 | 市区町村 |
| 法定調書合計表 | 翌年1月31日 | 税務署 |

### 13-2. 源泉徴収票の交付義務

- 全従業員に対して源泉徴収票を交付（退職者にも）
- 税務署への提出は一定の基準を超えるもののみ:
  - 年末調整をした場合: 給与支払額500万円超（役員は150万円超）
  - 年末調整をしなかった場合: 給与支払額250万円超等
