# インボイス制度（適格請求書等保存方式）完全ガイド

> 税理士・会計事務所向けCRMナレッジ
> 最終更新: 2026-03-23

---

## 概要

2023年10月1日から開始された「適格請求書等保存方式（インボイス制度）」の完全ガイド。
消費税の仕入税額控除を受けるために、原則として適格請求書（インボイス）の保存が必要となった。

---

## 1. 適格請求書発行事業者の登録

### 登録要件

| 項目 | 内容 |
|------|------|
| 対象者 | 消費税の課税事業者（免税事業者は登録と同時に課税事業者となる） |
| 申請先 | 所轄税務署（e-Taxでも申請可能） |
| 申請書 | 「適格請求書発行事業者の登録申請書」 |
| 登録番号の形式 | 法人: 「T」+法人番号（13桁）、個人: 「T」+13桁の番号（マイナンバーとは別） |
| 登録の効力発生 | 登録通知書に記載された登録日から |
| 登録の公表 | 国税庁「適格請求書発行事業者公表サイト」で公表 |

### 免税事業者が登録する場合の特例

- 2023年10月1日〜2029年9月30日の日の属する課税期間中に登録を受ける場合、「消費税課税事業者選択届出書」の提出不要
- 登録日から課税事業者となる（経過措置）
- 登録を取りやめる場合は「適格請求書発行事業者の登録の取消しを求める旨の届出書」を提出

### 登録の取消し

| 区分 | 届出書 | 効力失効日 |
|------|--------|-----------|
| 任意の取消し | 登録の取消しを求める旨の届出書 | 届出書提出日の属する課税期間の翌課税期間の初日（届出書の提出が期間の末日から起算して30日前の前日までに行われた場合） |
| 事業廃止 | 事業廃止届出書 | 届出日 |

## 2. 適格請求書の記載要件（6項目）

適格請求書には以下の6つの事項を記載しなければならない。

| No. | 記載事項 | 詳細・注意点 |
|-----|---------|-------------|
| 1 | 適格請求書発行事業者の氏名又は名称及び登録番号 | 登録番号は「T」+13桁の数字 |
| 2 | 取引年月日 | 課税資産の譲渡等を行った年月日 |
| 3 | 取引内容（軽減税率の対象品目である旨） | 軽減税率対象品目がある場合は「※」等で区分し、「※は軽減税率対象」等と記載 |
| 4 | 税率ごとに区分して合計した対価の額（税抜又は税込）及び適用税率 | 10%対象と8%対象を区分して合計額を記載。税率も明記 |
| 5 | 税率ごとに区分した消費税額等 | 端数処理は1つのインボイスにつき、税率ごとに1回のみ |
| 6 | 書類の交付を受ける事業者の氏名又は名称 | 買手の氏名・名称を記載 |

### 端数処理のルール

- **1つのインボイスにつき、税率ごとに1回の端数処理**
- 個々の商品ごとに消費税額を計算し、端数処理を行ってはならない
- 端数処理の方法（切捨て・切上げ・四捨五入）は事業者の任意

### 記載例

```
請求書
────────────────────────────────────
株式会社○○ 御中

請求書番号: INV-2026-001
発行日: 2026年3月15日

適格請求書発行事業者: 株式会社デジタルツール研究所
登録番号: T1234567890123

  品名               数量  単価      金額
  ──────────────────────────────────
  DXコンサルティング    1   200,000   200,000円（10%対象）
  研修テキスト          10    2,000    20,000円（10%対象）
  ──────────────────────────────────
  10%対象 小計                       220,000円
  消費税（10%）                       22,000円
  ──────────────────────────────────
  合計                              242,000円
────────────────────────────────────
```

## 3. 適格簡易請求書

### 発行できる事業者（限定列挙）

以下の事業を行う事業者は、適格簡易請求書を交付できる。

| 対象事業 |
|---------|
| 小売業 |
| 飲食店業 |
| 写真業 |
| 旅行業 |
| タクシー業 |
| 駐車場業（不特定多数に対するもの） |
| その他不特定多数の者に資産の譲渡等を行う事業 |

### 適格簡易請求書の記載要件（5項目）

適格請求書との違いは「書類の交付を受ける事業者の氏名又は名称」が**不要**であること、及び「税率ごとに区分した消費税額等」又は「適用税率」のいずれか一方の記載で足りること。

| No. | 記載事項 |
|-----|---------|
| 1 | 適格請求書発行事業者の氏名又は名称及び登録番号 |
| 2 | 取引年月日 |
| 3 | 取引内容（軽減税率の対象品目である旨） |
| 4 | 税率ごとに区分して合計した対価の額（税抜又は税込） |
| 5 | 税率ごとに区分した消費税額等 **又は** 適用税率 |

## 4. 返還インボイス（適格返還請求書）

### 交付義務

売上に係る対価の返還等（値引き、割戻し、返品等）を行った場合、適格返還請求書を交付しなければならない。

### 記載要件

| No. | 記載事項 |
|-----|---------|
| 1 | 適格請求書発行事業者の氏名又は名称及び登録番号 |
| 2 | 対価の返還等を行う年月日及びその対価の返還等の基となった取引を行った年月日 |
| 3 | 対価の返還等の基となる取引の内容（軽減税率の対象品目である旨） |
| 4 | 税率ごとに区分して合計した対価の返還等の金額（税抜又は税込） |
| 5 | 対価の返還等の金額に係る消費税額等又は適用税率 |

### 少額な返還インボイスの交付義務免除

**税込1万円未満の対価の返還等**については、適格返還請求書の交付義務が免除される（恒久措置）。
振込手数料を売上値引きとして処理する場合など、実務上頻繁に発生するケースに対応。

## 5. 仕入税額控除の要件

### 原則

仕入税額控除を受けるためには、以下の帳簿及び請求書等の保存が必要。

| 保存すべきもの | 内容 |
|---------------|------|
| 帳簿 | 課税仕入れの相手方の氏名又は名称、取引年月日、取引内容、支払対価の額を記載 |
| 請求書等 | 適格請求書、適格簡易請求書、仕入明細書（相手方の確認を受けたもの）等 |

### 帳簿のみの保存で仕入税額控除が認められる取引

| 取引 | 条件 |
|------|------|
| 公共交通機関（3万円未満） | 鉄道、バス、船舶の旅客運賃 |
| 自動販売機及び自動サービス機（3万円未満） | 自動販売機での購入 |
| 郵便切手を対価とする郵便サービス | 郵便ポストに差し出した場合 |
| 入場券等が回収される取引 | 映画館、遊園地等 |
| 古物営業者等が適格請求書発行事業者でない者から棚卸資産を取得する取引 | 古物商、質屋、宅地建物取引業者等 |
| 従業員等に支給する通常必要な出張旅費等 | 出張旅費、宿泊費、日当等 |

## 6. 少額特例（少額な課税仕入れに係る経過措置）

### 制度概要

| 項目 | 内容 |
|------|------|
| 対象者 | 基準期間における課税売上高が1億円以下の事業者、または特定期間における課税売上高が5,000万円以下の事業者 |
| 内容 | **税込1万円未満の課税仕入れ**について、インボイスの保存がなくても帳簿のみで仕入税額控除が可能 |
| 適用期間 | 2023年10月1日〜**2029年9月30日** |
| 判定単位 | 1回の取引の課税仕入れに係る金額（税込）が1万円未満かどうかで判定 |

### 注意点

- 1商品ごとではなく**1回の取引の合計額**で判定
- 例: 1回の取引で税込8,000円の商品を2つ購入（合計16,000円）→ 特例の対象外
- 例: 1回の取引で税込8,000円の商品を1つ購入 → 特例の対象

---

## 7. 経過措置（免税事業者からの仕入れに係る経過措置）

### 概要

適格請求書発行事業者以外（免税事業者等）からの課税仕入れについて、一定割合を仕入税額とみなして控除できる経過措置。

| 期間 | 控除割合 |
|------|---------|
| 2023年10月1日〜2026年9月30日 | 仕入税額相当額の **80%** |
| 2026年10月1日〜2029年9月30日 | 仕入税額相当額の **50%** |
| 2029年10月1日〜 | 控除不可（0%） |

### 適用要件

- 区分記載請求書等と同様の事項が記載された請求書等の保存
- 帳簿に「80%控除対象」等の経過措置の適用を受ける旨の記載

### 仕訳例（80%控除の場合）

免税事業者から税込55,000円の仕入れを行った場合:

```
（借）仕入高       50,000
（借）仮払消費税    4,000  ← 5,000円 × 80%
（借）雑損失        1,000  ← 5,000円 × 20%（控除できない部分）
（貸）買掛金       55,000
```

## 8. 2割特例（小規模事業者に係る税額控除に関する経過措置）

### 制度概要

| 項目 | 内容 |
|------|------|
| 対象者 | インボイス制度を機に免税事業者から課税事業者になった事業者（登録日が2023年10月1日の属する課税期間〜2026年9月30日の属する課税期間の各課税期間） |
| 内容 | 納付税額を「売上に係る消費税額 × **20%**」とすることができる |
| 適用期間 | 2023年10月1日〜**2026年9月30日の属する課税期間** |
| 届出 | 事前届出不要。確定申告書に付記するだけでOK |
| 適用除外 | 課税期間の特例の適用を受ける課税期間、基準期間の課税売上高が1,000万円超の課税期間 |

### 2割特例の計算例

売上（税込）: 550万円（うち消費税50万円）の場合

```
納付税額 = 50万円 × 20% = 10万円
```

簡易課税（サービス業: みなし仕入率50%）の場合:
```
納付税額 = 50万円 - 50万円 × 50% = 25万円
```

2割特例の方が有利な場合が多い。

### 注意点（2026年10月以降）

- 2割特例は2026年9月30日の属する課税期間で終了
- 終了後は**本則課税**か**簡易課税**を選択する必要がある
- 簡易課税を選択する場合は「消費税簡易課税制度選択届出書」を提出
- 2割特例の適用を受けた課税期間の翌課税期間中に届出書を提出すれば、その届出書を提出した課税期間から簡易課税の適用が可能（届出特例）

## 9. 免税事業者からの仕入れの処理

### 実務上の判断フロー

```
仕入先がインボイス登録事業者か？
├─ Yes → 適格請求書を受領 → 全額仕入税額控除OK
└─ No  → 経過措置の適用期間か？
          ├─ 2026年9月30日まで → 80%控除可能
          ├─ 2029年9月30日まで → 50%控除可能
          └─ 2029年10月1日以降 → 控除不可
```

### 免税事業者との取引における検討事項

| 検討項目 | 内容 |
|---------|------|
| 価格交渉 | 消費税分の値引き要請は独禁法・下請法に抵触する可能性あり。公正取引委員会のガイドラインを確認 |
| 取引継続の判断 | 経過措置終了後の控除不可分のコスト増を考慮 |
| 登録の促し | 取引先に登録を促す場合、あくまで情報提供にとどめる |
| 経理処理 | 経過措置の適用率に応じた仕訳が必要 |

## 10. インボイス番号の検証方法

### 国税庁「適格請求書発行事業者公表サイト」

| 項目 | 内容 |
|------|------|
| URL | https://www.invoice-kohyo.nta.go.jp/ |
| 検索方法 | 登録番号（T+13桁）で検索 |
| 確認できる情報 | 事業者の氏名又は名称、登録番号、登録年月日、法人の場合は本店所在地 |
| Web-API | 提供あり（システム連携可能） |

### Web-APIの仕様

| 項目 | 内容 |
|------|------|
| エンドポイント | `https://web-api.invoice-kohyo.nta.go.jp/1/num?id=T1234567890123&type=21` |
| レスポンス形式 | JSON / CSV |
| 利用制限 | 認証不要、ただし過度なアクセスは制限される可能性あり |
| 活用方法 | CRMにインボイス番号検証機能を組み込み、登録の有効性を自動チェック |

### CRMでの実装推奨事項

1. 取引先マスタにインボイス登録番号フィールドを設置
2. 新規登録・更新時に公表サイトAPIで有効性を自動検証
3. 登録が取り消された場合のアラート機能
4. 免税事業者（未登録）のフラグ管理
5. 経過措置の適用率を取引日に応じて自動判定
