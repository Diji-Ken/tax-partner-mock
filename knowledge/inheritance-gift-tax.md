# 相続税・贈与税ガイド

## 1. 相続税の基礎控除

### 1-1. 基礎控除額
```
基礎控除額 = 3,000万円 + 600万円 × 法定相続人の数
```

| 法定相続人の数 | 基礎控除額 |
|--------------|-----------|
| 1人 | 3,600万円 |
| 2人 | 4,200万円 |
| 3人 | 4,800万円 |
| 4人 | 5,400万円 |
| 5人 | 6,000万円 |

### 1-2. 基礎控除額の計算における留意点
- **養子**: 実子がいる場合は1人まで、実子がいない場合は2人まで法定相続人に含める
- **相続放棄**: 相続放棄した者がいても、放棄がなかったものとして法定相続人の数に含める
- **代襲相続人**: 法定相続人に含める

## 2. 相続税の税率表

### 2-1. 税率表（法定相続分に応ずる取得金額に対する税率）

| 法定相続分に応ずる取得金額 | 税率 | 控除額 |
|------------------------|------|--------|
| 1,000万円以下 | **10%** | なし |
| 1,000万円超〜3,000万円以下 | **15%** | 50万円 |
| 3,000万円超〜5,000万円以下 | **20%** | 200万円 |
| 5,000万円超〜1億円以下 | **30%** | 700万円 |
| 1億円超〜2億円以下 | **40%** | 1,700万円 |
| 2億円超〜3億円以下 | **45%** | 2,700万円 |
| 3億円超〜6億円以下 | **50%** | 4,200万円 |
| 6億円超 | **55%** | 7,200万円 |

### 2-2. 相続税の計算手順
1. 課税遺産総額を算出（遺産総額 - 非課税財産 - 債務・葬式費用 - 基礎控除額）
2. 課税遺産総額を法定相続分で按分（法定相続分に応ずる各取得金額を算出）
3. 各取得金額に税率を適用して各人の仮の税額を算出
4. 仮の税額を合計して相続税の総額を算出
5. 相続税の総額を実際の取得割合で按分して各人の相続税額を算出
6. 各種税額控除を適用

## 3. 法定相続人と法定相続分

### 3-1. 法定相続人の順位
| 順位 | 法定相続人 | 備考 |
|------|-----------|------|
| 常に相続人 | **配偶者** | 離婚した元配偶者は含まない。内縁関係は含まない |
| 第1順位 | **子**（代襲相続: 孫→ひ孫） | 実子・養子（普通養子・特別養子）を含む。胎児も含む |
| 第2順位 | **直系尊属**（父母→祖父母） | 第1順位がいない場合 |
| 第3順位 | **兄弟姉妹**（代襲相続: 甥姪まで） | 第1・2順位がいない場合。再代襲なし |

### 3-2. 法定相続分
| 相続人の組み合わせ | 配偶者の相続分 | 他の相続人の相続分 |
|-------------------|--------------|------------------|
| 配偶者と子 | **1/2** | 子が**1/2**（子が複数の場合は均等に按分） |
| 配偶者と直系尊属 | **2/3** | 直系尊属が**1/3**（複数の場合は均等に按分） |
| 配偶者と兄弟姉妹 | **3/4** | 兄弟姉妹が**1/4**（複数の場合は均等に按分） |
| 配偶者のみ | **全部** | - |
| 子のみ | - | **全部**（複数の場合は均等に按分） |

**注意**: 非嫡出子（婚外子）の相続分は嫡出子と同等（2013年9月5日最高裁決定以降）

## 4. 配偶者の税額軽減

### 4-1. 内容
配偶者が取得した正味の遺産額が、以下のいずれか多い金額までであれば、相続税はかからない:

- **法定相続分相当額**
- **1億6,000万円**

### 4-2. 計算例
- 遺産総額5億円、法定相続人: 配偶者と子2人の場合
  - 配偶者の法定相続分: 5億円 × 1/2 = 2億5,000万円
  - 1億6,000万円 vs 2億5,000万円 → 2億5,000万円まで非課税
  - 配偶者が2億5,000万円以下を取得すれば相続税ゼロ

### 4-3. 適用要件
- 法律上の配偶者であること（内縁は不可）
- 相続税の申告書を提出すること（税額がゼロでも申告が必要）
- 申告期限までに遺産分割が確定していること（未分割の場合は「申告期限後3年以内の分割見込書」を提出して、分割確定後に更正の請求）

### 4-4. 二次相続への影響
- 配偶者の税額軽減を最大限活用すると、配偶者の死亡時（二次相続）の税負担が重くなる場合がある
- 一次相続と二次相続のトータルで最適な遺産分割を検討すべき

## 5. 小規模宅地等の特例

### 5-1. 概要
被相続人等の事業用・居住用の宅地等について、一定の面積まで相続税の課税価格を減額する特例。

### 5-2. 特例の種類と減額割合

| 区分 | 限度面積 | 減額割合 | 主な要件 |
|------|---------|---------|---------|
| **特定居住用宅地等** | **330㎡** | **80%**減額 | 配偶者が取得、同居親族が相続後も居住、家なき子特例（持ち家なし別居親族） |
| **特定事業用宅地等** | **400㎡** | **80%**減額 | 相続人が事業を承継し、申告期限まで事業を継続 |
| **特定同族会社事業用宅地等** | **400㎡** | **80%**減額 | 相続人が同族会社の役員で、申告期限まで事業を継続 |
| **貸付事業用宅地等** | **200㎡** | **50%**減額 | 相続人が貸付事業を承継し、申告期限まで貸付を継続 |

### 5-3. 併用の場合の限度面積調整
- 特定居住用と特定事業用は完全併用可能（330㎡ + 400㎡ = 730㎡まで）
- 貸付事業用と他の宅地を併用する場合は以下の算式で限度面積を調整:
```
特定事業用の面積 × 200/400 + 特定居住用の面積 × 200/330 + 貸付事業用の面積 ≦ 200㎡
```

### 5-4. 家なき子特例（特定居住用宅地等の特例の一形態）
以下の全てを満たす別居親族が取得した場合に適用:
- 被相続人に配偶者がいないこと
- 被相続人と同居していた法定相続人がいないこと
- 相続開始前3年以内に、自己・自己の配偶者・3親等内の親族・特別の関係のある法人の所有する家屋に居住したことがないこと
- 相続開始時に居住している家屋を過去に所有していたことがないこと
- 相続税の申告期限まで保有していること

## 6. 生命保険金の非課税

### 6-1. 非課税限度額
```
非課税限度額 = 500万円 × 法定相続人の数
```

| 法定相続人の数 | 非課税限度額 |
|--------------|-------------|
| 1人 | 500万円 |
| 2人 | 1,000万円 |
| 3人 | 1,500万円 |
| 4人 | 2,000万円 |

### 6-2. 適用要件
- 被相続人が保険料を負担していた生命保険金であること（みなし相続財産）
- 受取人が相続人であること（相続放棄した者が受け取った場合は非課税の適用なし）

---

## 7. 退職手当金の非課税

### 7-1. 非課税限度額
```
非課税限度額 = 500万円 × 法定相続人の数
```
- 生命保険金の非課税とは別枠

### 7-2. 対象
- 被相続人の死亡後3年以内に支給が確定した退職手当金・功労金等
- 弔慰金は別途非課税枠あり:
  - 業務上の死亡: 賞与を除く普通給与の3年分
  - 業務外の死亡: 賞与を除く普通給与の半年分

## 8. 贈与税の基礎控除

### 8-1. 暦年課税の基礎控除
- **年間110万円**（1月1日〜12月31日の1年間）
- 贈与者ごとではなく、受贈者（もらう側）ごとに年間110万円
- 110万円以下の贈与は申告不要

### 8-2. 相続開始前の贈与の加算
- **2024年1月1日以後の贈与**: 相続開始前**7年以内**の贈与を相続財産に加算（段階的に延長）
  - 2024年1月1日〜2026年12月31日の相続開始: 経過措置により3年〜段階的に延長
  - 最終的に2031年以後の相続開始分から完全に7年
  - 延長された4年分（4年前〜7年前）については、合計100万円までは加算しない
- **2023年12月31日以前の贈与**: 従来通り相続開始前3年以内の贈与を加算

- 基礎控除後: 500万円 - 110万円 = 390万円
- 贈与税額: 390万円 × 15% - 10万円 = 48.5万円

## 10. 相続時精算課税制度

### 10-1. 制度の概要
- 原則60歳以上の父母・祖父母から18歳以上の子・孫への贈与に適用
- 選択届出書を提出することで適用（一度選択すると暦年課税に戻れない）
- 贈与者ごとに選択可能（父からの贈与は精算課税、母からの贈与は暦年課税等）

### 10-2. 特別控除
- **累計2,500万円**まで贈与税が非課税
- 2,500万円を超えた部分には一律**20%**の贈与税
- 贈与者が死亡した場合、精算課税の適用を受けた贈与財産を相続財産に加算して相続税を計算
- 既に支払った贈与税は相続税から控除（払い過ぎの場合は還付）

### 10-3. 2024年改正: 年110万円の基礎控除の追加
- **2024年1月1日以後の贈与から適用**
- 相続時精算課税制度を選択した場合でも、**年110万円の基礎控除**が適用される
- 基礎控除（年110万円）の範囲内の贈与は:
  - 贈与税の申告不要
  - 相続財産への加算も不要（従来は全額加算だった）
- 基礎控除を超える部分が累計2,500万円の特別控除の対象

### 10-4. 計算例（2024年以後）
- 父から子へ毎年500万円を5年間贈与（相続時精算課税選択済み）
  - 各年の贈与税: (500万円 - 110万円) = 390万円 → 特別控除の範囲内なら0円
  - 5年間の累計: 390万円 × 5 = 1,950万円（特別控除2,500万円の範囲内）→ 贈与税0円
  - 相続時に加算される金額: 1,950万円（110万円 × 5年 = 550万円は加算不要）

## 11. 事業承継税制（特例措置）

### 11-1. 制度の概要
- 中小企業の株式を後継者に贈与・相続する際の納税猶予制度
- **一般措置**と**特例措置**（2024年3月31日までに特例承継計画を提出した場合）がある

### 11-2. 特例措置の内容
| 項目 | 特例措置 | 一般措置 |
|------|---------|---------|
| 納税猶予割合 | **100%** | 贈与: 100%、相続: 80% |
| 対象株式 | 全株式 | 総株式数の2/3まで |
| 後継者の人数 | 最大3人 | 1人 |
| 雇用確保要件 | 実質撤廃（未達の場合は理由書提出） | 5年間平均で8割維持 |
| 特例承継計画の提出期限 | 2026年3月31日（延長済み） | 不要 |
| 適用期限 | 2027年12月31日まで | 期限なし |

### 11-3. 認定要件
- **会社の要件**: 中小企業者に該当、上場会社でない、風俗営業会社でない、資産管理会社でない
- **先代経営者の要件**: 代表者であったこと、議決権の50%超を保有していたこと等
- **後継者の要件**: 代表者であること、議決権の50%超を保有すること（複数後継者の場合は各10%以上かつ上位3名以内）

### 11-4. 取消事由（主なもの）
- 後継者が代表者を退任した場合
- 後継者の議決権が50%以下になった場合
- 会社が解散した場合
- 会社が資産管理会社に該当した場合
- 株式を譲渡した場合

### 11-5. 個人版事業承継税制
- 個人事業者の事業用資産（土地400㎡まで、建物800㎡まで、その他の減価償却資産）に係る贈与税・相続税の100%納税猶予
- 特例承継計画の提出期限: 2026年3月31日
- 適用期限: 2028年12月31日まで

## 12. 申告期限

### 12-1. 相続税の申告期限
- **相続の開始があったことを知った日の翌日から10ヶ月以内**
- 例: 2024年4月15日に死亡 → 2025年2月15日が申告期限
- 申告期限が土日祝の場合は翌営業日

### 12-2. 贈与税の申告期限
- **贈与を受けた年の翌年2月1日〜3月15日**
- 暦年課税で110万円以下の場合は申告不要
- 相続時精算課税の基礎控除（年110万円）以下の場合は申告不要（2024年以後）

### 12-3. 延納と物納
**延納**:
- 相続税額が10万円を超え、金銭一時納付が困難な場合
- 担保提供が必要（延納税額が100万円以下かつ延納期間3年以下の場合は不要）
- 延納期間: 原則5年以内（不動産等の割合に応じて最長20年）
- 利子税が発生

**物納**:
- 延納によっても金銭納付が困難な場合に限り認められる
- 物納に充てることができる財産の順位:
  1. 国債、地方債、不動産、船舶
  2. 社債、株式、証券投資信託の受益証券
  3. 動産
- 物納財産の収納価額: 相続税の課税価格計算の基礎となった価額

## 13. その他の特例・控除

### 13-1. 未成年者控除
```
控除額 = (18歳 - 相続開始時の年齢) × 10万円
```

### 13-2. 障害者控除
```
一般障害者: (85歳 - 相続開始時の年齢) × 10万円
特別障害者: (85歳 - 相続開始時の年齢) × 20万円
```

### 13-3. 相次相続控除
- 10年以内に2回以上の相続があった場合、前の相続で課された相続税の一定割合を控除
- 1年ごとに10%ずつ逓減

### 13-4. 外国税額控除
- 国外財産について外国で相続税に相当する税が課された場合、二重課税を排除

### 13-5. 贈与税の特例（非課税措置）
| 特例 | 非課税限度額 | 適用期限 |
|------|------------|---------|
| 住宅取得等資金の贈与 | 省エネ等住宅: 1,000万円、それ以外: 500万円 | 2026年12月31日まで |
| 教育資金の一括贈与 | 1,500万円（学校等以外は500万円まで） | 2026年3月31日まで |
| 結婚・子育て資金の一括贈与 | 1,000万円（結婚関連は300万円まで） | 2025年3月31日まで |
