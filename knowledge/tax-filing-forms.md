# 各種届出書・申請書 完全一覧

## 1. 個人事業関連

### 1-1. 個人事業の開業・廃業等届出書
| 項目 | 内容 |
|------|------|
| **様式** | 個人事業の開業・廃業等届出書 |
| **提出先** | 納税地の所轄税務署 |
| **提出期限** | 事業開始等の事実があった日から1ヶ月以内 |
| **添付書類** | なし（マイナンバー確認書類は窓口提示） |
| **届出が必要な場合** | 新規開業、事業所の新設・増設・移転・廃止 |

### 1-2. 所得税の青色申告承認申請書
| 項目 | 内容 |
|------|------|
| **様式** | 所得税の青色申告承認申請書 |
| **提出先** | 納税地の所轄税務署 |
| **提出期限** | 青色申告をしようとする年の3月15日まで（新規開業の場合は開業日から2ヶ月以内）。1月16日以後に開業した場合は開業日から2ヶ月以内 |
| **添付書類** | なし |
| **効果** | 青色申告特別控除（65万円/55万円/10万円）、青色事業専従者給与の必要経費算入、純損失の繰越控除（3年間）・繰戻還付、貸倒引当金の計上、少額減価償却資産の特例 |

### 1-3. 青色事業専従者給与に関する届出書
| 項目 | 内容 |
|------|------|
| **様式** | 青色事業専従者給与に関する届出書（変更届出書） |
| **提出先** | 納税地の所轄税務署 |
| **提出期限** | 青色事業専従者給与額を必要経費に算入しようとする年の3月15日まで（新たに専従者がいることとなった日から2ヶ月以内） |
| **添付書類** | なし |
| **要件** | 青色申告者と生計を一にする配偶者その他の親族で、年齢15歳以上、かつ、もっぱらその事業に従事していること（年間6ヶ月超） |

### 1-4. 所得税の青色申告の取りやめ届出書
| 項目 | 内容 |
|------|------|
| **様式** | 所得税の青色申告の取りやめ届出書 |
| **提出先** | 納税地の所轄税務署 |
| **提出期限** | 青色申告を取りやめようとする年の翌年3月15日まで |
| **添付書類** | なし |

### 1-5. 個人事業の開業・廃業等届出書（廃業届）
| 項目 | 内容 |
|------|------|
| **様式** | 個人事業の開業・廃業等届出書（廃業にチェック） |
| **提出先** | 納税地の所轄税務署 |
| **提出期限** | 廃業の事実があった日から1ヶ月以内 |
| **添付書類** | なし |
| **併せて提出** | 事業廃止届出書（消費税）、給与支払事務所等の廃止届出書、所得税の青色申告の取りやめ届出書（該当する場合） |

### 1-6. 所得税の棚卸資産の評価方法の届出書
| 項目 | 内容 |
|------|------|
| **提出先** | 納税地の所轄税務署 |
| **提出期限** | 最初の確定申告書の提出期限まで |
| **届出がない場合** | 最終仕入原価法により算出した取得価額による原価法 |

### 1-7. 所得税の減価償却資産の償却方法の届出書
| 項目 | 内容 |
|------|------|
| **提出先** | 納税地の所轄税務署 |
| **提出期限** | 最初の確定申告書の提出期限まで |
| **届出がない場合** | 定額法（建物・建物附属設備・構築物は定額法のみ） |

### 1-8. 所得税の納税地の変更に関する届出書
| 項目 | 内容 |
|------|------|
| **提出先** | 変更前の納税地の所轄税務署 |
| **提出期限** | 納税地を変更した後、遅滞なく |
| **添付書類** | なし |

## 2. 法人関連

### 2-1. 法人設立届出書
| 項目 | 内容 |
|------|------|
| **様式** | 法人設立届出書 |
| **提出先** | 本店所在地の所轄税務署 |
| **提出期限** | 設立の日以後2ヶ月以内 |
| **添付書類** | 定款等の写し、登記事項証明書（履歴事項全部証明書）、株主等の名簿の写し、設立趣意書（任意）、設立時の貸借対照表 |

### 2-2. 法人の青色申告の承認申請書
| 項目 | 内容 |
|------|------|
| **様式** | 青色申告の承認申請書 |
| **提出先** | 本店所在地の所轄税務署 |
| **提出期限** | 設立第1期: 設立の日以後3ヶ月を経過した日と最初の事業年度終了の日のいずれか早い日の前日まで。それ以外: 青色申告をしようとする事業年度開始の日の前日まで |
| **添付書類** | なし |
| **効果** | 欠損金の繰越控除（10年間）、欠損金の繰戻還付、少額減価償却資産の特例、各種特別償却・税額控除 |

### 2-3. 給与支払事務所等の開設届出書
| 項目 | 内容 |
|------|------|
| **様式** | 給与支払事務所等の開設・移転・廃止届出書 |
| **提出先** | 給与支払事務所等の所在地の所轄税務署 |
| **提出期限** | 開設の事実があった日から1ヶ月以内 |
| **添付書類** | なし |
| **注意** | 法人設立届出書に給与支払の開始年月日を記載していれば、別途の届出は不要とされるが、実務上は提出が推奨される |

### 2-4. 棚卸資産の評価方法の届出書
| 項目 | 内容 |
|------|------|
| **様式** | 棚卸資産の評価方法の届出書 |
| **提出先** | 本店所在地の所轄税務署 |
| **提出期限** | 設立第1期の確定申告書の提出期限まで |
| **届出がない場合** | 最終仕入原価法による原価法 |
| **選択可能な方法** | 原価法（個別法、先入先出法、総平均法、移動平均法、最終仕入原価法、売価還元法）、低価法 |

### 2-5. 減価償却資産の償却方法の届出書
| 項目 | 内容 |
|------|------|
| **様式** | 減価償却資産の償却方法の届出書 |
| **提出先** | 本店所在地の所轄税務署 |
| **提出期限** | 設立第1期の確定申告書の提出期限まで |
| **届出がない場合** | 建物・建物附属設備・構築物: 定額法（変更不可）。機械装置・車両等: 定率法 |
| **選択可能な方法** | 定額法、定率法（200%定率法）※資産の種類ごとに選択可能 |

### 2-6. 有価証券の一単位当たりの帳簿価額の算出方法の届出書
| 項目 | 内容 |
|------|------|
| **提出先** | 本店所在地の所轄税務署 |
| **提出期限** | 有価証券を取得した日の属する事業年度の確定申告書の提出期限まで |
| **届出がない場合** | 移動平均法による原価法 |

### 2-7. 申告期限の延長の特例の申請書
| 項目 | 内容 |
|------|------|
| **様式** | 申告期限の延長の特例の申請書 |
| **提出先** | 本店所在地の所轄税務署 |
| **提出期限** | 延長を受けようとする事業年度終了の日まで |
| **延長期間** | 最大3ヶ月（会計監査人の監査を受ける場合等は4ヶ月） |
| **注意** | 利子税が発生する（申告期限延長期間に対して年0.9% ※2024年） |

### 2-8. 事前確定届出給与に関する届出書
| 項目 | 内容 |
|------|------|
| **提出先** | 本店所在地の所轄税務署 |
| **提出期限** | 株主総会等の決議をした日から1ヶ月以内、または事業年度開始の日から4ヶ月以内のいずれか早い日 |
| **記載事項** | 届出対象者（役員名）、届出額、届出の支給時期 |
| **注意** | 届出額と異なる金額を支給した場合、全額が損金不算入 |

## 3. 消費税関連

### 3-1. 消費税課税事業者選択届出書
| 項目 | 内容 |
|------|------|
| **様式** | 消費税課税事業者選択届出書 |
| **提出先** | 納税地の所轄税務署 |
| **提出期限** | 課税事業者になろうとする課税期間の開始の日の前日まで（新規開業の場合はその課税期間中） |
| **効力** | 提出した翌課税期間から適用（新規開業の場合はその課税期間から） |
| **制限** | 選択後2年間は免税事業者に戻れない。調整対象固定資産を取得した場合は3年間 |
| **用途** | 免税事業者が還付を受けるために課税事業者を選択する場合（設備投資が多い場合等） |

### 3-2. 消費税課税事業者選択不適用届出書
| 項目 | 内容 |
|------|------|
| **提出先** | 納税地の所轄税務署 |
| **提出期限** | 免税事業者に戻ろうとする課税期間の開始の日の前日まで |
| **制限** | 課税事業者選択後2年間は提出不可 |

### 3-3. 消費税簡易課税制度選択届出書
| 項目 | 内容 |
|------|------|
| **様式** | 消費税簡易課税制度選択届出書 |
| **提出先** | 納税地の所轄税務署 |
| **提出期限** | 簡易課税制度を適用しようとする課税期間の開始の日の前日まで（新規開業の場合はその課税期間中） |
| **適用要件** | 基準期間の課税売上高が5,000万円以下 |
| **制限** | 選択後2年間は原則課税に戻れない |

### 3-4. 消費税簡易課税制度選択不適用届出書
| 項目 | 内容 |
|------|------|
| **提出先** | 納税地の所轄税務署 |
| **提出期限** | 原則課税に戻ろうとする課税期間の開始の日の前日まで |
| **制限** | 簡易課税選択後2年間は提出不可 |

### 3-5. 消費税課税期間特例選択届出書
| 項目 | 内容 |
|------|------|
| **提出先** | 納税地の所轄税務署 |
| **提出期限** | 特例を適用しようとする課税期間の開始の日の前日まで |
| **選択可能な期間** | 1ヶ月ごと、3ヶ月ごと |
| **用途** | 還付申告を早期に行いたい場合 |

### 3-6. 消費税の新設法人に該当する旨の届出書
| 項目 | 内容 |
|------|------|
| **提出先** | 納税地の所轄税務署 |
| **提出期限** | 速やかに |
| **該当要件** | 事業年度開始の日における資本金が1,000万円以上の新設法人 |

### 3-7. 適格請求書発行事業者の登録申請書（インボイス）
| 項目 | 内容 |
|------|------|
| **提出先** | インボイス登録センター（管轄の税務署経由も可） |
| **提出期限** | 登録を受けようとする課税期間の初日から起算して15日前の日まで（経過措置あり） |
| **効力** | 登録日から適格請求書発行事業者となる |
| **注意** | 免税事業者が登録する場合は自動的に課税事業者となる |

### 3-8. 消費税申告期限延長届出書
| 項目 | 内容 |
|------|------|
| **提出先** | 納税地の所轄税務署 |
| **提出期限** | 延長を受けようとする事業年度終了の日の属する課税期間の末日まで |
| **延長期間** | 1ヶ月（法人税の申告期限延長の特例の適用を受けている場合） |
| **注意** | 利子税が発生する |

## 4. 源泉所得税関連

### 4-1. 源泉所得税の納期の特例の承認に関する申請書
| 項目 | 内容 |
|------|------|
| **様式** | 源泉所得税の納期の特例の承認に関する申請書 |
| **提出先** | 給与支払事務所等の所在地の所轄税務署 |
| **提出期限** | 随時（提出した日の翌月に支払う給与等から適用） |
| **適用要件** | 給与の支給人員が常時10人未満 |
| **効果** | 源泉所得税の納付が年2回（7月10日、1月20日）になる |

### 4-2. 源泉所得税の納期の特例の要件に該当しなくなったことの届出書
| 項目 | 内容 |
|------|------|
| **提出先** | 給与支払事務所等の所在地の所轄税務署 |
| **提出期限** | 該当しなくなった場合に速やかに |
| **効果** | 届出書提出の翌月から毎月納付に変更 |

### 4-3. 源泉所得税の納期の特例の適用についての届出書（不適用届出）
| 項目 | 内容 |
|------|------|
| **提出先** | 給与支払事務所等の所在地の所轄税務署 |
| **提出期限** | 随時 |
| **用途** | 10人未満のまま自主的に毎月納付に戻したい場合 |

## 5. 異動届出書

### 5-1. 異動届出書（法人）
| 異動事項 | 提出期限 | 添付書類 |
|---------|---------|---------|
| 本店所在地の変更（管轄内） | 異動後速やかに | 登記事項証明書 |
| 本店所在地の変更（管轄外） | 異動後速やかに（旧所轄税務署に提出） | 登記事項証明書 |
| 代表者の変更 | 異動後速やかに | 登記事項証明書 |
| 商号（社名）の変更 | 異動後速やかに | 登記事項証明書 |
| 事業年度の変更 | 異動後速やかに | 定款の写し（変更後） |
| 資本金の変更 | 異動後速やかに | 登記事項証明書 |
| 事業目的の変更 | 異動後速やかに | 登記事項証明書 |
| 届出書の届出事項の変更 | 異動後速やかに | 変更内容に応じた書類 |

### 5-2. 異動届出書の提出先
- **税務署**: 異動届出書
- **都道府県税事務所**: 法人の届出書（異動届）
- **市区町村**: 法人の届出書（異動届）
- **注意**: 3箇所全てに提出が必要

### 5-3. 個人事業の異動届出
| 異動事項 | 届出書 | 提出期限 |
|---------|--------|---------|
| 納税地の変更 | 所得税の納税地の変更に関する届出書 | 変更後遅滞なく |
| 屋号の変更 | 個人事業の開業・廃業等届出書 | 変更後1ヶ月以内 |
| 事業内容の変更 | 個人事業の開業・廃業等届出書 | 変更後1ヶ月以内 |

## 6. その他の届出書・申請書

### 6-1. 所得税関連
| 届出書・申請書 | 提出期限 | 備考 |
|--------------|---------|------|
| 現金主義の所得計算による旨の届出書 | 青色申告承認申請と同時 | 前々年分の不動産所得・事業所得の合計が300万円以下 |
| 有価証券の一単位当たりの帳簿価額の算出方法の届出書 | 確定申告書の提出期限まで | 届出がない場合は総平均法 |
| 予定納税額の減額申請書 | 第1期: 7月15日まで、第2期: 11月15日まで | 廃業・休業等で所得が減少した場合 |

### 6-2. 法人税関連
| 届出書・申請書 | 提出期限 | 備考 |
|--------------|---------|------|
| 収益事業開始届出書（公益法人等） | 収益事業を開始した日から2ヶ月以内 | |
| 外国普通法人となった旨の届出書 | 内国法人から外国法人になった場合速やかに | |
| 連結納税の承認申請書 | 連結事業年度開始の日の3ヶ月前の日まで | グループ通算制度への移行 |
| グループ通算制度の承認申請書 | 最初に適用する事業年度開始の日の3ヶ月前の日まで | 2022年4月以後開始事業年度から |
| 欠損金の繰戻しによる還付請求書 | 確定申告書と同時 | 青色申告法人のみ |
