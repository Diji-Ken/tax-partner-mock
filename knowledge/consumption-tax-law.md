# 消費税法 完全ガイド（インボイス制度以外）

> 最終更新: 2026年3月
> 根拠法令: 消費税法、消費税法施行令、消費税法施行規則、租税特別措置法
> 本ガイドはインボイス制度（適格請求書等保存方式）の詳細を除く消費税法全般を網羅

---

## 1. 課税事業者・免税事業者の判定

### 1-1. 判定フロー

```
Step 1: 基準期間（前々事業年度）の課税売上高が1,000万円超か？
  → Yes → 課税事業者
  → No → Step 2へ

Step 2: 特定期間の課税売上高（又は給与等支払額）が1,000万円超か？
  → Yes → 課税事業者
  → No → Step 3へ

Step 3: 新設法人で事業年度開始日の資本金が1,000万円以上か？
  → Yes → 課税事業者（設立1期目・2期目）
  → No → Step 4へ

Step 4: 特定新規設立法人に該当するか？
  （課税売上高5億円超の事業者等に50%超支配されている新設法人）
  → Yes → 課税事業者
  → No → Step 5へ

Step 5: 課税事業者選択届出書を提出しているか？
  → Yes → 課税事業者
  → No → 免税事業者

Step 6: 適格請求書発行事業者の登録をしているか？
  → Yes → 課税事業者（登録日から）
  → No → 免税事業者
```

### 1-2. 基準期間

| 事業者 | 基準期間 |
|--------|---------|
| 個人事業者 | 前々年（1月1日〜12月31日） |
| 法人 | 前々事業年度 |

- 基準期間が1年でない場合: 課税売上高を12ヶ月に換算して判定
- 基準期間の課税売上高: 税込経理の場合は税抜換算して判定

### 1-3. 特定期間

| 事業者 | 特定期間 |
|--------|---------|
| 個人事業者 | 前年1月1日〜6月30日 |
| 法人 | 前事業年度開始の日以後6ヶ月間 |

- 課税売上高に代えて給与等支払額で判定することも可能（いずれかが1,000万円超で課税事業者）

### 1-4. 新設法人の特例

**資本金1,000万円以上の新設法人**: 基準期間がない設立1期目・2期目は納税義務免除なし（自動的に課税事業者）

### 1-5. 課税事業者選択届出書

- 提出期限: 適用を受けようとする課税期間の初日の前日まで（個人は前年12月31日まで）
- 新規開業の場合: 開業した課税期間中に提出すればその期間から適用
- 選択後2年間は免税事業者に戻れない（2年縛り）
- 調整対象固定資産を取得した場合: 3年間変更不可

## 2. 課税取引の4要件

消費税の課税対象となる取引は、以下の4要件を全て満たすもの:

| 要件 | 内容 |
|------|------|
| 1. 国内取引 | 国内において行われる取引（内外判定基準あり） |
| 2. 事業者が事業として | 個人事業者は事業用資産の譲渡等、法人はすべての取引 |
| 3. 対価を得て | 無償取引は原則対象外（みなし譲渡を除く） |
| 4. 資産の譲渡・貸付け、役務の提供 | 有形・無形資産の譲渡、貸付け、サービス提供 |

### みなし譲渡（消費税法第4条第5項）

- 個人事業者が棚卸資産又は事業用資産を家事のために消費又は使用した場合
- 法人が資産を役員に贈与した場合

### 内外判定基準

| 取引種類 | 国内取引の判定基準 |
|---------|------------------|
| 資産の譲渡・貸付け（有形） | 引渡し時の所在場所が国内 |
| 資産の譲渡・貸付け（無形: 特許権等） | 登録機関の所在地が国内 |
| 役務の提供 | 役務提供地が国内 |
| 電気通信利用役務の提供 | 役務の提供を受ける者の住所等が国内 |

## 3. 非課税取引（消費税法第6条、別表第一）

消費税の性格上課税対象とすることが適当でないもの、又は社会政策的配慮から非課税とされるもの。全13項目:

### 消費税の性格上非課税とされるもの

| 番号 | 取引内容 | 詳細 |
|------|---------|------|
| 1 | 土地の譲渡及び貸付け | 土地の上に存する権利を含む。ただし1ヶ月未満の貸付け、駐車場等の施設利用は課税 |
| 2 | 有価証券等の譲渡 | 株式、国債、社債、手形等の譲渡。ゴルフ場利用株式等を除く |
| 3 | 支払手段の譲渡 | 銀行券、政府紙幣、硬貨、小切手、約束手形等。収集用は除く |
| 4 | 預貯金の利子・保険料等 | 預貯金利子、貸付金利子、信用保証料、合同運用信託等の信託報酬、保険料、共済掛金等 |
| 5 | 日本郵便株式会社等が行う郵便切手類の譲渡 | 郵便切手、印紙、証紙の譲渡（郵便局等での譲渡に限る） |
| 6 | 地方公共団体等が法令に基づき徴収する手数料等 | 国、地方公共団体等が法令に基づいて行う登記、登録、特許、免許、許可、認可、検査等に係る手数料等 |

### 社会政策的配慮から非課税とされるもの

| 番号 | 取引内容 | 詳細 |
|------|---------|------|
| 7 | 社会保険医療の給付等 | 健康保険法等に基づく医療、介護保険法に基づくサービス等。自由診療は課税 |
| 8 | 介護保険サービス | 介護保険法に基づく居宅サービス、施設サービス等 |
| 9 | 社会福祉事業等として行われる資産の譲渡等 | 社会福祉法に規定する社会福祉事業として行われるサービス |
| 10 | 助産に係る資産の譲渡等 | 医師、助産師等による助産に関するサービス |
| 11 | 火葬料・埋葬料 | 墓地、埋葬等に関する法律に基づく埋葬・火葬 |
| 12 | 身体障害者用物品の譲渡、貸付け等 | 義肢、車いす、補聴器等の一定の身体障害者用物品 |
| 13 | 学校教育 | 学校教育法に基づく学校の授業料、入学金、施設設備費等。学習塾等は課税 |
| ※ | 住宅の貸付け | 契約において住宅用であることが明らかなもの。貸付期間1ヶ月未満は課税 |

## 4. 不課税取引

消費税の課税4要件を満たさない取引。課税の対象外となる。

**主な不課税取引**:

| 取引 | 理由 |
|------|------|
| 給与・賃金の支払い | 雇用契約に基づくもので「事業として」に該当しない |
| 寄附金、祝金、香典、見舞金 | 対価性がない |
| 保険金、共済金の受取り | 資産の譲渡等の対価ではない |
| 株式の配当金 | 株主としての地位に基づく利益配当 |
| 損害賠償金（対価性なし） | 心身・資産に加えられた損害の補填で対価性がない |
| 補助金、助成金 | 特定の政策目的のために交付されるもので対価性がない |
| 国外取引 | 国内において行われる取引ではない |
| 個人の家事用資産の売却 | 事業者が事業として行うものではない |

## 5. 免税取引（輸出免税）

### 5-1. 輸出免税の概要（消費税法第7条）

消費税は国内消費に対する税であるため、輸出取引については消費税が免除される（0%課税）。

**免税と非課税の違い**: 免税取引は課税売上割合の計算上、課税売上に含まれる。仕入税額控除も可能。非課税取引は課税売上に含まれない。

### 5-2. 免税取引の範囲

| 取引 | 具体例 |
|------|--------|
| 本邦からの輸出として行われる資産の譲渡又は貸付け | 商品の輸出 |
| 外国貨物の譲渡又は貸付け | 保税地域内での外国貨物の売買 |
| 国内と国外との間の通信・郵便 | 国際電話、国際郵便 |
| 国内と国外との間の旅客・貨物の輸送 | 国際航空運賃、国際海上運賃 |
| 外航船舶等の譲渡・貸付け・修理 | 遠洋漁業船等 |
| 非居住者に対する一定の役務提供 | 非居住者に対するサービス（国内で直接便益を享受するものを除く） |

### 5-3. 輸出免税の要件

- 輸出許可書、税関長の証明書等の保存が必要
- 輸出物品販売場（免税店）: 外国人旅行者等の非居住者に対する一定の物品の販売

## 6. 税率

### 6-1. 標準税率と軽減税率

| 区分 | 消費税率 | 地方消費税率 | 合計 |
|------|---------|------------|------|
| 標準税率 | 7.8% | 2.2% | **10%** |
| 軽減税率 | 6.24% | 1.76% | **8%** |

### 6-2. 軽減税率（8%）の対象品目

**2019年10月1日から適用**

| 対象品目 | 具体例 | 注意事項 |
|---------|--------|---------|
| 飲食料品の譲渡 | 食品全般、飲料、菓子、調味料等 | 酒類は除く（酒税法に規定する酒類） |
| | テイクアウト・宅配 | 軽減税率8% |
| | 外食（店内飲食） | 標準税率10%（食事の提供に該当） |
| | ケータリング・出張料理 | 標準税率10%（有料老人ホーム等での一定の食事提供を除く） |
| | 一体資産（食品+食品以外のセット商品） | 税抜価格1万円以下かつ食品の価額割合が2/3以上なら軽減税率 |
| 定期購読契約の新聞 | 週2回以上発行される新聞の定期購読契約 | 駅売り・コンビニ売りは標準税率10% |

**判定ポイント**:
- 「飲食料品」: 食品表示法に規定する食品（人の飲用又は食用に供されるもの）
- 医薬品・医薬部外品は食品に該当しないため標準税率10%
- みりん（酒類）は標準税率10%、みりん風調味料は軽減税率8%
- ミネラルウォーターは軽減税率8%、水道水は標準税率10%
- ペットフードは軽減税率対象外（人の飲食用ではない）→ 標準税率10%

## 7. 本則課税の計算方法

### 7-1. 基本計算式

```
納付税額 = 課税売上に係る消費税額 - 仕入税額控除額
```

### 7-2. 課税売上に係る消費税額

```
課税標準額 = 税込課税売上高 × 100/110（標準税率分）
            + 税込課税売上高 × 100/108（軽減税率分）

課税売上に係る消費税額 = 課税標準額（標準税率分）× 7.8%
                      + 課税標準額（軽減税率分）× 6.24%
```

- 課税標準額は1,000円未満切り捨て

### 7-3. 仕入税額控除

課税仕入れ等に係る消費税額を控除する。

**仕入税額控除の要件**:
1. 帳簿の保存（一定の記載事項を満たすもの）
2. 請求書等の保存（適格請求書等保存方式に基づく適格請求書）
3. 課税仕入れの事実があること

---

## 8. 課税売上割合と仕入税額控除の制限

### 8-1. 課税売上割合

```
課税売上割合 = 課税期間の課税売上高（税抜）+ 免税売上高
              ÷ 課税期間の総売上高（課税売上高 + 免税売上高 + 非課税売上高）（税抜）
```

### 8-2. 仕入税額控除の計算方法

| 課税売上割合 | 計算方法 |
|-------------|---------|
| **95%以上**かつ課税売上高5億円以下 | 課税仕入れに係る消費税額の**全額**を控除 |
| **95%未満**又は課税売上高5億円超 | 個別対応方式又は一括比例配分方式のいずれかで計算 |

### 8-3. 個別対応方式

課税仕入れを3つに区分して計算:

| 区分 | 控除額 |
|------|--------|
| 課税売上にのみ対応するもの | 全額控除 |
| 非課税売上にのみ対応するもの | 控除不可 |
| 課税・非課税売上に共通するもの | 課税売上割合を乗じた金額を控除 |

```
仕入税額控除額 = 課税売上対応分の全額
              + 共通対応分 × 課税売上割合
```

### 8-4. 一括比例配分方式

```
仕入税額控除額 = 課税仕入れに係る消費税額の合計 × 課税売上割合
```

**注意**: 一括比例配分方式を選択した場合、2年間は個別対応方式に変更不可

### 8-5. 課税売上割合に準ずる割合

所轄税務署長の承認を受けた場合、課税売上割合に代えて合理的な割合（使用面積割合、従業員割合等）を使用可能

## 9. 簡易課税制度（消費税法第37条）

### 9-1. 適用要件

1. 基準期間の課税売上高が**5,000万円以下**
2. 「消費税簡易課税制度選択届出書」を適用を受けようとする課税期間の初日の前日までに提出

### 9-2. 計算方法

```
納付税額 = 課税売上に係る消費税額 - 課税売上に係る消費税額 × みなし仕入率
         = 課税売上に係る消費税額 × （1 - みなし仕入率）
```

### 9-3. みなし仕入率（6事業区分）

| 事業区分 | 該当事業 | みなし仕入率 |
|---------|---------|------------|
| **第1種事業** | 卸売業 | **90%** |
| **第2種事業** | 小売業、農林水産業（食用） | **80%** |
| **第3種事業** | 農林水産業（非食用）、鉱業、建設業、製造業、電気業、ガス業、熱供給業、水道業 | **70%** |
| **第4種事業** | 第1種〜第3種、第5種、第6種以外の事業（飲食店業等） | **60%** |
| **第5種事業** | 運輸通信業、金融保険業、サービス業（飲食店業を除く） | **50%** |
| **第6種事業** | 不動産業 | **40%** |

### 9-4. 2種以上の事業を営む場合の計算

#### 原則法

各事業区分ごとに課税売上に係る消費税額を区分し、それぞれにみなし仕入率を適用

```
仕入控除税額 = 第1種の消費税額 × 90% + 第2種の消費税額 × 80% + ...
```

#### 75%ルール（特例計算）

1つの事業の課税売上高が全体の75%以上を占める場合:
- その事業のみなし仕入率を全体に適用可能

2つの事業の課税売上高の合計が全体の75%以上を占める場合:
- その2つの事業のうち、みなし仕入率の高い方の事業以外の課税売上高に対して、その高い方のみなし仕入率を適用可能

#### 事業区分をしていない場合

区分されていない課税売上高については、その区分されていない事業のうち最も低いみなし仕入率を適用

### 9-5. 簡易課税の注意事項

- 選択後**2年間**は取りやめ不可
- 調整対象固定資産を取得した場合は一定期間変更不可
- 「消費税簡易課税制度選択不適用届出書」の提出: やめようとする課税期間の初日の前日まで
- 簡易課税では還付は発生しない（輸出業者等は本則課税が有利）

## 10. 2割特例（インボイス発行事業者となる小規模事業者に対する負担軽減措置）

### 10-1. 概要

免税事業者がインボイス発行事業者となった場合の経過措置

### 10-2. 適用期間

**2023年（令和5年）10月1日 〜 2026年（令和8年）9月30日** を含む課税期間

### 10-3. 計算方法

```
納付税額 = 課税売上に係る消費税額 × 20%（= 課税売上に係る消費税額 - 課税売上に係る消費税額 × 80%）
```

- みなし仕入率80%相当の控除
- 業種を問わず一律

### 10-4. 適用要件

- インボイス制度の開始に伴い課税事業者となった事業者
- 基準期間の課税売上高が1,000万円以下
- 事前届出不要（確定申告書に適用する旨を記載するだけ）
- 簡易課税との選択適用可能

## 11. 中間申告（消費税法第42条〜第44条）

### 11-1. 中間申告の回数と納付額

直前の課税期間の年税額（国税分）に応じて中間申告が必要:

| 直前の課税期間の年税額（国税分） | 中間申告回数 | 各回の納付額 | 申告・納付期限 |
|-------------------------------|------------|------------|--------------|
| 48万円以下 | 不要 | - | - |
| 48万円超〜400万円以下 | **年1回** | 年税額の1/2 | 課税期間開始後6ヶ月経過日から2ヶ月以内 |
| 400万円超〜4,800万円以下 | **年3回** | 年税額の1/4 | 各3ヶ月経過後2ヶ月以内 |
| 4,800万円超 | **年11回** | 年税額の1/12 | 各月経過後2ヶ月以内 |

### 11-2. 仮決算による中間申告

前期の実績に基づく中間申告に代えて、中間申告対象期間を一課税期間とみなして仮決算を行い、実際の金額で中間申告することも可能

- 仮決算で計算した納付税額が前期実績に基づく金額を超える場合でも、仮決算の金額で申告可能
- 仮決算による中間申告で還付となる場合は、中間申告による還付はなし（0円として申告可能）

### 11-3. 任意の中間申告

年税額が48万円以下の事業者でも、「任意の中間申告書を提出する旨の届出書」を提出することで、年1回の中間申告が可能（資金繰り平準化のため）

## 12. 届出書一覧

### 12-1. 課税事業者関連

| 届出書 | 提出期限 | 効力発生 |
|--------|---------|---------|
| 消費税課税事業者届出書（基準期間用） | 速やかに | 基準期間の課税売上高が1,000万円超となった課税期間 |
| 消費税課税事業者届出書（特定期間用） | 速やかに | 特定期間の判定により課税事業者となった課税期間 |
| 消費税の納税義務者でなくなった旨の届出書 | 速やかに | 基準期間の課税売上高が1,000万円以下となった課税期間（ただし特定期間等の判定あり） |
| 消費税課税事業者選択届出書 | 適用を受けようとする課税期間の初日の前日まで | 届出書提出日の属する課税期間の翌課税期間から（新規開業は開業課税期間から） |
| 消費税課税事業者選択不適用届出書 | やめようとする課税期間の初日の前日まで | 届出書提出日の属する課税期間の翌課税期間から |

### 12-2. 簡易課税関連

| 届出書 | 提出期限 | 効力発生 |
|--------|---------|---------|
| 消費税簡易課税制度選択届出書 | 適用を受けようとする課税期間の初日の前日まで | 届出書提出日の属する課税期間の翌課税期間から（新規開業は開業課税期間から） |
| 消費税簡易課税制度選択不適用届出書 | やめようとする課税期間の初日の前日まで | 届出書提出日の属する課税期間の翌課税期間から |

### 12-3. その他の届出書

| 届出書 | 提出期限 | 備考 |
|--------|---------|------|
| 消費税課税期間特例選択届出書 | 短縮しようとする課税期間の初日の前日まで | 1ヶ月又は3ヶ月ごとの課税期間を選択 |
| 消費税課税期間特例選択不適用届出書 | 元に戻そうとする課税期間の初日の前日まで | 2年間は変更不可 |
| 任意の中間申告書を提出する旨の届出書 | 適用を受けようとする6ヶ月の期間の末日まで | 年税額48万円以下でも中間申告可能に |
| 任意の中間申告書を提出することの取りやめ届出書 | 取りやめようとする6ヶ月の期間の末日まで | |
| 消費税の新設法人に該当する旨の届出書 | 速やかに | 資本金1,000万円以上の新設法人 |
| 高額特定資産の取得等に係る届出書 | 速やかに | 1,000万円以上の棚卸資産・調整対象固定資産取得時 |

### 12-4. 届出書提出時の注意事項

- 届出書の効力は原則として翌課税期間から発生（当期からの適用は不可が原則）
- **「届出書の提出日」と「効力発生日」は異なる**ことに注意
- 届出書の提出を失念した場合の救済措置は原則としてない（宥恕規定あり: 災害等やむを得ない理由がある場合）
- 適格請求書発行事業者の登録申請書は別途対応が必要

## 13. 会計処理（税込経理方式と税抜経理方式）

### 13-1. 税込経理方式

消費税額を売上・仕入の金額に含めて処理する方法

**仕訳例**（商品110,000円（税込）を現金で販売）:
```
（借方）現金 110,000 ／（貸方）売上 110,000
```

**仕訳例**（商品55,000円（税込）を現金で仕入）:
```
（借方）仕入 55,000 ／（貸方）現金 55,000
```

**決算時**: 確定した消費税額を「租税公課」として費用計上（還付の場合は「雑収入」）
```
（借方）租税公課 ×××  ／（貸方）未払消費税等 ×××
```

### 13-2. 税抜経理方式

消費税額を売上・仕入の金額と区分して処理する方法

**仕訳例**（商品110,000円（税込）を現金で販売）:
```
（借方）現金 110,000 ／（貸方）売上 100,000
                            仮受消費税等 10,000
```

**仕訳例**（商品55,000円（税込）を現金で仕入）:
```
（借方）仕入 50,000      ／（貸方）現金 55,000
      仮払消費税等 5,000
```

**決算時**: 仮受消費税等と仮払消費税等を相殺し、差額を未払消費税等（又は未収消費税等）に振替
```
（借方）仮受消費税等 ×××  ／（貸方）仮払消費税等 ×××
                               未払消費税等 ×××
                               （差額は雑収入又は雑損失）
```

### 13-3. 税込・税抜経理方式の選択

| 事業者区分 | 選択可能な方式 |
|-----------|-------------|
| 課税事業者 | 税込経理方式 又は 税抜経理方式（選択可） |
| 免税事業者 | 税込経理方式のみ |

**影響の違い**:
| 項目 | 税込経理 | 税抜経理 |
|------|---------|---------|
| 固定資産の取得価額 | 消費税込み（減価償却費が大きい） | 消費税抜き |
| 少額減価償却資産の判定 | 税込金額で判定 | 税抜金額で判定 |
| 交際費の損金不算入額 | 税込金額で判定 | 税抜金額で判定 |
| 期末の損益 | 租税公課で調整 | 仮受・仮払の差額で調整 |

## 14. 調整対象固定資産・高額特定資産

### 14-1. 調整対象固定資産

税抜100万円以上の棚卸資産以外の資産（建物、機械、車両等）

取得した課税期間から3年間:
- 免税事業者になれない
- 簡易課税を選択できない
- 課税売上割合が著しく変動した場合に仕入税額控除の調整が必要

### 14-2. 高額特定資産

税抜1,000万円以上の棚卸資産又は調整対象固定資産

取得した課税期間から3年間:
- 免税事業者になれない
- 簡易課税を選択できない

## 15. 国境を越えた役務の提供に係る消費税（リバースチャージ方式）

### 15-1. 電気通信利用役務の提供

| 区分 | 課税方式 | 例 |
|------|---------|-----|
| 事業者向け（B2B） | リバースチャージ方式（役務の提供を受けた事業者が申告納税） | クラウドサービス、広告配信等の事業者向けサービス |
| 消費者向け（B2C） | 国外事業者申告納税方式 | 電子書籍、音楽配信、動画配信等 |

### 15-2. リバースチャージ方式の経過措置

当分の間、課税売上割合が95%以上の事業者は、リバースチャージ方式による申告納税の適用なし
