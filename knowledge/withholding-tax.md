# 源泉徴収実務ガイド

## 1. 源泉徴収の対象となる所得の一覧

源泉徴収義務者（給与等の支払者）は、以下の所得を支払う際に所得税及び復興特別所得税を天引きして国に納付する義務がある。

### 1-1. 給与所得
- 給料、賞与、各種手当
- 税額は「給与所得の源泉徴収税額表」（月額表・日額表・賞与に対する源泉徴収税額の算出率の表）により算出
- 甲欄（扶養控除等申告書提出者）、乙欄（未提出者）、丙欄（日雇い）の区分あり

### 1-2. 退職所得
- 退職金、一時恩給等
- 「退職所得の受給に関する申告書」提出の有無で税額計算が異なる
  - 提出あり: (退職金 - 退職所得控除額) × 1/2 × 税率
  - 提出なし: 退職金 × 20.42%

### 1-3. 報酬・料金等（所得税法第204条第1項）
- 後述の各区分参照

### 1-4. 利子所得
- 公社債の利子、預貯金の利子等
- 源泉分離課税: 15.315%（所得税15% + 復興特別所得税0.315%）+ 住民税5% = 合計20.315%

### 1-5. 配当所得
- 上場株式等の配当: 15.315% + 住民税5% = 20.315%
- 非上場株式の配当: 20.42%（住民税なし）

### 1-6. 公的年金等
- 老齢年金等
- 「公的年金等の受給者の扶養親族等申告書」に基づき源泉徴収

### 1-7. 特定口座内の株式等の譲渡所得（源泉徴収選択口座）
- 15.315% + 住民税5% = 20.315%

## 2. 報酬・料金等への源泉徴収

### 2-1. 税理士報酬
| 支払金額 | 源泉徴収税率 |
|----------|-------------|
| 100万円以下の部分 | 10.21%（所得税10% + 復興特別所得税0.21%） |
| 100万円超の部分 | 20.42%（所得税20% + 復興特別所得税0.42%） |

**計算例:**
- 税理士報酬150万円の場合
  - 100万円 × 10.21% = 102,100円
  - 50万円 × 20.42% = 102,100円
  - 合計源泉徴収税額: 204,200円
  - 手取額: 1,295,800円

**注意:**
- 消費税が明確に区分されている場合は、税抜金額に対して源泉徴収する
- 消費税込みで請求されている場合は、税込金額に対して源泉徴収する
- 請求書に消費税額を明記させることが重要

### 2-2. 弁護士・司法書士・社労士等の報酬

| 対象 | 税率 | 特記事項 |
|------|------|---------|
| 弁護士報酬 | 10.21%（100万円超: 20.42%） | 顧問料・着手金・成功報酬全て対象 |
| 司法書士報酬 | (支払金額 - 1回の支払につき1万円) × 10.21% | 1万円控除の特例あり |
| 社会保険労務士報酬 | 10.21%（100万円超: 20.42%） | |
| 公認会計士報酬 | 10.21%（100万円超: 20.42%） | |
| 不動産鑑定士報酬 | 10.21%（100万円超: 20.42%） | |
| 土地家屋調査士報酬 | 10.21%（100万円超: 20.42%） | |
| 行政書士報酬 | **源泉徴収不要** | 所得税法204条の対象外 |
| 海事代理士報酬 | 10.21%（100万円超: 20.42%） | |
| 測量士報酬 | 10.21%（100万円超: 20.42%） | |
| 建築士報酬 | 10.21%（100万円超: 20.42%） | |

### 2-3. 原稿料・講演料
- 税率: 10.21%（100万円超: 20.42%）
- 対象: 原稿料、挿絵料、作曲料、脚本料、放送謝金、著作権使用料、講演料等
- 同一人に対し1回に支払う金額で判定

### 2-4. デザイン料
- 税率: 10.21%（100万円超: 20.42%）
- 対象: ロゴデザイン、パッケージデザイン、広告デザイン等
- 法人への支払いは源泉徴収不要（個人への支払いのみ対象）

### 2-5. 外交員報酬
- 税率: (支払金額 - 12万円 ※月額) × 10.21%
- 固定給部分がある場合は、固定給部分を差し引いた金額から12万円を控除
- 12万円控除は月額の場合であり、日割り計算は行わない

### 2-6. その他の対象報酬
| 種類 | 税率 |
|------|------|
| プロスポーツ選手の報酬 | 10.21%（100万円超: 20.42%） |
| モデル・タレントの報酬 | 10.21%（100万円超: 20.42%） |
| ホステス等の報酬 | (支払金額 - 5,000円 × 計算期間の日数) × 10.21% |
| プロボクサーの報酬 | (支払金額 - 5万円) × 10.21% |
| 芸能人の出演料 | 10.21%（100万円超: 20.42%） |
| バー・キャバレー等のホステス | 上記ホステス等参照 |

## 3. 非居住者への支払い

### 3-1. 基本税率
- **20.42%**（所得税20% + 復興特別所得税0.42%）
- 国内源泉所得に該当する場合に源泉徴収が必要

### 3-2. 対象となる国内源泉所得
| 所得の種類 | 税率 |
|-----------|------|
| 事業の対価 | 20.42% |
| 人的役務の提供事業の対価 | 20.42% |
| 不動産の賃貸料 | 20.42% |
| 利子 | 15.315% |
| 配当 | 20.42% |
| 使用料（著作権・特許権等） | 20.42% |
| 給与・報酬 | 20.42% |
| 退職手当等 | 20.42% |
| 不動産の譲渡対価 | 10.21% |
| 年金 | 20.42% |

### 3-3. 租税条約の適用
- 日本が締結している租税条約により、税率が軽減・免除される場合がある
- 適用を受けるには「租税条約に関する届出書」を支払日の前日までに所轄税務署に提出
- 主な条約の軽減税率例:
  - 米国: 使用料0%、配当10%
  - 中国: 使用料10%、配当10%
  - 英国: 使用料0%、配当10%

## 4. 源泉徴収票の種類と作成方法

### 4-1. 給与所得の源泉徴収票
- **交付期限**: 退職後1ヶ月以内、年末調整後翌年1月31日まで
- **作成対象**: 全ての給与受給者
- **記載事項**:
  - 支払金額（年間の給与・賞与の総額）
  - 給与所得控除後の金額
  - 所得控除の額の合計額
  - 源泉徴収税額
  - 社会保険料等の金額
  - 扶養親族等の人数・氏名
  - 各種控除の内容（生命保険料控除、地震保険料控除、住宅ローン控除等）
  - マイナンバー（税務署提出用のみ記載、本人交付用には記載不要）

### 4-2. 退職所得の源泉徴収票
- **交付期限**: 退職後1ヶ月以内
- **記載事項**:
  - 支払金額、退職所得控除額、勤続年数
  - 源泉徴収税額、特別徴収税額（住民税）

## 5. 法定調書の種類（全6種類の詳細）

### 5-1. 給与所得の源泉徴収票（給与支払報告書）
- **提出基準**:
  - 年末調整した受給者: 年間支払額500万円超（役員は150万円超）
  - 年末調整しなかった受給者: 年間支払額250万円超（退職者は50万円超、乙欄適用者は50万円超）
- **提出先**: 税務署（源泉徴収票）、市区町村（給与支払報告書 ※全員分）

### 5-2. 退職所得の源泉徴収票（特別徴収票）
- **提出基準**: 法人の役員に対して支払った退職手当等（金額制限なし）
- **注意**: 役員以外の従業員は税務署への提出不要（本人交付のみ）

### 5-3. 報酬、料金、契約金及び賞金の支払調書
- **提出基準**:
  - 外交員報酬: 年間50万円超
  - 税理士等の報酬: 年間5万円超
  - 原稿料・講演料: 年間5万円超
  - 社会保険診療報酬支払基金が支払う診療報酬: 年間50万円超
  - プロスポーツ選手の報酬: 年間5万円超
  - 芸能人等の報酬: 年間5万円超
  - ホステス等の報酬: 年間50万円超
  - 広告宣伝のための賞金: 年間50万円超
  - 馬主に支払う競馬の賞金: 年間75万円超

### 5-4. 不動産の使用料等の支払調書
- **提出基準**: 同一人に対する年間支払額が15万円超
- **対象**: 地代、家賃、権利金、更新料、承諾料、名義書換料等
- **注意**: 法人に対する支払いも提出対象（個人のみではない）
- **除外**: 不動産業者である個人に対する支払いで、主として建物の賃貸借の代理・仲介を目的とするもの

### 5-5. 不動産等の譲受けの対価の支払調書
- **提出基準**: 同一人に対する年間支払額が100万円超
- **対象**: 不動産、不動産の上に存する権利、船舶の譲受けの対価

### 5-6. 不動産等の売買又は貸付けのあっせん手数料の支払調書
- **提出基準**: 同一人に対する年間支払額が15万円超
- **対象**: 不動産等の売買・貸付けのあっせん手数料

## 6. 法定調書合計表の作成・提出

### 6-1. 正式名称
「給与所得の源泉徴収票等の法定調書合計表」

### 6-2. 提出期限
- **毎年1月31日**（土日の場合は翌営業日）

### 6-3. 提出先
- 源泉徴収義務者の所轄税務署

### 6-4. 記載内容
合計表には以下の6種類の法定調書の枚数と支払金額の合計を記載:
1. 給与所得の源泉徴収票の合計（人員、支払金額、源泉徴収税額等）
2. 退職所得の源泉徴収票の合計
3. 報酬、料金、契約金及び賞金の支払調書の合計
4. 不動産の使用料等の支払調書の合計
5. 不動産等の譲受けの対価の支払調書の合計
6. 不動産等の売買又は貸付けのあっせん手数料の支払調書の合計

### 6-5. 提出方法
- 書面提出
- e-Tax（前々年の法定調書の枚数が100枚以上の場合は電子提出義務）
- 光ディスク等

## 8. マイナンバーの収集・管理・記載方法

### 8-1. 収集対象
- 従業員及びその扶養親族
- 報酬・料金等の支払先（個人）
- 不動産の賃貸人（個人）
- 株主等（個人）

### 8-2. 本人確認の方法
マイナンバーの収集時には「番号確認」と「身元確認」の2つが必要:

| 確認方法 | 番号確認 | 身元確認 |
|---------|---------|---------|
| パターン1 | マイナンバーカード（表面） | マイナンバーカード（裏面） |
| パターン2 | 通知カード | 運転免許証 or パスポート |
| パターン3 | マイナンバー記載の住民票 | 運転免許証 or パスポート |

**雇用関係にある場合の特例**: 2回目以降は身元確認省略可

### 8-3. 管理方法
- **安全管理措置**（個人情報保護委員会ガイドラインに基づく）:
  - 基本方針の策定
  - 取扱規程の策定
  - 組織的安全管理措置（責任者の設置、取扱状況の記録）
  - 人的安全管理措置（従業者への教育・監督）
  - 物理的安全管理措置（施錠管理、持ち出し制限）
  - 技術的安全管理措置（アクセス制御、暗号化）
- **中小規模事業者の特例**: 従業員100人以下の場合、簡易な管理方法が認められる

### 8-4. 記載方法
- **税務署提出用**: マイナンバーを記載する
- **本人交付用**: マイナンバーを記載**しない**（源泉徴収票）
- **市区町村提出用（給与支払報告書）**: マイナンバーを記載する
- **法人番号**: 支払者欄に法人番号（13桁）を記載

### 8-5. 保管と廃棄
- 法定調書作成事務に必要がなくなった場合は速やかに廃棄
- 保管期限: 法定調書の提出期限の翌年1月10日まで（実務上は7年間保管が一般的）
- 廃棄方法: シュレッダー、焼却、溶解等

## 9. 納期の特例

### 9-1. 適用要件
- **給与の支給人員が常時10人未満**であること
- 「源泉所得税の納期の特例の承認に関する申請書」を所轄税務署に提出

### 9-2. 納付期限

| 対象月 | 通常の納付期限 | 納期の特例の納付期限 |
|--------|-------------|-------------------|
| 1月〜6月分 | 各月の翌月10日 | **7月10日** |
| 7月〜12月分 | 各月の翌月10日 | **翌年1月20日** |

### 9-3. 対象となる源泉所得税
- 給与・賞与に係る源泉所得税
- 退職手当等に係る源泉所得税
- 税理士・弁護士等の報酬に係る源泉所得税

### 9-4. 対象外の源泉所得税（毎月納付が必要）
- 原稿料・講演料
- デザイン料
- 不動産の賃借料から天引きした源泉所得税
- 非居住者に対する支払い

### 9-5. 特例の不適用届出
- 常時10人以上になった場合は「源泉所得税の納期の特例の要件に該当しなくなったことの届出書」を提出
- 届出書提出後の翌月から毎月納付に切り替わる

## 10. 源泉所得税の納付書の書き方

### 10-1. 納付書の種類
| 納付書 | 用途 |
|--------|------|
| 所得税徴収高計算書（一般用） | 毎月納付用 |
| 所得税徴収高計算書（納期特例用） | 納期の特例適用者用 |

### 10-2. 一般用の記載方法
1. **税務署名**: 所轄税務署名を記載
2. **整理番号**: 税務署から通知された番号
3. **納期等の区分**: 支払った月（例: 0401〜0430 → 4月分）
4. **区分**: 該当する所得の種類にチェック
   - 俸給・給料等: 01
   - 賞与: 02
   - 日雇労務者の賃金: 03
   - 退職手当等: 04
   - 税理士等の報酬: 05
   - 役員賞与: 08
5. **支払年月日**: 実際に支払った日
6. **人員**: 支払った人数（延べ人数ではなく実人数）
7. **支給額（税込）**: 源泉徴収前の支払総額
8. **税額**: 源泉徴収税額の合計
9. **本税**: 上記税額と同じ
10. **合計額**: 本税の合計（延滞税がある場合はその合計）

### 10-3. 納付方法
- 金融機関の窓口（納付書持参）
- e-Tax（ダイレクト納付、インターネットバンキング）
- クレジットカード納付（国税クレジットカードお支払サイト）
- スマホアプリ納付（PayPay、d払い、au PAY等 ※30万円以下）
- コンビニ納付（バーコード付納付書 ※30万円以下）

### 10-4. 納付が遅れた場合のペナルティ
- **不納付加算税**: 納付税額の10%（自主的に納付した場合は5%）
- **延滞税**: 納期限の翌日から計算
  - 2ヶ月以内: 年2.4%（2024年）
  - 2ヶ月超: 年8.7%（2024年）
- **注意**: 法定納期限から1ヶ月以内に自主的に納付し、過去1年間に期限後納付がない場合は不納付加算税免除

## 11. 実務上のチェックリスト

### 毎月の業務
- [ ] 給与計算と源泉徴収税額の算出
- [ ] 報酬・料金等の支払い時の源泉徴収
- [ ] 源泉所得税の納付（翌月10日まで）※納期特例の場合は半年ごと

### 半年ごとの業務（納期特例の場合）
- [ ] 1月〜6月分の源泉所得税納付（7月10日まで）
- [ ] 7月〜12月分の源泉所得税納付（翌年1月20日まで）

### 年次業務（1月）
- [ ] 年末調整の実施
- [ ] 源泉徴収票の作成・交付
- [ ] 法定調書の作成
- [ ] 法定調書合計表の作成・提出（1月31日まで）
- [ ] 給与支払報告書の市区町村への提出（1月31日まで）
