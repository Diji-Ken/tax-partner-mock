# 主要補助金・支援制度一覧

> 税理士・会計事務所向けCRMナレッジ
> 最終更新: 2026-03-23

---

## 概要

中小企業が活用できる主要な補助金・助成金・支援制度の一覧。
税理士事務所がクライアントに提案・案内する際のリファレンスとして使用する。

**注意事項**:
- 補助金・助成金の公募要領は毎年度変更される。最新情報は各制度の公式サイトで確認すること
- 金額・補助率は2025年度〜2026年度の実績に基づく目安であり、今後の公募回で変更される可能性がある
- 申請期限は公募回ごとに異なる

## 1. IT導入補助金

### 制度概要

| 項目 | 内容 |
|------|------|
| 目的 | 中小企業・小規模事業者がITツールを導入する際の費用を補助 |
| 実施機関 | 中小企業庁（事務局: 一般社団法人サービスデザイン推進協議会等） |
| 公式サイト | https://it-shien.smrj.go.jp/ |

### 申請類型（2025年度実績ベース）

| 類型 | 対象 | 補助率 | 補助額 |
|------|------|--------|--------|
| 通常枠 | ソフトウェア、クラウド利用料、導入関連費等 | 1/2以内 | 1万円〜150万円未満 / 150万円〜450万円以下 |
| インボイス枠（インボイス対応類型） | 会計・受発注・決済ソフト（インボイス対応） | 中小企業: 3/4以内、小規模事業者: 4/5以内（50万円以下の部分）、2/3以内（50万円超部分） | 〜350万円 |
| インボイス枠（電子取引類型） | 受発注ソフト（発注側が負担） | 中小企業: 2/3以内 | 〜350万円 |
| セキュリティ対策推進枠 | サイバーセキュリティ対策サービス | 1/2以内 | 5万円〜100万円 |
| 複数社連携IT導入枠 | 商業集積地等の複数社連携ITツール導入 | 1/2〜2/3以内 | 〜3,000万円 |

### 対象者

| 条件 | 内容 |
|------|------|
| 中小企業 | 業種ごとの資本金・従業員数の条件を満たす企業 |
| 小規模事業者 | 商業・サービス業: 常時使用する従業員5人以下、製造業等: 20人以下 |
| その他 | 日本国内で事業を行っていること、gBizIDプライムを取得していること |

### 税理士事務所がクライアントに案内すべきポイント

- **会計ソフト導入は最も活用しやすい類型**: インボイス枠で会計ソフト・請求書ソフトの導入が可能
- **クラウド利用料も対象**: 最大2年分のクラウド利用料が補助対象
- **IT導入支援事業者の選定が必要**: 事前にIT導入支援事業者として登録されたベンダーのツールのみ対象
- **gBizIDプライムの事前取得**: 申請にはgBizIDプライムが必要（取得に2〜3週間）
- **事業計画の策定支援**: 顧問先の業務課題を整理し、最適なITツールを提案

## 2. ものづくり補助金（ものづくり・商業・サービス生産性向上促進補助金）

### 制度概要

| 項目 | 内容 |
|------|------|
| 目的 | 革新的サービス開発・試作品開発・生産プロセスの改善のための設備投資を支援 |
| 実施機関 | 中小企業庁（事務局: 全国中小企業団体中央会） |
| 公式サイト | https://portal.monodukuri-hojo.jp/ |

### 申請類型（目安）

| 類型 | 補助上限額 | 補助率 |
|------|-----------|--------|
| 通常枠 | 750万円〜1,250万円（従業員数による） | 1/2（小規模事業者2/3） |
| 回復型賃上げ・雇用拡大枠 | 同上 | 2/3 |
| デジタル枠 | 同上 | 2/3 |
| グリーン枠 | 750万円〜4,000万円 | 2/3 |
| グローバル市場開拓枠 | 3,000万円 | 1/2（小規模事業者2/3） |

### 対象経費

- 機械装置・システム構築費
- 技術導入費
- 専門家経費
- 運搬費
- クラウドサービス利用費
- 原材料費
- 外注費
- 知的財産権等関連経費

### 税理士事務所がクライアントに案内すべきポイント

- **事業計画の審査がある**: 付加価値額年率3%以上、給与支給総額年率1.5%以上の向上等の計画が必要
- **賃上げ要件**: 事業計画期間中に給与支給総額を年率1.5%以上増加させる計画が必要
- **設備投資の減価償却**: 補助金で取得した資産の会計処理（圧縮記帳の検討）
- **加点項目の活用**: 経営革新計画の承認、事業継続力強化計画の認定等で加点
- **電子申請のみ**: gBizIDプライムでの電子申請が必須

## 3. 小規模事業者持続化補助金

### 制度概要

| 項目 | 内容 |
|------|------|
| 目的 | 小規模事業者の販路開拓・業務効率化の取組みを支援 |
| 実施機関 | 中小企業庁（事務局: 日本商工会議所・全国商工会連合会） |
| 公式サイト | https://s23.jizokukahojokin.info/ |

### 申請類型（目安）

| 類型 | 補助上限額 | 補助率 |
|------|-----------|--------|
| 通常枠 | 50万円 | 2/3 |
| 賃金引上げ枠 | 200万円 | 2/3（赤字事業者は3/4） |
| 卒業枠 | 200万円 | 2/3 |
| 後継者支援枠 | 200万円 | 2/3 |
| 創業枠 | 200万円 | 2/3 |

### 対象経費

- 機械装置等費
- 広報費（チラシ作成、Web広告等）
- ウェブサイト関連費
- 展示会等出展費
- 旅費
- 開発費
- 資料購入費
- 雑役務費
- 借料
- 設備処分費
- 委託・外注費

### 税理士事務所がクライアントに案内すべきポイント

- **最も手軽な補助金**: 小規模事業者であれば幅広く利用可能
- **商工会議所・商工会の支援が必須**: 経営計画書の作成に商工会議所等の支援を受ける必要あり
- **ウェブサイト関連費の上限**: ウェブサイト関連費は補助金総額の1/4が上限
- **販路開拓が主目的**: 新たな顧客層の開拓、新商品の開発等が対象
- **確定申告書の添付**: 直近の確定申告書の提出が求められる

## 4. 事業再構築補助金

### 制度概要

| 項目 | 内容 |
|------|------|
| 目的 | ポストコロナ・ウィズコロナ時代の経済社会の変化に対応するための事業再構築を支援 |
| 実施機関 | 中小企業庁（事務局: 中小企業基盤整備機構） |
| 公式サイト | https://jigyou-saikouchiku.go.jp/ |
| 備考 | 公募の継続は年度予算による。最新の公募状況を確認すること |

### 申請類型（直近公募ベース目安）

| 類型 | 補助上限額 | 補助率 |
|------|-----------|--------|
| 成長枠 | 2,000万円〜7,000万円（従業員数による） | 中小: 1/2、中堅: 1/3 |
| グリーン成長枠 | 中小: 4,000万円〜1億円、中堅: 1億円〜1.5億円 | 中小: 1/2、中堅: 1/3 |
| 卒業促進枠 | 成長枠等の上限の上乗せ（中小→中堅の成長を促進） | 1/2 |
| 産業構造転換枠 | 2,000万円〜7,000万円 | 中小: 2/3、中堅: 1/2 |
| サプライチェーン強靱化枠 | 5億円以内 | 中小: 1/2、中堅: 1/3 |

### 税理士事務所がクライアントに案内すべきポイント

- **新分野展開・事業転換が条件**: 既存事業の延長ではなく、新たな事業への挑戦が必要
- **認定経営革新等支援機関の確認**: 3,000万円超の補助額の場合、金融機関の確認も必要
- **事前着手の承認**: 公募開始前に着手する場合は事前着手届出が必要
- **補助金の圧縮記帳**: 取得した固定資産に対する圧縮記帳の適用検討
- **付加価値額の増加要件**: 補助事業終了後3〜5年で付加価値額の年率平均3.0〜5.0%以上増加

## 5. 各種税制優遇措置

### 5-1. 中小企業投資促進税制（租税特別措置法第42条の6）

| 項目 | 内容 |
|------|------|
| 対象者 | 青色申告を行う中小企業者等（資本金1億円以下） |
| 対象設備 | 機械装置（160万円以上）、工具（測定工具及び検査工具: 120万円以上、30万円以上のもの一定数量以上）、ソフトウェア（70万円以上）等 |
| 特別措置 | **特別償却30%** 又は **税額控除7%**（個人事業主・資本金3,000万円以下の法人） |
| 適用期限 | 2027年3月31日まで（延長の可能性あり） |

### 5-2. 中小企業経営強化税制（租税特別措置法第42条の12の4）

| 項目 | 内容 |
|------|------|
| 対象者 | 経営力向上計画の認定を受けた中小企業者等 |
| 対象設備 | 生産性向上設備（A類型）、収益力強化設備（B類型）、デジタル化設備（C類型）、経営資源集約化設備（D類型） |
| 特別措置 | **即時償却** 又は **税額控除7%（資本金3,000万円超）/10%（資本金3,000万円以下）** |
| 適用期限 | 2027年3月31日まで（延長の可能性あり） |
| 必要書類 | 経営力向上計画の認定書、工業会等の証明書（A類型）、経済産業局の確認書（B・C類型） |

### 5-3. 所得拡大促進税制（中小企業向け賃上げ促進税制）

| 項目 | 内容 |
|------|------|
| 対象者 | 青色申告を行う中小企業者等 |
| 要件 | 雇用者給与等支給額が前年度比で**1.5%以上**増加 |
| 税額控除 | 増加額の**15%**を法人税額から控除（法人税額の20%が上限） |
| 上乗せ措置 | 2.5%以上増加: **30%**控除。教育訓練費10%以上増加: さらに**10%**上乗せ |
| 適用期限 | 2027年3月31日まで（延長の可能性あり） |

### 税理士事務所がクライアントに案内すべきポイント

- **設備投資前に検討**: 設備投資の計画段階で税制優遇の適用可能性を確認
- **経営力向上計画は事前認定**: 設備取得前に計画の認定を受ける必要がある（事後申請不可）
- **賃上げ税制は比較的適用しやすい**: 1.5%の賃上げ（昇給・賞与増額含む）で適用可能
- **他の補助金との併用**: 補助金を受けた設備でも税制優遇の適用は可能（ただし補助金分を除いた取得価額が基準）
- **適用額明細書の記載**: 法人税申告書に適用額明細書の添付が必要

## 6. 小規模企業共済

### 制度概要

| 項目 | 内容 |
|------|------|
| 目的 | 小規模企業の経営者・役員の退職金制度 |
| 運営 | 独立行政法人 中小企業基盤整備機構 |
| 加入対象 | 常時使用する従業員が20人以下（商業・サービス業は5人以下）の個人事業主・法人役員 |
| 掛金 | 月額1,000円〜70,000円（500円単位で自由に設定） |
| 掛金の税制メリット | **全額が小規模企業共済等掛金控除として所得控除**（最大年84万円） |
| 共済金の受取り | 廃業・退職時に一括または分割で受取り。一括受取りは退職所得扱い、分割受取りは公的年金等の雑所得扱い |
| 貸付制度 | 掛金の範囲内で事業資金等の貸付けが可能 |

### 税理士事務所がクライアントに案内すべきポイント

- **節税効果が大きい**: 最大年84万円の所得控除。所得税率20%の場合、年間約25万円の節税
- **退職金の準備**: 小規模企業の経営者には退職金制度がないため、この制度が事実上の退職金
- **途中解約のデメリット**: 掛金納付月数が240ヶ月（20年）未満の任意解約は元本割れの可能性あり
- **法人成りした場合**: 個人事業から法人に変更した場合、共済契約は継続可能
- **iDeCoとの比較**: 併用可能。用途に応じて使い分けを提案

## 7. 経営セーフティ共済（中小企業倒産防止共済）

### 制度概要

| 項目 | 内容 |
|------|------|
| 目的 | 取引先の倒産に備えた経営セーフティネット |
| 運営 | 独立行政法人 中小企業基盤整備機構 |
| 加入対象 | 1年以上事業を行っている中小企業者 |
| 掛金 | 月額5,000円〜200,000円（5,000円単位）。掛金総額800万円まで |
| 掛金の税制メリット | **全額が法人の損金又は個人事業の必要経費に算入** |
| 共済金の貸付け | 取引先が倒産した場合、掛金総額の10倍（最高8,000万円）まで無担保・無保証人で貸付け |
| 一時貸付金 | 取引先の倒産がなくても、解約手当金の95%を上限に貸付け可能 |
| 解約手当金 | 12ヶ月以上: 80%以上、40ヶ月以上: 100% |

### 2024年10月改正（重要）

| 改正内容 | 詳細 |
|---------|------|
| 解約後の再加入制限 | 解約後**2年間は再加入不可**（2024年10月1日以降の解約分から適用） |
| みなし解約制度の導入 | 掛金総額が800万円に達した後、**掛止めから2年経過で自動解約**（みなし解約） |

### 税理士事務所がクライアントに案内すべきポイント

- **節税の定番**: 掛金全額が損金算入。最大年240万円（月20万円×12ヶ月）の損金算入が可能
- **利益の平準化**: 利益が出た年に掛金を増額し、赤字の年に解約して収益計上する使い方
- **解約時の課税**: 解約手当金は全額が益金（雑収入）に算入されるため、解約のタイミングが重要
- **2024年10月改正の影響**: 解約→再加入の節税スキームが使えなくなった。長期的な運用計画が必要
- **40ヶ月以上で元本保証**: 40ヶ月以上加入すれば解約手当金は掛金の100%

## 8. 日本政策金融公庫の融資制度

### 主要な融資制度

| 制度名 | 対象 | 融資限度額 | 利率（目安） | 返済期間 |
|--------|------|-----------|-------------|---------|
| 一般貸付 | ほとんどの業種の中小企業 | 4,800万円（特別: 7,200万円） | 基準利率 2.0%前後 | 設備: 10年以内、運転: 7年以内 |
| 新規開業資金 | 新たに事業を始める方 | 7,200万円（うち運転資金4,800万円） | 基準利率（一部要件で利率引下げ） | 設備: 20年以内、運転: 10年以内 |
| マル経融資（小規模事業者経営改善資金） | 商工会議所等の推薦を受けた小規模事業者 | 2,000万円 | 特別利率（1.5%前後） | 設備: 10年以内、運転: 7年以内 |
| 中小企業経営力強化資金 | 認定経営革新等支援機関の支援を受けた中小企業 | 7,200万円 | 基準利率（一部利率引下げ） | 設備: 20年以内、運転: 7年以内 |
| セーフティネット貸付 | 経営環境の変化等により一時的に業況が悪化している中小企業 | 4,800万円 | 基準利率 | 設備: 15年以内、運転: 8年以内 |

### 融資を受ける際に必要な書類（一般的なもの）

| 書類 | 内容 |
|------|------|
| 借入申込書 | 日本政策金融公庫所定の様式 |
| 創業計画書（新規開業の場合） | 事業内容、経歴、資金計画等 |
| 企業概要書 | 事業内容、従業員数、取引先等 |
| 決算書（直近2期分） | 貸借対照表、損益計算書、勘定科目内訳明細書 |
| 確定申告書（個人の場合） | 直近2期分 |
| 事業計画書 | 今後の売上・利益の見通し |
| 見積書 | 設備資金の場合 |
| 不動産の登記簿謄本 | 担保提供がある場合 |

### 税理士事務所がクライアントに案内すべきポイント

- **認定経営革新等支援機関としての役割**: 税理士事務所が認定支援機関であれば、融資申込みの支援が可能
- **事業計画書の策定支援**: 金融機関向けの説得力ある事業計画書の作成をサポート
- **決算書の品質**: 月次で適切に処理された決算書は融資審査で有利
- **マル経融資の活用**: 無担保・無保証人で低利率のマル経融資は小規模事業者に最適
- **借換えの提案**: 既存の高利率融資からの借換えで金利負担を軽減
- **制度融資との比較**: 都道府県・市区町村の制度融資（信用保証協会付き）との比較検討

## 9. その他の支援制度

### 9-1. 経営革新計画

| 項目 | 内容 |
|------|------|
| 概要 | 新事業活動に取り組む計画を都道府県知事が承認。各種支援措置の対象になる |
| メリット | 政策金融機関の低利融資、信用保証の特例、補助金の加点等 |
| 申請先 | 都道府県の産業振興課等 |

### 9-2. 事業継続力強化計画（ジギョケイ）

| 項目 | 内容 |
|------|------|
| 概要 | 防災・減災の事前対策に関する計画を経済産業大臣が認定 |
| メリット | 防災・減災設備の税制優遇（特別償却20%）、補助金の加点、低利融資 |
| 申請先 | 経済産業局 |

### 9-3. 事業承継・引継ぎ支援センター

| 項目 | 内容 |
|------|------|
| 概要 | 事業承継に関する無料相談窓口 |
| 支援内容 | 後継者のマッチング、事業承継計画の策定支援、M&Aの仲介支援 |
| 設置場所 | 各都道府県に設置 |

### 9-4. 事業承継税制（特例措置）

| 項目 | 内容 |
|------|------|
| 概要 | 非上場株式等の贈与税・相続税の納税を猶予・免除 |
| 特例措置の適用期限 | 特例承継計画の提出: 2026年3月31日まで。贈与・相続: 2027年12月31日まで |
| 要件 | 特例承継計画の策定、認定経営革新等支援機関の所見、都道府県知事の認定 |
| 注意 | 雇用確保要件（5年間平均80%以上の維持）あるが、未達成でも理由書提出で継続可能 |

## 補助金と税務処理

### 圧縮記帳

補助金で取得した固定資産については、**圧縮記帳**が適用可能。

| 項目 | 内容 |
|------|------|
| 目的 | 補助金収入に対する課税を繰り延べる |
| 仕組み | 補助金相当額を固定資産の取得価額から減額（圧縮損を計上） |
| 効果 | 補助金を受けた年度の課税所得を圧縮。ただし翌年度以降の減価償却費が減少するため、長期的には課税の繰延べ |

### 仕訳例（機械装置1,000万円を取得、うち500万円が補助金の場合）

```
【補助金受入時】
（借）普通預金     5,000,000 ／（貸）雑収入       5,000,000

【固定資産取得時】
（借）機械装置    10,000,000 ／（貸）普通預金    10,000,000

【圧縮記帳（直接減額方式）】
（借）固定資産圧縮損 5,000,000 ／（貸）機械装置     5,000,000

→ 帳簿上の機械装置の取得価額: 5,000,000円
→ 以降の減価償却は5,000,000円を基に計算
```

### 消費税の取扱い

| 項目 | 内容 |
|------|------|
| 補助金収入 | 不課税（消費税の課税対象外） |
| 補助金で取得した資産の仕入税額控除 | **全額控除可能**（補助金部分を含む取得価額全体に係る消費税を控除できる） |
| 消費税の返還 | 一部の補助金では、仕入税額控除した消費税分を返還する義務がある場合あり（補助金の交付要綱を確認） |
