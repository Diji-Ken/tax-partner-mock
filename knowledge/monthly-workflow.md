# 月次処理の標準フロー

> 税理士・会計事務所向けCRMナレッジ
> 最終更新: 2026-03-23

---

## 概要

会計事務所における月次処理の標準的な手順をフロー化。
CRMのタスク自動生成やチェックリスト機能の基礎データとして使用する。

### 月次処理の全体タイムライン

```
翌月1日〜5日    : ステップ1 - 資料回収
翌月5日〜15日   : ステップ2 - 仕訳入力
翌月10日〜15日  : ステップ3 - 残高確認
翌月15日〜20日  : ステップ4 - 月次試算表の作成
翌月15日〜20日  : ステップ5 - 異常値チェック
翌月20日〜25日  : ステップ6 - 顧問先への報告
```

※上記は目安。顧問先との契約内容により調整。

---

## ステップ1: 資料回収

### やること

| 作業 | 内容 |
|------|------|
| 通帳コピーの回収 | 全口座（普通預金・当座預金・定期預金）の通帳コピー又はネットバンキングの入出金明細 |
| 現金出納帳の回収 | 小口現金の出入りを記録した現金出納帳（あれば） |
| 領収書・レシートの回収 | 月分の経費に係る領収書・レシート（日付順に整理されていることが望ましい） |
| 請求書の回収 | 売上に係る発行請求書の控え、仕入に係る受領請求書 |
| クレジットカード明細 | 法人カード・個人事業用カードの利用明細 |
| 売上・売掛金データ | 売上管理表、入金管理表（顧問先が管理している場合） |
| 給与関連データ | 給与明細、勤怠データ、賞与支給データ |
| その他の資料 | 契約書、リース契約書、借入金返済予定表、固定資産の取得・売却に関する資料 |

### チェックポイント

- [ ] 全口座分の通帳コピー（ネットバンキング明細）が揃っているか
- [ ] 領収書・レシートの日付が当月分に限定されているか（前月の漏れ、翌月分の混入がないか）
- [ ] クレジットカード明細の利用月と引落月の区別ができているか
- [ ] 売上の計上基準（発生主義か現金主義か）を確認
- [ ] インボイス（適格請求書）の有無を確認（登録番号の記載があるか）

### よくあるミス

| ミス | 対策 |
|------|------|
| 通帳コピーの一部ページが欠落 | 前月末残高と当月初残高の一致を確認 |
| 領収書の紛失 | 出金伝票での処理を顧問先に説明、再発行依頼のサポート |
| クレカ明細と領収書の重複計上 | クレカ引落日と利用日の照合ルールを明確化 |
| 個人用と事業用の経費混在（個人事業主） | 事業主貸勘定での処理、按分ルールの確認 |
| インボイス非対応の請求書の見落とし | 全請求書のインボイス番号チェックを習慣化 |

### 所要時間目安

| 顧問先規模 | 目安 |
|-----------|------|
| 個人事業主・小規模法人（月50仕訳以下） | 30分〜1時間 |
| 中規模法人（月50〜200仕訳） | 1〜2時間 |
| 大規模法人（月200仕訳超） | 2〜4時間 |

## ステップ2: 仕訳入力

### やること

| 作業 | 内容 |
|------|------|
| 証憑の確認 | 各証憑の日付・金額・取引内容・消費税区分を確認 |
| 仕訳入力 | 会計ソフトへの仕訳入力（勘定科目、補助科目、税区分、摘要） |
| 補助簿との照合 | 売掛金・買掛金の補助簿と請求書・入金データの照合 |
| 消費税区分の確認 | 課税/非課税/不課税/免税/軽減税率の区分が正しいか確認 |
| インボイスチェック | 適格請求書の要件を満たしているか、登録番号の有効性を確認 |
| 固定資産の登録 | 10万円以上の資産取得があれば固定資産台帳に登録 |
| 給与仕訳 | 給与・賞与・社会保険料・源泉所得税・住民税の仕訳入力 |
| 借入金仕訳 | 借入金の返済（元本・利息の区分）の仕訳入力 |

### チェックポイント

- [ ] 勘定科目が適切か（前月以前の処理と一貫性があるか）
- [ ] 消費税の税率（10%・8%軽減・非課税・不課税）が正しいか
- [ ] インボイスの登録番号が有効か（適格請求書でない場合、経過措置の適用処理を行ったか）
- [ ] 摘要欄に取引先名・取引内容が記載されているか
- [ ] 日付が正しいか（特に月末・月初の取引）
- [ ] 外貨建取引がある場合、為替レートは適切か
- [ ] 前払費用・未払費用の計上漏れがないか

### よくあるミス

| ミス | 影響 | 対策 |
|------|------|------|
| 消費税区分の誤り | 消費税申告額の過不足 | 科目ごとのデフォルト税区分設定、入力後のチェックリスト |
| 勘定科目の誤り | 試算表の歪み、税務調査リスク | 科目体系の統一、摘要辞書の整備 |
| 二重計上 | 費用の過大計上 | 通帳転記とクレカ明細の照合、チェック済みマーク |
| 入力漏れ | 費用の過小計上 | 通帳の全取引との突合せ |
| 税込・税抜の混同 | 消費税計算の誤り | 会計ソフトの入力設定確認 |
| 経費の期間帰属の誤り | 月次PLの歪み | 発生主義の徹底、前払/未払の確認 |

### 所要時間目安

| 顧問先規模 | 目安 |
|-----------|------|
| 個人事業主・小規模法人（月50仕訳以下） | 1〜2時間 |
| 中規模法人（月50〜200仕訳） | 3〜5時間 |
| 大規模法人（月200仕訳超） | 5〜10時間 |

## ステップ3: 残高確認

### やること

| 作業 | 内容 |
|------|------|
| 預金残高の照合 | 会計ソフトの預金残高と通帳（ネットバンキング）の残高を照合 |
| 現金残高の確認 | 会計ソフトの現金残高と実際の現金有高を照合（現金出納帳がある場合） |
| 売掛金残高の照合 | 売掛金の補助簿残高と売上管理台帳・入金管理台帳の突合せ |
| 買掛金残高の照合 | 買掛金の補助簿残高と仕入管理台帳・支払管理台帳の突合せ |
| 借入金残高の確認 | 借入金残高と返済予定表の照合 |
| 仮払金・仮受金の確認 | 仮勘定の内容確認、長期滞留の有無チェック |
| 預り金残高の確認 | 源泉所得税、住民税、社会保険料の預り金残高が正しいか |

### チェックポイント

- [ ] 全口座の預金残高が一致しているか
- [ ] 現金残高がマイナスになっていないか
- [ ] 売掛金の回収遅延がないか（入金予定日を過ぎているものの確認）
- [ ] 買掛金の支払漏れがないか
- [ ] 仮払金・仮受金が長期間（2ヶ月以上）滞留していないか
- [ ] 預り金残高が翌月の納付額と整合しているか

### よくあるミス

| ミス | 影響 | 対策 |
|------|------|------|
| 預金残高の不一致 | 入力漏れ・二重計上の発見遅れ | 毎月必ず残高照合を実施 |
| 売掛金の滞留放置 | 回収不能リスク増大 | 入金予定日管理、滞留一覧の作成 |
| 現金残高のマイナス | 入力漏れの可能性 | 現金出納帳との照合、事業主借の計上確認 |
| 仮払金の精算忘れ | 期末に精算が間に合わない | 仮勘定の定期確認アラート |

### 所要時間目安

| 顧問先規模 | 目安 |
|-----------|------|
| 個人事業主・小規模法人 | 30分〜1時間 |
| 中規模法人 | 1〜2時間 |
| 大規模法人 | 2〜3時間 |

## ステップ4: 月次試算表の作成

### やること

| 作業 | 内容 |
|------|------|
| 試算表の出力 | 会計ソフトから月次試算表（BS/PL）を出力 |
| 残高試算表のチェック | 貸借が一致しているか確認 |
| 部門別損益の確認 | 部門管理をしている場合、部門別PLの確認 |
| 推移表の作成 | 月別推移表を作成し、トレンドを把握 |
| 消費税の概算計算 | 決算時の消費税納付額の概算を把握 |

### チェックポイント

- [ ] 貸借対照表の資産合計と負債・純資産合計が一致しているか
- [ ] 損益計算書の売上高・売上原価・販管費が妥当な水準か
- [ ] 前月比で大きな変動がある科目はないか
- [ ] 各科目残高にマイナス（あり得ない残高）がないか

### 所要時間目安

| 顧問先規模 | 目安 |
|-----------|------|
| 全規模共通 | 30分〜1時間 |

## ステップ5: 異常値チェック

### やること

| 作業 | 内容 |
|------|------|
| 前月比較 | 全勘定科目の前月比を確認し、大きな増減（±20%以上）を検出 |
| 前年同月比較 | 主要科目の前年同月比を確認し、季節変動以外の異常を検出 |
| 科目残高の妥当性 | BS科目の残高が常識的な範囲か確認（現金が多すぎないか等） |
| 利益率の確認 | 売上総利益率・営業利益率が業種平均と比較して妥当か |
| 異常仕訳の検出 | 通常と異なる仕訳パターン（大額取引、逆仕訳等）を確認 |

### 主なチェック項目

| チェック項目 | 判定基準 | 異常の可能性 |
|-------------|---------|-------------|
| 現金残高が多い | 法人で100万円超、個人で50万円超 | 入力漏れ、架空計上 |
| 現金残高がマイナス | - | 入力漏れ、事業主借の計上漏れ |
| 売掛金の増加 | 前月比30%超増加 | 回収遅延、架空売上 |
| 買掛金の減少が売上と不整合 | - | 支払漏れの仕訳入力 |
| 仮払金・仮受金の滞留 | 2ヶ月以上残高あり | 精算忘れ |
| 接待交際費の急増 | 前月比2倍以上 | 科目誤り、私的経費の混入 |
| 売上総利益率の急変 | 前月比±5%以上 | 売上原価の計上漏れ、単価変動 |
| 消耗品費と工具器具備品の境界 | 10万円前後の取引 | 資産計上漏れ |
| 地代家賃の変動 | 前月と異なる金額 | 契約変更の確認、二重計上 |
| 給与手当の変動 | 前月と異なる金額 | 残業代変動、入退社の確認 |

### チェックポイント

- [ ] 前月比±20%以上変動している科目をリストアップしたか
- [ ] 変動理由を全て説明できるか
- [ ] 説明できない変動について顧問先に確認したか
- [ ] BS科目に異常な残高（マイナス、過大）がないか

### よくあるミス

| ミス | 対策 |
|------|------|
| 異常値を見逃す | チェックリストの定型化、自動検出ツールの活用 |
| 変動理由を確認せずに放置 | 疑問点リストの作成、顧問先への定期確認 |
| 季節変動と異常値の混同 | 前年同月比との併用 |

### 所要時間目安

| 顧問先規模 | 目安 |
|-----------|------|
| 個人事業主・小規模法人 | 30分〜1時間 |
| 中規模法人 | 1〜2時間 |
| 大規模法人 | 2〜3時間 |

## ステップ6: 顧問先への報告

### やること

| 作業 | 内容 |
|------|------|
| 月次報告書の作成 | 月次試算表にコメントを付けた報告書を作成 |
| 業績サマリーの作成 | 売上・利益の推移、前年比較のサマリー |
| 資金繰りの確認 | 今後の資金繰り予測（大きな支出・入金予定の確認） |
| 税金関連の案内 | 翌月以降の納税スケジュール、予定納税額の案内 |
| 経営アドバイス | 業績に基づく節税提案、設備投資の検討等 |
| 報告・面談の実施 | 訪問面談、Web面談、または報告書の送付 |

### 月次報告書の構成（例）

```
1. 業績概要（売上高、営業利益、経常利益の前月比・前年同月比）
2. 損益計算書の主要項目解説
3. 貸借対照表の主要項目解説
4. 資金繰り状況
5. 税務カレンダー（翌月の税務イベント）
6. 今月のトピック・提案事項
7. 確認事項・依頼事項
```

### チェックポイント

- [ ] 報告書に誤りや未確認事項がないか
- [ ] 顧問先が理解しやすい表現になっているか（専門用語の説明）
- [ ] 前回の確認事項・依頼事項のフォローアップができているか
- [ ] 税務上のアドバイス事項（節税、届出等）を漏れなく伝えているか
- [ ] 次回の報告日・資料提出期限を案内したか

### よくあるミス

| ミス | 対策 |
|------|------|
| 報告書の数字と試算表の不一致 | 出力後の最終確認 |
| 前回の宿題事項のフォロー漏れ | 宿題リストの管理、CRMでのタスク管理 |
| 税務イベントの案内漏れ | 税務カレンダーとの自動連携 |
| 報告が形式的で経営判断に役立たない | 顧問先の業種・規模に応じたアドバイスの準備 |

### 所要時間目安

| 顧問先規模 | 報告書作成 | 報告・面談 |
|-----------|-----------|-----------|
| 個人事業主・小規模法人 | 30分〜1時間 | 30分〜1時間 |
| 中規模法人 | 1〜2時間 | 1〜2時間 |
| 大規模法人 | 2〜3時間 | 1〜2時間 |

## 月次処理の品質管理

### チェックリスト（月次締めの最終確認）

| No. | 確認項目 | チェック |
|-----|---------|---------|
| 1 | 全口座の預金残高が通帳と一致するか | □ |
| 2 | 現金残高が妥当か（マイナスでないか） | □ |
| 3 | 売掛金・買掛金の残高が取引実態と整合するか | □ |
| 4 | 仮勘定（仮払金・仮受金）の内容を確認したか | □ |
| 5 | 消費税区分に誤りがないか | □ |
| 6 | インボイスの登録番号チェックを行ったか | □ |
| 7 | 前月比較で異常値がないか確認したか | □ |
| 8 | 借入金残高が返済予定表と一致するか | □ |
| 9 | 源泉所得税・住民税の預り金残高が正しいか | □ |
| 10 | 固定資産の取得・売却があれば台帳を更新したか | □ |

### CRMでのタスク管理推奨項目

1. **顧問先ごとの月次処理進捗管理**: 資料回収→仕訳入力→残高確認→試算表作成→チェック→報告の各ステップのステータス管理
2. **期限管理**: 各ステップの完了予定日と実績日の管理
3. **担当者管理**: 入力担当者とチェック担当者の分離
4. **宿題リスト**: 顧問先への確認事項・依頼事項のリスト管理
5. **月次カレンダー**: 全顧問先の月次処理スケジュールを一覧表示
