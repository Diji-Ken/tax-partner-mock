# 減価償却 完全ガイド

## 1. 減価償却の基本概念

### 1-1. 減価償却とは
- 建物・機械・車両等の固定資産は、取得した年度に全額を費用とするのではなく、その資産の使用可能期間（耐用年数）にわたって費用配分する
- 土地、借地権、電話加入権、書画骨董（1点100万円以上）等の非減価償却資産は減価償却の対象外
- 事業に使用していない資産（遊休資産）も原則として減価償却可能

### 1-2. 取得価額
取得価額 = 購入代価 + 付随費用（据付費、運送費、関税等）

- 購入代価: 本体価格（値引き・割戻しがあれば控除後）
- 付随費用: 引取運賃、荷役費、運送保険料、購入手数料、据付費、試運転費等
- 取得価額に含めないことができるもの: 不動産取得税、自動車取得税（環境性能割）、登録免許税、登録費用

## 2. 定額法の計算方法と計算例

### 2-1. 計算式
```
年間償却費 = 取得価額 × 定額法の償却率
```

### 2-2. 定額法の償却率（主要な耐用年数）
| 耐用年数 | 償却率 |

| 3年 | 0.334 |
| 4年 | 0.250 |
| 5年 | 0.200 |
| 6年 | 0.167 |
| 7年 | 0.143 |
| 8年 | 0.125 |
| 10年 | 0.100 |
| 15年 | 0.067 |
| 17年 | 0.059 |
| 20年 | 0.050 |
| 22年 | 0.046 |
| 30年 | 0.034 |
| 34年 | 0.030 |
| 38年 | 0.027 |
| 39年 | 0.026 |
| 47年 | 0.022 |
| 50年 | 0.020 |

### 2-3. 計算例
**条件**: 取得価額500万円、耐用年数10年、定額法
- 年間償却費 = 5,000,000 × 0.100 = 500,000円
- 最終年度は帳簿価額が1円（備忘価額）になるまで償却
- 期中取得の場合: 月割計算（月数/12）

**期中取得の例**: 取得価額500万円、耐用年数10年、10月取得（3月決算法人）
- 初年度: 5,000,000 × 0.100 × 6/12 = 250,000円（10月〜3月の6ヶ月分）

## 3. 定率法（200%定率法）の計算方法と計算例

### 3-1. 計算式
```
年間償却費 = 期首帳簿価額 × 定率法の償却率
```

ただし、上記で計算した償却費が「償却保証額」を下回る年度以降は、改定取得価額（その年度の期首帳簿価額）× 改定償却率で計算する。

### 3-2. 200%定率法の償却率・改定償却率・保証率（主要な耐用年数）
| 耐用年数 | 償却率 | 改定償却率 | 保証率 |
|---------|--------|-----------|--------|
| 2年 | 1.000 | - | - |
| 3年 | 0.667 | 1.000 | 0.11089 |
| 4年 | 0.500 | 1.000 | 0.12499 |
| 5年 | 0.400 | 0.500 | 0.10800 |
| 6年 | 0.333 | 0.334 | 0.09911 |
| 7年 | 0.286 | 0.334 | 0.08680 |
| 8年 | 0.250 | 0.334 | 0.07909 |
| 10年 | 0.200 | 0.250 | 0.06552 |
| 15年 | 0.133 | 0.143 | 0.04565 |
| 17年 | 0.118 | 0.125 | 0.04038 |
| 20年 | 0.100 | 0.112 | 0.03486 |

### 3-3. 計算例
**条件**: 取得価額500万円、耐用年数5年、定率法（200%定率法）

- 償却保証額 = 5,000,000 × 0.10800 = 540,000円

| 年度 | 期首帳簿価額 | 償却費 | 計算方法 | 期末帳簿価額 |
|------|------------|--------|---------|------------|
| 1年目 | 5,000,000 | 2,000,000 | 5,000,000 × 0.400 | 3,000,000 |
| 2年目 | 3,000,000 | 1,200,000 | 3,000,000 × 0.400 | 1,800,000 |
| 3年目 | 1,800,000 | 720,000 | 1,800,000 × 0.400 | 1,080,000 |
| 4年目 | 1,080,000 | 540,000 | 1,080,000 × 0.500（改定償却率） | 540,000 |
| 5年目 | 540,000 | 539,999 | 540,000 × 0.500 → 備忘価額1円を残す | 1 |

※4年目: 1,080,000 × 0.400 = 432,000 < 償却保証額540,000 → 改定償却率に切替

### 3-4. 適用対象
- **定額法のみ**: 建物（1998年4月1日以後取得）、建物附属設備・構築物（2016年4月1日以後取得）、鉱業用の建物・建物附属設備・構築物、無形固定資産、生物
- **定額法・定率法の選択可**: 機械装置、車両運搬具、工具器具備品（届出がない場合は定率法）

## 5. 少額資産の3区分

### 5-1. 10万円未満: 全額費用（消耗品費）
- **対象**: 取得価額が10万円未満の減価償却資産、または使用可能期間が1年未満のもの
- **処理方法**: 取得した事業年度に全額を損金（必要経費）に算入
- **勘定科目**: 消耗品費
- **注意**: 10万円の判定は、消費税の経理方式による
  - 税込経理: 税込金額で判定
  - 税抜経理: 税抜金額で判定

### 5-2. 10万円以上20万円未満: 一括償却資産（3年均等償却）
- **対象**: 取得価額が10万円以上20万円未満の減価償却資産
- **処理方法**: 事業年度ごとに、その年度に取得した一括償却資産の取得価額の合計額の1/3を損金算入
- **特徴**:
  - 個々の資産ごとではなく、年度ごとの合計額で計算
  - 月割計算は不要（年度の途中で取得しても1/3を償却）
  - 除却・売却しても残りの1/3ずつを償却し続ける
  - 償却資産税（固定資産税）の申告対象外
- **法人・個人ともに適用可能**

### 5-3. 30万円未満: 中小企業の特例（即時償却）
- **対象**: 取得価額が30万円未満の減価償却資産
- **適用要件**:
  - 青色申告法人（中小企業者等に該当）または青色申告の個人事業者
  - 中小企業者等: 資本金1億円以下の法人（大法人の子会社等を除く）
  - 常時使用する従業員数が500人以下
- **限度額**: 事業年度（個人は暦年）の合計で**300万円**まで
- **処理方法**: 取得価額の全額をその事業年度の損金に算入
- **注意**: 償却資産税の申告対象（一括償却資産と異なる）
- **適用期限**: 2026年3月31日までに取得し事業の用に供したもの（延長の可能性あり）

## 6. 特別償却・税額控除

### 6-1. 中小企業投資促進税制（措法42条の6）
| 項目 | 内容 |
|------|------|
| **対象法人** | 青色申告の中小企業者等（資本金3,000万円以下は税額控除も選択可） |
| **対象資産** | 機械装置（1台160万円以上）、工具（1台120万円以上で一定のもの）、ソフトウェア（1つ70万円以上）等 |
| **特別償却** | 取得価額の**30%**を普通償却に上乗せして償却 |
| **税額控除**（資本金3,000万円以下のみ） | 取得価額の**7%**を法人税額から控除（控除限度額: 法人税額の20%） |
| **適用期限** | 2027年3月31日まで（延長の可能性あり） |

### 6-2. 中小企業経営強化税制（措法42条の12の4）
| 項目 | 内容 |
|------|------|
| **対象法人** | 青色申告の中小企業者等で「経営力向上計画」の認定を受けた法人 |
| **対象資産** | 生産性向上設備（A類型）、収益力強化設備（B類型）、デジタル化設備（C類型）、経営資源集約化設備（D類型） |
| **即時償却** | 取得価額の**100%**を償却（全額費用化） |
| **税額控除** | 取得価額の**7%**（資本金3,000万円以下は**10%**）を法人税額から控除（控除限度額: 法人税額の20%） |
| **適用期限** | 2027年3月31日まで |
| **手続き** | 設備取得前に「経営力向上計画」を主務大臣に申請→認定 |

### 6-3. 特別償却と税額控除の比較
| 項目 | 特別償却 | 税額控除 |
|------|---------|---------|
| 効果 | 初年度の経費を増加（税の繰延べ） | 法人税額そのものを減少（永久的な節税） |
| 有利なケース | 初年度の利益が大きく、翌期以降の利益が不透明な場合 | 安定した利益がある場合 |
| 長期的な効果 | 耐用年数全体では同じ（タイミングの違い） | 減税効果が永久的 |

## 7. 中古資産の耐用年数の計算方法（簡便法）

### 7-1. 法定耐用年数の全部を経過した資産
```
耐用年数 = 法定耐用年数 × 20%（端数切り捨て、2年未満の場合は2年）
```

**計算例**: 法定耐用年数22年の木造建物（築25年）
- 22年 × 20% = 4.4年 → **4年**

### 7-2. 法定耐用年数の一部を経過した資産
```
耐用年数 = (法定耐用年数 - 経過年数) + 経過年数 × 20%（端数切り捨て、2年未満の場合は2年）
```

**計算例**: 法定耐用年数6年の普通自動車（3年経過）
- (6年 - 3年) + 3年 × 20% = 3 + 0.6 = 3.6年 → **3年**

### 7-3. 注意点
- 簡便法は取得後に「資本的支出」を行った場合、その金額が取得価額の50%を超えると使えない
- 中古資産でも取得後の資本的支出が再取得価額の50%以上の場合は法定耐用年数を適用
- 海外不動産の中古資産の簡便法は、2021年以後の国内の不動産所得の計算上、損益通算の対象外（2020年度税制改正）

## 8. 資本的支出と修繕費の判定基準

### 8-1. 基本的な考え方
| 区分 | 内容 | 処理 |
|------|------|------|
| **資本的支出** | 資産の価値を高めるもの、耐用年数を延長するもの | 資産計上し、減価償却 |
| **修繕費** | 資産の原状回復、維持管理のための支出 | 全額費用処理 |

### 8-2. 形式基準による判定フロー

```
支出額が20万円未満か？
  → Yes: 修繕費
  → No: 次へ

おおむね3年以内の周期で行われる修繕か？
  → Yes: 修繕費
  → No: 次へ

明らかに価値増加・耐用年数延長に該当するか？
  → Yes: 資本的支出
  → No: 次へ

支出額が60万円未満か？
  → Yes: 修繕費
  → No: 次へ

支出額が前期末の取得価額の10%以下か？
  → Yes: 修繕費
  → No: 次へ

継続適用を条件に、支出額の30%と前期末取得価額の10%のいずれか少ない金額を
修繕費とし、残りを資本的支出とする（7:3基準）
```

### 8-3. 具体例
| 支出内容 | 判定 | 理由 |
|---------|------|------|
| 屋根の雨漏り修理 | 修繕費 | 原状回復 |
| 屋根の葺き替え（同等の材料） | 修繕費 | 原状回復 |
| 屋根の葺き替え（グレードアップ） | 資本的支出 | 価値増加 |
| 外壁の塗装（同等の塗料） | 修繕費 | 維持管理 |
| 外壁の塗装（断熱塗料に変更） | 資本的支出 | 価値増加 |
| 事務所のトイレ修理 | 修繕費 | 原状回復 |
| 和式トイレから洋式トイレへの変更 | 資本的支出 | 価値増加 |
| パソコンのメモリ増設 | 資本的支出 | 性能向上 |
| パソコンのHDD交換（同容量） | 修繕費 | 原状回復 |
| 車両のタイヤ交換 | 修繕費 | 維持管理 |
| 車両へのカーナビ取付 | 資本的支出 | 価値増加 |

### 8-4. 資本的支出の償却方法
- **2007年4月1日以後の資本的支出**: 新たな資産を取得したものとして、その支出額を取得価額とする独立した資産として償却
- **耐用年数**: 本体資産と同じ耐用年数
- **償却方法**: 本体資産の取得時期に関係なく、定額法または定率法（200%定率法）
