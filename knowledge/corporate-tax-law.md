# 法人税法 完全ガイド

> 最終更新: 2026-03-24
> 対象: 税理士CRM実務参照用

## 1. 法人税の税率体系

### 普通法人

| 区分 | 税率 |
|------|------|
| 普通法人（資本金1億円超等） | 23.2% |
| 中小法人（資本金1億円以下）の年800万円以下の部分 | 15% |
| 中小法人（資本金1億円以下）の年800万円超の部分 | 23.2% |

**中小法人の定義**: 資本金の額または出資金の額が1億円以下の法人（ただし、資本金5億円以上の大法人の100%子法人等は除く）

### その他の法人

| 法人の種類 | 税率 |
|-----------|------|
| 協同組合等（年800万円以下の部分） | 15% |
| 協同組合等（年800万円超の部分） | 19% |
| 公益法人等（収益事業から生じた所得） | 普通法人と同じ |
| 人格のない社団等 | 普通法人と同じ |
| 特定の医療法人（年800万円以下の部分） | 15% |
| 特定の医療法人（年800万円超の部分） | 19% |

### 地方法人税

法人税額の10.3%を国に納付する（地方法人税法）。

## 2. 事業年度と申告期限

### 事業年度

- 法人が定款で定めた会計期間（通常1年以内）
- 定款に定めがない場合: 設立日から同日の属する月の末日まで

### 確定申告期限

- **原則**: 事業年度終了の日の翌日から**2ヶ月以内**
- 例: 3月決算法人 → 5月31日が申告期限

### 申告期限の延長特例

| 種類 | 延長期間 | 届出要件 |
|------|---------|---------|
| 会計監査人設置法人等の延長 | **1ヶ月**（事業年度終了後3ヶ月以内） | 「申告期限の延長の特例の申請書」を提出 |
| 災害等による延長 | 国税庁長官が指定する日まで | 申請による個別延長も可 |
| 連結納税（グループ通算）の延長 | **2ヶ月**（事業年度終了後4ヶ月以内） | 届出書提出 |

**注意**: 申告期限が延長されても、**納付期限は延長されない**。延長期間分の利子税（年7.3%または延滞税特例基準割合+1%のいずれか低い方）が発生する。

## 3. 損金算入の基準

### 3-1. 交際費

#### 基本ルール

| 資本金 | 損金算入限度額 |
|--------|--------------|
| 1億円以下 | **年800万円まで全額損金算入**（定額控除限度額）、**または**接待飲食費の50%相当額のいずれかを選択 |
| 1億円超〜100億円以下 | 接待飲食費の50%相当額のみ損金算入 |
| 100億円超 | 全額損金不算入 |

#### 5,000円基準（社外飲食費の少額基準）

1人あたり**5,000円以下**の飲食費は、以下の要件を満たせば**交際費から除外**（会議費等として全額損金算入可能）:

**記録保存要件**:
1. 飲食の年月日
2. 参加した得意先・仕入先等の事業関係者の氏名または名称とその関係
3. 参加者の数
4. 飲食費の額とその飲食店の名称・所在地
5. その他参考事項

**注意**: 社内飲食費（専ら自社の役員・従業員等に対する接待）は5,000円基準の適用対象外。

#### 2024年度税制改正

2024年4月1日以降開始事業年度から、1人あたりの基準額が**5,000円から10,000円**に引き上げ。

### 3-2. 寄附金

#### 一般寄附金の損金算入限度額

**計算式**:
```
損金算入限度額 = (期末資本金等の額 × 当期の月数/12 × 2.5/1,000 + 当期所得金額 × 2.5/100) × 1/4
```

#### 寄附金の種類別取扱い

| 種類 | 損金算入 |
|------|---------|
| 国・地方公共団体への寄附金 | **全額損金算入** |
| 指定寄附金（財務大臣指定） | **全額損金算入** |
| 特定公益増進法人等への寄附金 | **特別損金算入限度額**まで損金算入 |
| 認定NPO法人への寄附金 | **特別損金算入限度額**まで損金算入 |
| その他の一般寄附金 | **一般損金算入限度額**まで |

**特別損金算入限度額**の計算式:
```
特別損金算入限度額 = (期末資本金等の額 × 当期の月数/12 × 3.75/1,000 + 当期所得金額 × 6.25/100) × 1/2
```

#### 損金不算入となる寄附金

- 限度額を超える部分
- 国外関連者への寄附金（移転価格税制の対象）

### 3-3. 貸倒損失

#### 3類型

**1. 法律上の貸倒れ（法基通9-6-1）**

以下の事由により金銭債権が消滅した場合、その消滅した事業年度に全額損金算入:

| 事由 | 内容 |
|------|------|
| 更生計画認可の決定 | 会社更生法の更生計画で切り捨てられた金額 |
| 再生計画認可の決定 | 民事再生法の再生計画で切り捨てられた金額 |
| 特別清算の協定認可 | 協定で切り捨てられた金額 |
| 債権者集会の協議 | 合理的な基準により切り捨てられた金額 |
| 書面による債務免除 | 債務超過の状態が相当期間継続し、回収不能と認められる場合の書面による通知額 |

**2. 事実上の貸倒れ（法基通9-6-2）**

- 債務者の資産状況・支払能力等からみて**全額回収不能**であることが明らかな場合
- 担保物がある場合は、担保物を処分した後でなければ認められない
- 保証人がいる場合は、保証人からの回収も見込めないこと
- **損金算入時期**: 回収不能が明らかになった事業年度

**3. 形式上の貸倒れ（法基通9-6-3）**

以下の場合に備忘価額（1円）を残して損金算入:

| 要件 | 内容 |
|------|------|
| 取引停止後1年以上経過 | 継続的な取引を行っていた債務者について、最後の弁済期・最後の取引から**1年以上経過**した場合（担保物のある場合を除く） |
| 同一地域の売掛債権の総額が取立費用に満たない | 旅費その他の取立費用に満たない場合で、**督促しても弁済がない**場合 |

### 3-4. 貸倒引当金

#### 一括評価（法定繰入率方式）

**適用対象**: 中小法人等（資本金1億円以下等）

| 業種 | 法定繰入率 |
|------|-----------|
| 卸売業・小売業（飲食店業を含む） | 10/1,000 |
| 製造業 | 8/1,000 |
| 金融業・保険業 | 3/1,000 |
| 割賦販売小売業等 | 13/1,000 |
| その他の事業 | 6/1,000 |

**計算式**:
```
繰入限度額 = (期末一括評価金銭債権の帳簿価額 − 実質的に債権とみられないものの額) × 法定繰入率
```

**注意**: 大法人（資本金1億円超）は2012年4月1日以降開始事業年度から一括評価貸倒引当金の繰入が廃止（銀行・保険会社等を除く）。

#### 個別評価

以下の事由に該当する場合、個別に見積もった回収不能額を繰入可能:

| 事由 | 繰入可能額 |
|------|-----------|
| 更生手続開始の申立て | 取立て等見込額を控除した残額の**50%** |
| 再生手続開始の申立て | 取立て等見込額を控除した残額の**50%** |
| 破産手続開始の申立て | 取立て等見込額を控除した残額の**50%** |
| 特別清算開始の申立て | 取立て等見込額を控除した残額の**50%** |
| 債務超過が相当期間継続（おおむね1年以上） | 取立て等見込額を控除した残額 |
| 外国政府等に対する金銭債権で長期にわたり履行遅滞 | 取立て等見込額を控除した残額の**50%** |

## 4. 繰越欠損金

### 繰越期間

- **10年間**（2018年4月1日以後開始事業年度に生じた欠損金）
- 2018年3月31日以前開始事業年度に生じた欠損金: 9年間

### 控除限度額

| 法人の区分 | 控除限度額 |
|-----------|-----------|
| 中小法人等（資本金1億円以下等） | 所得金額の**100%**（全額控除可能） |
| 大法人（資本金1億円超等） | 所得金額の**50%** |
| 更生法人等（更生計画認可から7年以内） | 所得金額の**100%** |
| 新設法人（設立後7年以内） | 所得金額の**100%** |

### 適用要件

1. 欠損金の生じた事業年度に**青色申告書**を提出していること
2. その後の事業年度について**連続して確定申告書を提出**していること（白色でも可）
3. 帳簿書類を保存していること

## 5. 繰戻還付

### 適用対象

- **中小法人等**のみ（資本金1億円以下等）
- 大法人は適用不可（不適用措置）

### 制度内容

| 項目 | 内容 |
|------|------|
| 還付対象 | **前1年間**に納付した法人税 |
| 計算式 | 還付金額 = 前期の法人税額 × (当期の欠損金額 / 前期の所得金額) |
| 要件 | 前期・当期ともに**青色申告書**を提出していること |
| 請求期限 | 確定申告書の提出期限まで |

### 具体例

前期所得: 1,000万円（法人税150万円）、当期欠損金: 600万円の場合:
```
還付金額 = 150万円 × (600万円 / 1,000万円) = 90万円
```

## 6. 役員報酬の税務

### 6-1. 定期同額給与

**要件**:
- その支給時期が1月以下の一定期間ごとに支給されること
- 各支給時期における支給額が同額であること

**改定が認められる場合**:

| 改定事由 | 改定時期 |
|---------|---------|
| 通常改定 | 事業年度開始から**3ヶ月以内**（株主総会での決議） |
| 臨時改定事由 | 役員の職制上の地位の変更、職務内容の重大な変更等 |
| 業績悪化改定事由 | 経営状況の著しい悪化等（客観的な事情が必要） |

**注意**: 3ヶ月以内の改定で増額・減額した場合、改定前と改定後の各期間内でそれぞれ同額であれば定期同額給与として損金算入可能。

### 6-2. 事前確定届出給与

**届出期限**:

| 場合 | 届出期限 |
|------|---------|
| 株主総会で決議した場合 | 以下のいずれか**早い日**: (1) 株主総会等の決議日から**1ヶ月を経過する日** (2) 事業年度開始から**4ヶ月を経過する日** |
| 新設法人の場合 | 設立日から**2ヶ月を経過する日** |
| 臨時改定事由が生じた場合 | その事由が生じた日から**1ヶ月を経過する日** |

**届出書の記載事項**:
1. 届出に係る定めをした株主総会等の決議日
2. 届出に係る役員の氏名
3. 届出に係る役員の役職名
4. 届出に係る給与の支給時期
5. 届出に係る給与の支給額
6. 届出に係る定めの内容の概要

**重要**: 届出額と実際の支給額が**1円でも異なる**場合、その全額が損金不算入となる。届出額より少なく支給した場合も同様。複数回の届出がある場合、1回でも届出と異なると全回分が損金不算入。

### 6-3. 業績連動給与

**要件**（全て満たす必要あり）:

1. **同族会社以外**の法人であること（同族会社は不可、ただし同族会社のうち非同族法人との完全支配関係がある法人を除く）
2. **利益に関する指標**（営業利益、経常利益、当期純利益、株価等の客観的な数値基準）を基礎として算定されること
3. その算定方法が**報酬委員会**等の決定、または**株主総会**等の決議により決定されていること
4. 有価証券報告書等に記載されていること
5. 損金経理していること

### 6-4. 過大役員報酬の判定

以下のいずれかに該当する場合、過大部分は損金不算入:

| 判定基準 | 内容 |
|---------|------|
| 形式基準 | 定款の規定または株主総会の決議により定められた**限度額を超える**部分 |
| 実質基準 | 役員の**職務内容、法人の収益状況、使用人に対する給与の支給状況**、同種同規模法人の役員報酬等に照らし、**不相当に高額**な部分 |

## 7. 受取配当の益金不算入

### 持株割合別の除外割合

| 区分 | 持株割合 | 益金不算入割合 |
|------|---------|--------------|
| 完全子法人株式等 | 100% | **100%**（全額益金不算入） |
| 関連法人株式等 | 1/3超〜100%未満 | **100%**（負債利子控除あり） |
| その他の株式等 | 5%超〜1/3以下 | **50%** |
| 非支配目的株式等 | 5%以下 | **20%** |

### 関連法人株式等の負債利子控除

関連法人株式等に係る受取配当から控除する負債利子の額:
```
控除額 = 関連法人株式等に係る配当等の額 × 4%（簡便法）
```
または、実際の負債利子を基に計算する原則法も選択可能。

### 適用除外

- **短期所有株式等**: 配当等の計算期間の末日以前1月以内に取得し、同末日後2月以内に売却した株式に係る配当は益金不算入の対象外
- 外国子会社からの配当: 外国子会社配当益金不算入制度（95%益金不算入、別制度）

## 8. 別表一覧と各別表の概要

### 主要別表

| 別表番号 | 名称 | 内容 |
|---------|------|------|
| **別表一（一）** | 各事業年度の所得に係る申告書 | 法人税額の計算結果をまとめた最終表。課税所得、法人税額、控除税額、差引納付税額等を記載 |
| **別表二** | 同族会社等の判定に関する明細書 | 株主等の持株割合を記載し、同族会社に該当するか判定 |
| **別表三（一）** | 特定同族会社の留保金課税に関する明細書 | 特定同族会社に対する留保金課税の計算 |
| **別表四** | 所得の金額の計算に関する明細書 | 会計上の利益から税務上の所得金額を算出する加算・減算の調整表。最重要別表 |
| **別表五（一）** | 利益積立金額及び資本金等の額の計算に関する明細書 | 利益積立金額（税務上の純資産）と資本金等の額の期首・増減・期末を管理 |
| **別表五（二）** | 租税公課の納付状況等に関する明細書 | 法人税、住民税、事業税等の期首残高、発生額、充当額、納付額等を記載 |
| **別表六（一）** | 所得税額の控除に関する明細書 | 利子・配当等から源泉徴収された所得税額のうち、法人税から控除する額を計算 |
| **別表七（一）** | 欠損金又は災害損失金の損金算入等に関する明細書 | 繰越欠損金の当期控除額を計算。各事業年度の欠損金の発生額と控除額を管理 |
| **別表八（一）** | 受取配当等の益金不算入に関する明細書 | 受取配当金の益金不算入額を計算。持株割合に応じた区分と控除額を記載 |
| **別表十（七）** | 社会保険診療報酬の所得の計算に関する明細書 | 医療法人等の社会保険診療報酬に係る概算経費の特例計算 |
| **別表十一（一）** | 個別評価金銭債権に係る貸倒引当金の損金算入に関する明細書 | 個別評価による貸倒引当金の繰入限度額を計算 |
| **別表十一（一の二）** | 一括評価金銭債権に係る貸倒引当金の損金算入に関する明細書 | 一括評価による貸倒引当金の繰入限度額を計算 |
| **別表十四（二）** | 寄附金の損金算入に関する明細書 | 寄附金の種類別の損金算入限度額と損金不算入額を計算 |
| **別表十五** | 交際費等の損金算入に関する明細書 | 交際費等の支出額、損金算入限度額、損金不算入額を計算 |
| **別表十六（一）** | 旧定額法又は定額法による減価償却資産の償却額の計算に関する明細書 | 定額法による個別資産の償却限度額を計算 |
| **別表十六（二）** | 旧定率法又は定率法による減価償却資産の償却額の計算に関する明細書 | 定率法（200%定率法）による個別資産の償却限度額を計算 |
| **別表十六（四）** | 旧国外リース期間定額法もしくは旧リース期間定額法又はリース期間定額法による償却額の計算に関する明細書 | リース資産の償却額を計算 |
| **別表十六（六）** | 繰延資産の償却額の計算に関する明細書 | 繰延資産の償却限度額を計算 |
| **別表十六（七）** | 少額減価償却資産の取得価額の損金算入の特例に関する明細書 | 中小企業者等の30万円未満の少額減価償却資産の即時償却を記載 |
| **別表十六（八）** | 一括償却資産の損金算入に関する明細書 | 10万円以上20万円未満の一括償却資産の3年均等償却を計算 |

## 9. 同族会社の判定と特別規定

### 同族会社の判定

**判定基準**: 上位**3人以下**の株主グループが発行済株式総数の**50%超**を保有する会社

**株主グループ**: 株主とその同族関係者（配偶者、6親等以内の血族、3親等以内の姻族、その株主が支配する法人等）

### 特定同族会社の留保金課税

**対象**: 被支配会社（1人の株主グループで50%超を保有）で、かつ**資本金1億円超**の法人

**課税方式**: 留保所得金額のうち留保控除額を超える部分に特別税率を適用:

| 留保金額 | 特別税率 |
|---------|---------|
| 年3,000万円以下の部分 | 10% |
| 年3,000万円超〜1億円以下の部分 | 15% |
| 年1億円超の部分 | 20% |

**留保控除額**: 以下のうち最も大きい金額
1. 所得等の金額の40%相当額
2. 年2,000万円
3. 利益積立金額が資本金の25%に達するまでの金額

**適用除外**: 資本金1億円以下の中小法人は留保金課税の対象外。

### 同族会社の行為計算否認

税務署長は、同族会社の行為または計算で、**これを容認した場合には法人税の負担を不当に減少させる結果**となると認められるものがあるときは、その行為または計算にかかわらず、税務署長の認めるところにより法人税の課税標準・税額を計算できる（法人税法132条）。

## 10. 少額減価償却資産の特例

### 一般ルール（全法人共通）

| 取得価額 | 処理方法 |
|---------|---------|
| 10万円未満 | 取得時に全額損金算入（少額減価償却資産） |
| 10万円以上20万円未満 | **3年均等償却**（一括償却資産、月割不要） |
| 20万円以上 | 通常の減価償却 |

### 中小企業者等の特例（租税特別措置法67条の5）

| 項目 | 内容 |
|------|------|
| 対象法人 | 中小企業者等（資本金1億円以下、従業員1,000人以下等） |
| 対象資産 | 取得価額が**30万円未満**の減価償却資産 |
| 損金算入限度 | 事業年度ごとに**合計300万円まで** |
| 適用期限 | 2026年3月31日までに取得・事業供用（租税特別措置法） |

**注意**: 300万円の上限は事業年度が12ヶ月に満たない場合、月数按分（300万円 × 当期月数/12）。

## 11. 中間申告

### 予定申告（原則）

| 項目 | 内容 |
|------|------|
| 対象 | 前年の法人税額が**20万円超**の法人 |
| 申告期限 | 事業年度開始から**6ヶ月を経過した日から2ヶ月以内** |
| 税額 | 前期の法人税額 × 6/前期の月数 |

### 仮決算による中間申告

| 項目 | 内容 |
|------|------|
| 対象 | 予定申告に代えて選択可能 |
| 方法 | 事業年度開始から6ヶ月間を1事業年度とみなして仮決算 |
| 制限 | 仮決算の税額が予定申告の税額を**超える場合は選択不可** |

### 中間申告不要の場合

- 前期の法人税額が**20万円以下**
- 設立第1期目の法人（前期がないため）

## 12. 法人税の電子申告義務化

### 対象法人

2020年4月1日以後開始事業年度から、以下の法人は**e-Tax（電子申告）が義務化**:

| 対象 | 要件 |
|------|------|
| 大法人 | 事業年度開始時に資本金が**1億円超**の内国法人 |
| 相互会社 | 保険業法の相互会社 |
| 投資法人 | 投資信託及び投資法人に関する法律の投資法人 |
| 特定目的会社 | 資産の流動化に関する法律の特定目的会社 |

### 対象書類

- 確定申告書
- 中間申告書
- 修正申告書
- 添付書類（財務諸表、勘定科目内訳明細書等）

### やむを得ない事由

電気通信回線の故障、災害等のやむを得ない事由がある場合は、書面による申告が認められる（事前または事後に届出が必要）。

## 13. グループ通算制度の概要

### 制度概要

2022年4月1日以後開始事業年度から旧連結納税制度に代わり導入。完全支配関係のある法人グループが適用できる。

### 主な特徴

| 項目 | 内容 |
|------|------|
| 適用単位 | 親法人と完全支配関係（100%保有）のある全ての内国法人 |
| 申告方式 | **各法人が個別に申告**（旧制度は親法人が一括申告） |
| 所得通算 | グループ内の欠損法人の欠損金をグループ全体で通算 |
| 繰越欠損金 | グループ加入前の欠損金はその法人の所得の範囲内でのみ使用可能（特定欠損金） |
| 税率 | 各法人が個別に税率を適用 |
| 開始届出 | 適用開始事業年度の**3ヶ月前の日まで**に届出書を提出 |
| 離脱 | 完全支配関係がなくなった法人は自動的にグループから離脱 |

### 通算される主な項目

- 所得金額と欠損金額の通算
- 受取配当の益金不算入（持株割合の判定にグループ全体を考慮）
- 外国税額控除（グループ全体で計算）
- 研究開発税制（グループ全体で計算）
- 寄附金の損金算入限度額

### 通算されない主な項目

- 交際費の損金不算入額（各法人単位で計算）
- 貸倒引当金の繰入限度額（各法人単位で計算）
- 各法人固有の加算・減算項目

## 14. その他の重要事項

### 税額控除制度

| 制度 | 控除額 | 適用要件 |
|------|--------|---------|
| 中小企業投資促進税制 | 取得価額の7%（特定中小企業は10%） | 機械装置160万円以上等 |
| 中小企業経営強化税制 | 取得価額の7%または10% | 経営力向上計画の認定 |
| 研究開発税制 | 試験研究費の6%〜14% | 試験研究費の額が基準額を超える場合等 |
| 所得拡大促進税制（賃上げ税制） | 給与等増加額の15%〜40% | 前期比で給与等支給額が1.5%以上増加（中小）等 |
| DX投資促進税制 | 取得価額の3%または5% | デジタル関連投資、DX認定 |

### 確定申告に必要な主な添付書類

1. 確定申告書（別表一から必要な別表一式）
2. 決算報告書（貸借対照表、損益計算書、株主資本等変動計算書、個別注記表）
3. 勘定科目内訳明細書
4. 法人事業概況説明書
5. 適用額明細書（特別措置の適用を受ける場合）
6. 税務代理権限証書（税理士が代理する場合）

### 修正申告と更正の請求

| 項目 | 修正申告 | 更正の請求 |
|------|---------|-----------|
| 内容 | 申告額が過少だった場合の増額修正 | 申告額が過大だった場合の減額請求 |
| 期限 | 期限なし（ただし自主的に早期提出が望ましい） | 法定申告期限から**5年以内** |
| 加算税 | 過少申告加算税10%（期限内申告の場合）、50万円超部分は15% | なし |
| 自主的修正 | 税務調査前の自主的修正は加算税なし | - |

### 重加算税

| 区分 | 税率 |
|------|------|
| 過少申告の場合 | 35% |
| 無申告の場合 | 40% |

仮装・隠蔽があった場合に課される。過去5年以内に重加算税を課されたことがある場合は、さらに10%加算。
