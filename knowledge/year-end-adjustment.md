# 年末調整フロー

> 税理士・会計事務所向けCRMナレッジ
> 最終更新: 2026-03-23

---

## 概要

年末調整は、給与所得者の所得税を年間ベースで精算する手続き。
11月の準備から1月末の法定調書提出までの業務フロー。

### 全体タイムライン

```
10月下旬〜11月上旬 : 準備 - 申告書の配布・説明
11月中旬〜12月上旬 : ステップ1 - 対象者の確認
11月中旬〜12月上旬 : ステップ2 - 申告書の回収
12月上旬〜中旬     : ステップ3 - 給与総額の確定
12月中旬           : ステップ4 - 各種控除の計算
12月中旬〜下旬     : ステップ5 - 年税額の計算
12月の最終給与     : ステップ6 - 過不足税額の精算
12月末〜1月中旬    : ステップ7 - 源泉徴収票の作成・交付
1月上旬〜1月31日   : ステップ8 - 法定調書の作成・提出
1月上旬〜1月31日   : ステップ9 - 給与支払報告書の提出
```

---

## ステップ1: 対象者の確認

### 年末調整の対象者

| 対象 | 条件 |
|------|------|
| 12月31日に在職する給与所得者 | 「給与所得者の扶養控除等（異動）申告書」を提出している者 |
| 年の中途で退職した者（一部） | 死亡退職、心身の障害による退職、12月の給与支払後に退職、年間給与が103万円以下の退職者 |
| 非居住者から居住者になった者 | 年の中途で非居住者から居住者になった場合 |

### 年末調整の対象外（確定申告が必要）

| 対象外 | 理由 |
|--------|------|
| 年間給与収入が2,000万円を超える者 | 法律上の除外規定 |
| 2ヶ所以上から給与を受けている者で、他の給与支払者に扶養控除等申告書を提出している者 | 主たる給与支払者のみが年末調整を実施 |
| 年の中途で退職した者（上記の例外を除く） | 退職時の源泉徴収票で確定申告 |
| 非居住者 | 非居住者は年末調整の対象外 |
| 日雇い労働者 | 丙欄適用者は対象外 |
| 災害減免法の適用を受けている者 | 確定申告で精算 |

### チェックポイント

- [ ] 全従業員の対象/対象外を判定したか
- [ ] 扶養控除等申告書の提出状況を確認したか
- [ ] 中途入社者の前職の源泉徴収票を入手したか
- [ ] 2ヶ所以上の給与がある従業員を把握しているか

## ステップ2: 申告書の回収

### 回収すべき申告書

| 申告書 | 正式名称 | 記載内容 |
|--------|---------|---------|
| 扶養控除等（異動）申告書 | 給与所得者の扶養控除等（異動）申告書 | 配偶者・扶養親族の情報、障害者控除、寡婦/ひとり親控除、勤労学生控除 |
| 基礎控除申告書 | 給与所得者の基礎控除申告書 | 合計所得金額の見積額に基づく基礎控除額の計算 |
| 配偶者控除等申告書 | 給与所得者の配偶者控除等申告書 | 配偶者の合計所得金額、配偶者控除/配偶者特別控除の適用判定 |
| 所得金額調整控除申告書 | 所得金額調整控除申告書 | 給与収入850万円超の場合の所得金額調整控除（23歳未満の扶養親族がいる等） |
| 保険料控除申告書 | 給与所得者の保険料控除申告書 | 生命保険料控除、地震保険料控除、社会保険料控除（自分で支払ったもの）、小規模企業共済等掛金控除 |
| 住宅借入金等特別控除申告書 | 給与所得者の（特定増改築等）住宅借入金等特別控除申告書 | 住宅ローン控除（2年目以降）の適用 |

※基礎控除申告書、配偶者控除等申告書、所得金額調整控除申告書は1枚の兼用様式。

### 添付書類

| 申告書 | 添付書類 |
|--------|---------|
| 扶養控除等申告書 | 原則不要（国外居住親族がいる場合は親族関係書類・送金関係書類） |
| 保険料控除申告書 | 生命保険料控除証明書、地震保険料控除証明書、国民年金保険料控除証明書等 |
| 住宅借入金等特別控除申告書 | 住宅借入金等特別控除申告書（税務署から送付される用紙）、金融機関の残高証明書 |

### よくあるミス

| ミス | 対策 |
|------|------|
| 申告書の回収漏れ | チェックリストで全員分の提出状況を管理 |
| 保険料控除証明書の添付忘れ | 申告書と証明書をセットで回収 |
| 配偶者の所得の記載誤り | 配偶者の源泉徴収票や確定申告書で所得金額を確認 |
| 前職の源泉徴収票の未入手 | 中途入社者への早期案内 |
| 住所変更の未届 | 翌年1月1日時点の住所を確認（給与支払報告書の提出先に影響） |

## ステップ5: 年税額の計算

### 計算手順

```
1. 給与所得の計算
   給与収入 − 給与所得控除 = 給与所得

2. 課税所得の計算
   給与所得 − 所得控除の合計 = 課税所得（千円未満切捨て）

3. 算出税額の計算
   課税所得 × 税率 − 控除額 = 算出税額

4. 年調年税額の計算
   算出税額 − 税額控除（住宅ローン控除等） = 年調年税額（100円未満切捨て）
```

### 給与所得控除額（2020年分以降）

| 給与等の収入金額 | 給与所得控除額 |
|----------------|---------------|
| 1,625,000円まで | 550,000円 |
| 1,625,001円〜1,800,000円 | 収入金額×40% − 100,000円 |
| 1,800,001円〜3,600,000円 | 収入金額×30% + 80,000円 |
| 3,600,001円〜6,600,000円 | 収入金額×20% + 440,000円 |
| 6,600,001円〜8,500,000円 | 収入金額×10% + 1,100,000円 |
| 8,500,001円以上 | 1,950,000円（上限） |

### 所得税の速算表（2015年分以降）

| 課税所得金額 | 税率 | 控除額 |
|-------------|------|--------|
| 1,000円〜1,949,000円 | 5% | 0円 |
| 1,950,000円〜3,299,000円 | 10% | 97,500円 |
| 3,300,000円〜6,949,000円 | 20% | 427,500円 |
| 6,950,000円〜8,999,000円 | 23% | 636,000円 |
| 9,000,000円〜17,999,000円 | 33% | 1,536,000円 |
| 18,000,000円〜39,999,000円 | 40% | 2,796,000円 |
| 40,000,000円以上 | 45% | 4,796,000円 |

※上記に復興特別所得税（所得税額×2.1%）が加算される。

## ステップ6: 過不足税額の精算

### やること

| 作業 | 内容 |
|------|------|
| 年調年税額と既徴収税額の比較 | 年調年税額（年間の正しい税額）と、毎月の給与・賞与から源泉徴収した税額の合計を比較 |
| 還付（過納の場合） | 年調年税額 < 既徴収税額 → 差額を12月の給与で還付 |
| 徴収（不足の場合） | 年調年税額 > 既徴収税額 → 差額を12月の給与で徴収 |

### 精算の実施

| 区分 | 処理 |
|------|------|
| 還付の場合 | 12月の給与（又は賞与）の手取り額が増える形で精算。12月の源泉徴収税額から控除しきれない場合は翌月以降の他の従業員の源泉徴収税額から順次控除 |
| 徴収の場合 | 12月の給与（又は賞与）の手取り額が減る形で精算 |

### チェックポイント

- [ ] 年調年税額の計算に誤りがないか（控除の計算を再確認）
- [ ] 既徴収税額の合計が源泉徴収簿と一致しているか
- [ ] 過不足税額の精算が12月の給与に正しく反映されているか
- [ ] 精算後の源泉徴収税額が正しいか

## ステップ7: 源泉徴収票の作成・交付

### やること

| 作業 | 内容 |
|------|------|
| 源泉徴収票の作成 | 全従業員分の「給与所得の源泉徴収票」を作成 |
| 従業員への交付 | 各従業員に源泉徴収票を交付（紙又は電子） |
| 税務署への提出 | 一定の者の源泉徴収票を税務署に提出（法定調書として） |

### 源泉徴収票の交付期限

| 区分 | 交付期限 |
|------|---------|
| 年末調整を行った従業員 | その年の最後の給与支払日の翌月10日まで（通常は翌年1月中） |
| 退職した従業員 | 退職日から1ヶ月以内 |

### 税務署への提出が必要な源泉徴収票

| 区分 | 提出基準 |
|------|---------|
| 役員（現在） | 年間給与150万円超 |
| 役員（退職） | 年間給与50万円超 |
| 弁護士、税理士等 | 年間給与250万円超 |
| 上記以外の従業員 | 年間給与500万円超 |
| 退職者（上記以外） | 年間給与250万円超（受給者に交付した場合） |

## ステップ8: 法定調書の作成・提出

### やること

| 作業 | 内容 | 期限 |
|------|------|------|
| 法定調書合計表の作成 | 各種法定調書の合計を記載した「給与所得の源泉徴収票等の法定調書合計表」を作成 | **1月31日** |
| 法定調書の提出 | 下記の法定調書を所轄税務署に提出 | **1月31日** |

### 主な法定調書（6種類）

| No. | 法定調書 | 提出基準 |
|-----|---------|---------|
| 1 | 給与所得の源泉徴収票 | 上記ステップ7の提出基準に該当する者 |
| 2 | 退職所得の源泉徴収票 | 役員に対して退職金を支払った場合（税務署提出は法人役員のみ） |
| 3 | 報酬、料金、契約金及び賞金の支払調書 | 弁護士・税理士等への報酬（年5万円超）、外交員報酬（年50万円超）等 |
| 4 | 不動産の使用料等の支払調書 | 同一人に年15万円超の不動産使用料を支払った場合 |
| 5 | 不動産等の譲受けの対価の支払調書 | 同一人に年100万円超の不動産等の譲受け対価を支払った場合 |
| 6 | 不動産等の売買又は貸付けのあっせん手数料の支払調書 | 同一人に年15万円超のあっせん手数料を支払った場合 |

### 提出方法

| 方法 | 内容 |
|------|------|
| e-Tax（電子申告） | 前々年の法定調書の提出枚数が100枚以上の場合は電子提出が義務 |
| 光ディスク等 | CD、DVD等の光ディスクでの提出 |
| 書面 | 紙での提出（提出枚数100枚未満の場合） |

### チェックポイント

- [ ] 法定調書合計表の各欄が正しく集計されているか
- [ ] 提出基準に該当する源泉徴収票・支払調書が全て作成されているか
- [ ] マイナンバー（個人番号）が正しく記載されているか
- [ ] e-Tax提出義務に該当するかどうか確認したか

## ステップ9: 給与支払報告書の提出

### やること

| 作業 | 内容 | 期限 |
|------|------|------|
| 給与支払報告書（個人別明細書）の作成 | 全従業員分（退職者含む）の給与支払報告書を作成 | **1月31日** |
| 給与支払報告書（総括表）の作成 | 提出先市区町村ごとに総括表を作成 | **1月31日** |
| 提出 | 従業員の1月1日時点の住所地の市区町村に提出 | **1月31日** |

### 注意点

| 項目 | 内容 |
|------|------|
| 提出対象者 | 前年中に給与を支払った**全員**（税務署への源泉徴収票の提出基準とは異なり、金額にかかわらず全員分を提出） |
| 提出先 | 従業員の**1月1日時点の住所地**の市区町村 |
| 退職者 | 前年中に退職した者（年間給与30万円以下は提出を省略できるが、提出が望ましい） |
| 特別徴収の指定 | 普通徴収切替理由書の提出がない限り、原則として特別徴収（給与天引き） |
| eLTAX | 電子提出が可能（前々年の提出枚数100枚以上は電子提出義務） |

### 普通徴収への切替が認められるケース

| ケース | 切替理由 |
|--------|---------|
| 退職者・退職予定者 | 翌年4月1日までに退職予定 |
| 給与が少額 | 毎月の給与から特別徴収税額を引ききれない |
| 給与の支払が不定期 | 給与の支払期間が不定期 |

## CRMでの管理推奨事項

1. **年末調整タスクの自動生成**: 10月に入ったら自動的に年末調整関連タスクを生成

4. **期限管理**: 法定調書・給与支払報告書の提出期限（1/31）のカウントダウン
