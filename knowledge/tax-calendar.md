# 税務イベント年間カレンダー

> 税理士・会計事務所向けCRMナレッジ
> 最終更新: 2026-03-23

---

## 概要

法人・個人それぞれの税務イベントを月別に整理したカレンダー。
CRMのリマインド機能やタスク自動生成の基礎データとして使用する。

**注意事項**:
- 期限日が土日祝日の場合は翌営業日に繰り延べ
- 法人の申告期限は原則「決算月末日の翌日から2ヶ月以内」
- 申告期限の延長特例（法人税1ヶ月、消費税は不可）あり
- 地方税の納期限は自治体により異なる場合がある

## 法人の決算月別イベント一覧

法人は決算月により申告期限が異なるため、CRMでは法人ごとの決算月に基づいてリマインドを設定する。

### 確定申告（決算月末日の翌日から2ヶ月以内）

| 決算月 | 確定申告期限 | 主な対象 |
|--------|-------------|---------|
| 1月決算 | 3/31 | - |
| 2月決算 | 4/30 | - |
| 3月決算 | 5/31 | 最も多い（日本企業の約20%） |
| 4月決算 | 6/30 | - |
| 5月決算 | 7/31 | - |
| 6月決算 | 8/31 | - |
| 7月決算 | 9/30 | - |
| 8月決算 | 10/31 | - |
| 9月決算 | 11/30 | 2番目に多い |
| 10月決算 | 12/31 | - |
| 11月決算 | 1/31 | - |
| 12月決算 | 2/28（2/29） | 個人事業主の多くが該当 |

### 中間申告（事業年度開始から6ヶ月経過後の2ヶ月以内）

| 決算月 | 中間申告期限 | 備考 |
|--------|-------------|------|
| 3月決算 | 11/30（事業年度開始4/1→中間期末9/30→期限11/30） | - |
| 9月決算 | 5/31（事業年度開始10/1→中間期末3/31→期限5/31） | - |
| 12月決算 | 8/31（事業年度開始1/1→中間期末6/30→期限8/31） | - |

**中間申告の要否判定**: 前事業年度の法人税額が20万円超の場合に必要。消費税は前年の確定消費税額が48万円超で中間申告が必要（金額により年1回・3回・11回）。

### 申告期限の延長特例

| 税目 | 延長期間 | 届出 | 注意 |
|------|---------|------|------|
| 法人税 | 1ヶ月（定款で株主総会が決算日後3ヶ月以内と定められている場合等） | 「申告期限の延長の特例の申請書」を提出 | 延長期間の利子税が発生 |
| 法人住民税・事業税 | 法人税に合わせて延長可 | 各自治体に届出 | 法人税の延長が前提 |
| 消費税 | 延長不可 | - | 法人税の延長特例を受けていても消費税は原則通り |

## 延滞税・加算税の税率一覧

### 延滞税（2026年分の目安）

| 区分 | 税率 |
|------|------|
| 納期限の翌日〜2ヶ月以内 | 年2.4%程度（特例基準割合+1%） |
| 2ヶ月超 | 年8.7%程度（特例基準割合+7.3%） |

※特例基準割合は毎年変動するため、最新の国税庁告示を確認すること。

### 加算税

| 種類 | 税率 | 適用場面 |
|------|------|---------|
| 過少申告加算税 | 10%（増差税額50万円超部分は15%） | 修正申告・更正 |
| 無申告加算税 | 15%（50万円超部分20%、300万円超部分30%） | 期限後申告 |
| 不納付加算税 | 10%（自主納付は5%） | 源泉所得税の納付遅れ |
| 重加算税 | 35%（無申告の場合40%） | 仮装・隠蔽があった場合 |
