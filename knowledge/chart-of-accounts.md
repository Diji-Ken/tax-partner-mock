# 標準勘定科目マスタ

> 税理士・会計事務所向けCRMナレッジ
> 最終更新: 2026-03-23

---

## 概要

法人・個人事業主の標準的な勘定科目体系。
勘定奉行の科目コード体系を基準とし、各科目の内容説明・消費税区分・よくある仕訳例を記載。

**消費税区分の凡例**:
- **課税**: 消費税の課税対象（税率10%または軽減8%）
- **非課税**: 消費税法上の非課税取引（土地の譲渡、保険料、利子等）
- **不課税**: 消費税の課税対象外（給与、寄附金、損害賠償金等）
- **免税**: 輸出免税（税率0%だが仕入税額控除可能）
- **対象外**: 資産・負債・純資産の勘定で消費税区分が発生しないもの

## 消費税区分の判定フロー

```
取引発生
 ├─ 国内取引か？
 │   ├─ Yes → 事業者が事業として行うか？
 │   │         ├─ Yes → 対価を得ているか？
 │   │         │         ├─ Yes → 資産の譲渡等か？
 │   │         │         │         ├─ Yes → 非課税取引に該当するか？
 │   │         │         │         │         ├─ Yes → 【非課税】
 │   │         │         │         │         └─ No  → 【課税】
 │   │         │         │         └─ No  → 【不課税】
 │   │         │         └─ No  → 【不課税】
 │   │         └─ No  → 【不課税】
 │   └─ No  → 輸出取引か？
 │             ├─ Yes → 【免税】
 │             └─ No  → 【不課税（国外取引）】
```
