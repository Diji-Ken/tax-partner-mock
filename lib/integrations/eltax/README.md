# eLTAX連携

## 公式ドキュメント
- eLTAXポータル: https://www.eltax.lta.go.jp/
- PCdeskの特徴と取得方法: https://www.eltax.lta.go.jp/eltax/software/
- 開発者向け: https://www.eltax.lta.go.jp/support/software/
- マニュアル: https://www.eltax.lta.go.jp/support/manual/
- PCdesk Next特設ページ: https://www.eltax.lta.go.jp/news/07816

## 重要: eLTAXはREST APIを公開していない

eLTAXは一般公開されたREST APIを提供していません。連携は以下の2つの方式で行います。

### 連携方式一覧

| 方式 | 概要 | 仕様書入手 |
|------|------|-----------|
| **PCdeskインポート方式** | XMLファイルを生成してPCdeskにインポート | 公開仕様 |
| **API方式** | ソフトウェアが直接ポータルセンターと通信 | 申請制（要同意書） |

### PCdeskインポート方式（推奨）
1. 税務ソフトでXML形式の申告データを作成
2. PCdesk（eLTAXソフト）にインポート
3. PCdeskで電子署名を付与
4. PCdeskからポータルセンターに送信

### API方式（ソフトウェア開発者向け）
1. 地方税共同機構に「eLTAX対応ソフトウェア公開仕様書」の開示を申請
2. 同意書に押印してPDF形式で提出
3. 仕様書に基づいてデータ作成・電子署名・送信を実装
4. ソフトウェア自体が全機能（XML作成、電子署名、送信）を担う

仕様書の申請先:
- 申込書: https://www.eltax.lta.go.jp/support/software/ からダウンロード
- 提出先: 地方税共同機構

## 地方税共通納税システム

- PCdesk(WEB版)またはPCdesk(DL版)から電子納税が可能
- 対応税目: 法人住民税、法人事業税、特別法人事業税、個人住民税（特別徴収）等
- 複数の地方自治体への一括納付に対応

## 特別徴収税額通知の電子化

| 項目 | 説明 |
|------|------|
| 通知方式 | 正本電子データ方式 / 副本電子データ方式 |
| 受取方法 | eLTAXメッセージボックスで受取 |
| データ形式 | XML形式 |
| 対応状況 | 各地方自治体のサービス状況による |

## 対応する申告種別

| 種別 | 説明 |
|------|------|
| 法人住民税 | 道府県民税・市町村民税の申告 |
| 法人事業税 | 事業税・特別法人事業税の申告 |
| 給与支払報告書 | 従業員の住民税特別徴収用データ |
| 固定資産税（償却資産） | 償却資産の申告 |
| 事業所税 | 事業所税の申告 |

## 注意事項・制約
- API方式の仕様書は非公開（申請制）
- 電子署名にはマイナンバーカードまたは税理士用電子証明書が必要
- PCdeskはWindows専用（Mac非対応）
- PCdesk Next（新バージョン）への移行が進行中
- 各地方自治体によって対応状況が異なる
- 給与支払報告書の電子提出は前々年の給与支払報告書の提出枚数が100枚以上の場合に義務化
