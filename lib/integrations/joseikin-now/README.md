## サービス概要

助成金なうは、日本最大級の助成金・補助金検索プラットフォーム。国・自治体の制度に加え、財団・協会の助成金も網羅。

### DB規模（2026年3月時点）

| カテゴリ | 件数 |
|---------|------|
| 国・自治体案件 | **145,169件** |
| 財団・協会案件 | **9,832件** |
| 過去採択企業DB | **829,706社** |
| **合計** | **約155,000件 + 採択企業83万社** |

### 更新頻度

- **日次更新**: 毎日100〜200件の公示を反映
- **月間更新**: 1,500〜2,000件の新規・更新
- 例: 2026年3月17日に国・自治体案件168件、財団・協会案件10件が公示

## データ取得方式

### 1. API（法人プラン / 要契約）

**公開APIドキュメントは存在しない。** 法人プラン契約後に専用APIが案内される。

- 契約: 法人プラン以上（価格は問い合わせ）
- 認証: 契約後に発行されるAPIキー（Bearer トークン）
- データ形式: JSON

### 2. Web検索（会員登録）

会員登録後にWeb上で検索が可能。

| プラン | 月額 | 機能 |
|--------|------|------|
| 無料会員 | 0円 | 基本検索のみ |
| 有料会員 | 1,000円 | 強化検索・お気に入り等 |
| 申請し放題プラン | 要問合せ | 申請代行込み |
| 事業拡大プラン | 要問合せ | コンサルティング付き |
| メガプラン | 要問合せ | フル機能 |
| 法人プラン | 要問合せ | API利用可 |

### 3. 販売代理店プログラム

税理士事務所向けにパートナー契約が可能。

- URL: `/service/joseikin-now/lp/salesagent.php`
- 既存パートナー: オフィスコム、クレオ、ディーシーエス等
- コンテンツ提供: パートナーサイト向け記事執筆サービスあり

### 4. RSS / CSV

**RSS・CSVエクスポート機能は確認されていない。**

## パートナー連携の可能性

### 税理士事務所向け推奨アプローチ

1. **販売代理店契約**: `/lp/salesagent.php` から申請
2. **法人プラン契約**: API利用で自動取り込みが可能に
3. **過去採択企業DB活用**: 顧問先の同業他社がどの補助金を活用したかを参照

### 連携フロー（推奨）

```
[法人プラン契約] → [APIキー取得] → [日次自動取り込み]
                                      ↓
                              [顧問先マッチング]
                                      ↓
                              [提案メール自動生成]
```

## 更新方法

### API利用時（法人プラン）
1. **日次バッチ**: 毎朝6:00にAPI経由で最新データを取得
2. **差分チェック**: 新規公示・更新・終了を検出
3. **顧問先マッチング**: 新着案件と顧問先プロフィールを自動照合

### Web利用時（フォールバック）
1. **ブログ監視**: joseikin-now.jp の新着記事をRSSライクに取得
2. **手動インポート**: Web検索結果をCSV化してインポート

## 税理士CRMポータルでの活用方法

### 1. 10万件超のDB横断検索

```typescript
import { searchSubsidies } from './client';

const result = await searchSubsidies({
  keyword: 'DX推進',
  prefecture: '埼玉県',
  status: '公募中',

```

### 2. 過去採択企業データの活用

顧問先の同業他社がどの補助金を採択されたかを参照し、成功確度の高い補助金を提案。

### 3. 財団・協会の助成金

jGrantsやスマート補助金では網羅しきれない財団・協会の独自助成金（9,832件）を補完。

## 注意事項

- 公開APIは提供されていないため、法人プラン契約が必要
- Web版の大量アクセスは利用規約違反の可能性あり
- 過去採択企業DBは有料プランの機能（無料会員では利用不可）
- 料金は時期により変動するため、最新は公式サイトで確認
