# STREAMED連携

## 公式ドキュメント
- 公式サイト: https://streamedup.com/
- 連携ヘルプ: https://streamedup.com/help/10581
- 勘定科目インポート/エクスポート: https://streamedup.com/help/1540
- 仕訳データ出力: https://streamedup.com/help/6138
- freee API連携: https://streamedup.com/help/12594
- MFクラウド会計連携: https://streamedup.com/help/6145

## 重要: STREAMEDは公開REST APIを提供していない

STREAMEDは一般公開されたREST APIを提供していません。連携は以下の方式で行います。

### 連携方式一覧

| 方式 | 対象 | 説明 |
|------|------|------|
| **freee会計 API連携** | freeeユーザー | STREAMED管理画面から設定。仕訳データを直接送信 |
| **MFクラウド会計 API連携** | MFクラウドパートナー | 記帳システム連携機能を使用 |
| **CSVエクスポート** | 全会計ソフト | 会計ソフト別のCSV形式でダウンロード |
| **CSVインポート** | STREAMED | 過去仕訳参照用に仕訳データをインポート |

### CSVエクスポート手順
1. STREAMEDの「設定 > 会計ソフト 勘定科目」画面でお使いの会計ソフトを選択
2. 仕訳データの出力画面で対象期間を指定
3. CSVファイルをダウンロード
4. 会計ソフトにインポート

### 勘定科目マスタのインポート/エクスポート
- STREAMED管理画面から勘定科目・補助科目をCSVでエクスポート可能
- 編集後に再インポートして一括更新が可能
- 会計ソフトの仕訳データから科目設定をインポートすることも可能

## freee会計 API連携の設定
1. STREAMEDの「設定 > 連携」画面を開く
2. freee会計の「連携する」ボタンをクリック
3. freeeのOAuth認証画面でログイン・許可
4. 仕訳データの出力画面で「freeeへ送信」を選択

## MFクラウド会計 API連携の設定
1. マネーフォワードクラウドパートナーの方のみ利用可能
2. STREAMEDの「設定 > 連携」画面からMFクラウド会計を選択
3. 記帳システム連携機能を使用してデータ連携

## 注意事項・制約
- STREAMEDの公開REST APIは存在しないため、API経由の直接操作は不可
- CSVファイルの形式は会計ソフトごとに異なる（ファイル拡張子もCSV/TXT/SLP等）
- ファイルの文字コードは会計ソフトの要件に合わせて自動設定される
- freee API連携とMFクラウド API連携はSTREAMED側の管理画面から設定
- A-SaaSやフリーウェイ経理向けのファイル名には独自の命名規則がある
- 過去仕訳インポート機能を使うと、STREAMEDのAI学習に過去仕訳データを活用可能
